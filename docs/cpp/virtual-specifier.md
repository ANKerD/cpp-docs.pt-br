# Especificador virtual
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A palavra\-chave [virtual](../cpp/virtual-cpp.md) pode ser aplicado somente a funções de membro de classe não estática.  Significa que a associação de chamadas à função é adiada até o tempo de execução.  Para obter mais informações, consulte [Funções virtuais](../cpp/virtual-functions.md).
