# <a name="jump-statements-c"></a>Instruções de salto (C++)
Uma instrução de salto do C++ executa uma transferência de controle local imediata.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
