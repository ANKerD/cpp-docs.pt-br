# Instru&#231;&#245;es de salto (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Uma instrução de salto do C\+\+ executa uma transferência de controle local imediata.  
  
## Sintaxe  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
