# <a name="program-and-linkage--c"></a>Programa e ligação (C++)
Um programa consiste em uma ou mais unidades de tradução vinculadas. Execução (conceitualmente) começa na unidade de tradução que contém a função **principal**. (Para obter mais informações sobre unidades de tradução, consulte [fases de tradução](../preprocessor/phases-of-translation.md), além de *referência de pré-processador*.) Para obter mais informações sobre o **principal** funcionam, consulte [inicialização de programa: a função principal](../cpp/main-program-startup.md).)
