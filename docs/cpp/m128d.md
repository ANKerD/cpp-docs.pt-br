# <a name="m128d"></a>__m128d
## <a name="microsoft-specific"></a>Específico da Microsoft  
 O `__m128d` tipo de dados, para uso com os Streaming de extensões SIMD 2 instruções intrínsecos, é definido em \<emmintrin.h >.  
  
```  
// data_types__m128d.cpp  
#include <emmintrin.h>  
int main() {  
   __m128d x;  
}  
```  
  
## <a name="remarks"></a>Comentários  
 Você não deve acessar os campos `__m128d` diretamente. No entanto, você pode consultar esses tipos no depurador. Uma variável do tipo `__m128` mapeia para os registros XMM[0-7].  
  
 Variáveis do tipo **_m128d** automaticamente são alinhados em limites de 16 bytes.  
  
 O tipo de dados `__m128d` não é suportado em processadores ARM.  
  
**Fim da seção específica da Microsoft**
