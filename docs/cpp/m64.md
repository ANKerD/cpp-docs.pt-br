# <a name="m64"></a>__m64
## <a name="microsoft-specific"></a>Específico da Microsoft  
 O `__m64` é do tipo de dados para uso com o MMX e 3DNow! intrínsecos e é definido em xmmintrin.h.  
  
```  
// data_types__m64.cpp  
#include <xmmintrin.h>  
int main()  
{  
   __m64 x;  
}  
```  
  
## <a name="remarks"></a>Comentários  
 Você não deve acessar os campos `__m64` diretamente. No entanto, você pode consultar esses tipos no depurador. Uma variável do tipo `__m64` mapeia para os registros MM[0-7].  
  
 Variáveis do tipo **m64** automaticamente são alinhados em limites de 8 bytes.  
  
 O tipo de dados `__m64` não é suportado em processadores [!INCLUDE[vcprx64](../assembler/inline/includes/vcprx64_md.md)]. Os aplicativos que usam __m64 como parte de intrínsecos de MMX devem ser reescritos para usar intrínsecos de SSE e SSE2.  
  
**Fim da seção específica da Microsoft**
