# Limites num&#233;ricos (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Os dois arquivos de inclusão padrão, LIMITS.H e FLOAT.H, definem os limites numéricos, ou os valores mínimo e máximo que uma variável de um tipo específico pode conter.  Esses mínimo e máximo são certamente portáteis para qualquer compilador do C\+\+ que use a mesma representação de dados que o ANSI C.  O arquivo de inclusão LIMITS.H define os [limites numéricos para tipos integrais](../Topic/Integer%20Limits.md), enquanto FLOAT.H define os [limites numéricos para tipos flutuantes](../cpp/floating-limits.md).
