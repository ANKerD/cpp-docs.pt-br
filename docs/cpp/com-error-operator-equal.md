# <a name="comerroroperator-"></a>_com_error::operator =

**Seção específica da Microsoft**

Atribui um objeto `_com_error` existente a outro.

## <a name="syntax"></a>Sintaxe

```
_com_error& operator = (
   const _com_error& that
) throw ( );
```

#### <a name="parameters"></a>Parâmetros

*Isso*<br/>
