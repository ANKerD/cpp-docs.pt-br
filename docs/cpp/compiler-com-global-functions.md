# <a name="compiler-com-global-functions"></a>Funções globais COM do compilador
**Seção específica da Microsoft**  
  
 As seguintes rotinas estão disponíveis:  
  
|Função|Descrição|  
|--------------|-----------------|  
|[_com_raise_error](../cpp/com-raise-error.md)|Gera um [com_error](../cpp/com-error-class.md) em resposta a uma falha.|  
|[_set_com_error_handler](../cpp/set-com-error-handler.md)|Substitui a função padrão que é usada para o tratamento de erros COM.|  
|[ConvertBSTRToString](../cpp/convertbstrtostring.md)|Converte um `BSTR` valor para um **char \* **.|  
|[ConvertStringToBSTR](../cpp/convertstringtobstr.md)|Converte um **char \* ** valor para um `BSTR`.|  
  
**Fim da seção específica da Microsoft**
