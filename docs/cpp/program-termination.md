# <a name="program-termination"></a>Término do programa
No C++, há várias maneiras de sair do programa:  
  
-   Chamar o [sair](../cpp/exit-function.md) função.  
  
-   Chamar o [anular](../cpp/abort-function.md) função.  
  
-   Executar um [retornar](../cpp/return-statement-in-program-termination-cpp.md) instrução do **principal**.
