# <a name="indirection-on-array-types"></a>Indireção em tipos de matriz

Use o operador de indireção (<strong>\*</strong>) em um *n*-tipo de matriz unidimensional produz um *n*matriz dimensional de-1. Se *n* é 1, um escalar (ou elemento de matriz) é gerado.
