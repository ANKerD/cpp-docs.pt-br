# Data e hora: Suporte a banco de dados
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Até a data de versão 4,0, a programação do banco de dados MFC usa as classes de [COleDateTime](../atl-mfc-shared/reference/coledatetime-class.md) e de [COleDateTimeSpan](../Topic/COleDateTimeSpan%20Class.md) para representar dados de data e hora.  Essas classes, também usadas na automação, são derivadas da classe [COleVariant](../mfc/reference/colevariant-class.md).  Fornecem melhor suporte para gerenciar dados de data e hora de que fazem [CTime](../Topic/CTime%20Class.md) e [CTimeSpan](../atl-mfc-shared/reference/ctimespan-class.md).
