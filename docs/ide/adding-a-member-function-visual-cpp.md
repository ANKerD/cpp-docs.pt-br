# Adicionando uma fun&#231;&#227;o de membro (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Em **Modo de Exibição de Classe**, você pode adicionar uma função de membro a qualquer classe.  Quando você fizer isso, uma declaração é adicionada ao arquivo de cabeçalho, e corpo da função de membro de stub é adicionado ao arquivo de implementação de classe, que você pode então modificar.  
  
### Para adicionar uma função de membro a uma classe  
  
1.  Em **Modo de Exibição de Classe**, expanda o nó de projeto para exibir as classes no projeto. \(Para abrir **Modo de Exibição de Classe**, na barra de menus, escolha **Modo de Visualização**, **Modo de Exibição de Classe**.\)  
  
2.  Abra o menu de atalho para a classe que você deseja adicionar uma função de membro a, escolha **Adicionar**, **Adicionar Função**.  
  
3.  Forneça os detalhes apropriadas sobre a função de membro.  Para obter mais informações, consulte [Assistente de Adição de Função de Membro](../ide/add-member-function-wizard.md).  
  
4.  Escolha o botão de **Concluir** para gerar código de função de membro.
