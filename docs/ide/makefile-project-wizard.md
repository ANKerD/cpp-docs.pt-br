# Assistente de Projeto Makefile
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Use este assistente para criar uma linha de comando para executar o mecanismo de compilação.  Depois de criar o projeto, você pode exibir e editar as propriedades do projeto do projeto  [página de propriedades de](../ide/working-with-project-properties.md).  
  
## Visão Geral  
 Esta página do assistente descreve as configurações atuais do projeto para o projeto que você está criando.  Por padrão, o projeto está configurado para criar os dois debug e liberar as configurações do  *nomedoprojeto*. exe.  
  
 Para alterar esses padrões, clique o  [As configurações do aplicativo](../ide/application-settings-makefile-project-wizard.md) guia na coluna à esquerda do assistente e faça as alterações desejadas.
