# Assistente Personalizado
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

O assistente personalizado do Visual C\+\+ é um tipo específico de assistente, projetado para ajudá\-lo a criar seus próprios assistentes.  Depois de configurado, o assistente gera os arquivos de starter que necessários para seus projetos.  
  
## Visão Geral  
 Esta página do assistente descreve as configurações do aplicativo atual para o projeto de assistente personalizada que você está criando.  Por padrão, o assistente cria um projeto da seguinte maneira:  
  
-   Nome amigável do assistente é definido como o nome do projeto que você indicou na  **caixa de diálogo New Project**.  
  
-   O assistente tem uma interface de usuário.  
  
-   A interface de usuário do assistente tem uma página.  
  
 Você pode alterar essas configurações padrão no  [As configurações do aplicativo](../Topic/Application%20Settings,%20Custom%20Wizard.md) página do assistente.
