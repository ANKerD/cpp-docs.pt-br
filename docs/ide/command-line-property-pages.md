# P&#225;ginas de propriedades Linha de Comando
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A maioria das pastas de página de propriedade contém uma  **a linha de comando** página de propriedades.  Esta página exibe quais propriedades são definidas na pasta.  O  **a linha de comando** página de propriedades também contém um  **Opções adicionais** caixa onde você pode especificar propriedades que são válidas para a ferramenta, mas para o qual não há nenhuma propriedade na pasta.  
  
 Qualquer comando digitado na caixa de edição será passado por meio da ferramenta para a pasta.  Nenhuma verificação ou a verificação será feita na entrada, nem haverá qualquer verificação de dependência.  
  
 Para obter informações sobre como acessar o  **a linha de comando** páginas de propriedades, consulte  [How To: especificar propriedades do projeto com páginas de propriedade](../Topic/How%20to:%20Specify%20Project%20Properties%20with%20Property%20Pages.md).
