# Navegando na estrutura da classe (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Você pode navegar de uma estrutura de classe do projeto, incluindo funções, herança e interfaces usando um  [Class View](http://msdn.microsoft.com/pt-br/8d7430a9-3e33-454c-a9e1-a85e3d2db925) ou o  [Pesquisador de objetos](http://msdn.microsoft.com/pt-br/f89acfc5-1152-413d-9f56-3dc16e3f0470).  
  
 Você pode ir para uma declaração de função clicando duas vezes no nome da função no modo de exibição de classe.  Você pode ir para uma definição de classe clicando duas vezes em um nome de classe no modo de exibição de classe.
