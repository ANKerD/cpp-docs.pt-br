# Adicionando uma classe do C++ gen&#233;rica
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Você pode adicionar uma classe genérica do C\+\+ usando  **Class View**.  Uma classe genérica do C\+\+ é uma classe que você definir ou que é derivada de uma classe que você define.  
  
### Para adicionar uma classe genérica do C\+\+ a um projeto  
  
1.  Em  **Class View**, o projeto ao qual você deseja adicionar a nova classe, clique com o botão direito  **Add**e, em seguida, clique em  **classe**.  
  
2.  No  [Add Class](../ide/add-class-dialog-box.md) caixa de diálogo, no painel templates, clique em  **Classe C\+\+**.  Clique em  **Add** para exibir o  [Assistente genérico de classe C\+\+](../ide/generic-cpp-class-wizard.md).  
  
3.  No assistente, forneça um nome de classe e definir as configurações ou aceite os padrões.  
  
4.  Para fechar o assistente e exibir a nova classe genérica do C\+\+ no projeto, clique em  **Concluir**.
