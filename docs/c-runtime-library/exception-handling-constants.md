# Constantes do tratamento de exce&#231;&#245;es
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

`EXCEPTION_CONTINUE_SEARCH`constante, `EXCEPTION_CONTINUE_EXECUTION`, ou `EXCEPTION_EXECUTE_HANDLER` são retornados quando uma exceção ocorre durante a execução da seção protegidos de uma instrução de **try\-except** .  O valor de retorno determina como a exceção é tratada.  Para obter mais informações, consulte [tentativa\-exceto a instrução](../cpp/try-except-statement.md)*na referência de linguagem C\+\+*.
