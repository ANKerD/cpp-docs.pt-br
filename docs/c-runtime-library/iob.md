# _iob
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A matriz de estruturas de controle de stdio.  
  
## Sintaxe  
  
```  
FILE _iob[_IOB_ENTRIES];  
```  
  
## Comentários  
 `IOB_ENTRIES` é definido como 20 em stdio.h.
