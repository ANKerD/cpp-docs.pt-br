# NULL (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**nulo** é o valor do ponteiro nulo usado com muitas operações e funções do ponteiro.  É equivalente a 0.  **nulo** é definido nos arquivos de cabeçalho: CRTDBG.H, LOCALE.H, STDDEF.H, STDIO.H, STDLIB.H, STRING.H, TCHAR.H, TIME.H e. WCHAR.H.
