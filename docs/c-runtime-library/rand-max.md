# <a name="randmax"></a>RAND_MAX
## <a name="syntax"></a>Sintaxe  
  
```  
  
#include <stdlib.h>  
  
```  
  
## <a name="remarks"></a>Comentários  
 A constante `RAND_MAX` é o valor máximo que pode ser retornado pela `rand` função. `RAND_MAX` é definido como o valor 0x7fff.
