# TMP_MAX, L_tmpnam
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Sintaxe  
  
```  
#include <stdio.h>  
```  
  
## Comentários  
 `TMP_MAX` é o número máximo de nomes de arquivo exclusivos que a função de `tmpnam` pode gerar.  `L_tmpnam` é o comprimento dos nomes de arquivos temporários gerados por `tmpnam`.
