# Manipula&#231;&#227;o da cadeia de caracteres (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Use as rotinas de buffer\- manipulação, descritas em [Manipulação de buffer](../Topic/Buffer%20Manipulation.md), para trabalhar com matrizes de caractere que não terminam com um caractere nulo.
