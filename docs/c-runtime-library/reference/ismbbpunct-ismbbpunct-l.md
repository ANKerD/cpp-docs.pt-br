# <a name="ismbbpunct-ismbbpunctl"></a>_ismbbpunct, _ismbbpunct_l
Determina se um dado caractere é um caractere de pontuação.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
int _ismbbpunct(  
   unsigned int c   
);  
int _ismbbpunct_l(  
   unsigned int c,  
   _locale_t locale  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `c`  
 Inteiro a ser testado.  
  
 `locale`  
 Localidade a usar.  
  
## <a name="return-value"></a>Valor de retorno  
 `_ismbbpunct` retornará um valor diferente de zero se o inteiro `c` for um símbolo de pontuação não ASCII. `_ismbbpunct` usa a localidade atual para qualquer configuração de caractere que dependa da localidade. `_ismbbpunct_l` é idêntico, exceto pelo fato de usar a localidade que é passada. Para obter mais informações, consulte [Localidade](../../c-runtime-library/locale.md).
