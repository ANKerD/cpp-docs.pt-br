# <a name="ismbbkpunct-ismbbkpunctl"></a>_ismbbkpunct, _ismbbkpunct_l
Verifica se um caractere multibyte é um caractere de pontuação.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
int _ismbbkpunct(  
   unsigned int c   
);  
int _ismbbkpunct_l(  
   unsigned int c,  
   _locale_t locale  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `c`  
 Inteiro a ser testado.  
  
 `locale`  
 Localidade a usar.  
  
## <a name="return-value"></a>Valor de retorno  
 `_ismbbkpunct` retornará um valor diferente de zero se o inteiro `c` for um símbolo de pontuação não ASCII ou 0 se não for. Por exemplo, na página de código 932 somente, `_ismbbkpunct` testa se é uma pontuação katakana. `_ismbbkpunct` usa a localidade atual para qualquer configuração de caractere que dependa da localidade. `_ismbbkpunct_l` é idêntico, exceto pelo fato de usar a localidade que é passada. Para obter mais informações, consulte [Localidade](../../c-runtime-library/locale.md).
