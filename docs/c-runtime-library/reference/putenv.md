# putenv
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Essa função POSIX foi preterida.  Use ISO C\+\+ de acordo com as conformidades [\_putenv](../../c-runtime-library/reference/putenv-wputenv.md) ou de segurança aprimorada [\_putenv\_s](../../c-runtime-library/reference/putenv-s-wputenv-s.md).  
  
> [!IMPORTANT]
>  Esta API não pode ser usada em aplicativos que executam no Tempo de Execução do Windows.  Para obter mais informações, consulte [Funções CRT sem suporte pelo \/ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).
