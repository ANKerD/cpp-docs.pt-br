# <a name="putenv"></a>putenv
Essa função POSIX foi preterida. Em vez disso, use [_putenv](../../c-runtime-library/reference/putenv-wputenv.md) compatível com ISO C++ ou [_putenv_s](../../c-runtime-library/reference/putenv-s-wputenv-s.md) de segurança avançada.  
  
> [!IMPORTANT]
>  Esta API não pode ser usada em aplicativos executados no Tempo de Execução do Windows. Para obter mais informações, consulte [Funções de CRT sem suporte com /ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).
