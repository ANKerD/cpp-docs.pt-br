# <a name="norm-normf-norml"></a>norm, normf, norml
Recupera a magnitude quadrada de um número complexo.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
double norm(   
   _Dcomplex z   
);  
float norm(   
   _Fcomplex z   
);  // C++ only  
long double norm(   
  _Lcomplex z   
);  // C++ only  
float normf(   
   _Fcomplex z   
);  
long double norml(   
   _Lcomplex z   
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `z`  
 Um número complexo.  
  
## <a name="return-value"></a>Valor de retorno  
 A magnitude quadrada de `z`.  
  
## <a name="remarks"></a>Comentários  
 Como o C++ permite sobrecargas, é possível chamar sobrecargas de `norm` e que usam valores `_Fcomplex` ou `_Lcomplex` e retornam valores `float` ou `long double`. Em um programa do C, `norm` sempre usa um valor `_Dcomplex` e retorna um valor `double`.
