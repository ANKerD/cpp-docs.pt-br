# FOPEN_MAX, _SYS_OPEN
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Sintaxe  
  
```  
  
#include <stdio.h>  
  
```  
  
## Comentários  
 Esse é o número máximo de arquivos que podem ser abertos simultaneamente.  `FOPEN_MAX` é o nome ANSI\-compatível.  `_SYS_OPEN` é fornecido para compatibilidade com o código existente.
