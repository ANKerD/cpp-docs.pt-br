# <a name="windows-platforms-crt"></a>Plataformas Windows (CRT)
As bibliotecas de tempo de execução C para [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] suporte [!INCLUDE[win8](../build/reference/includes/win8_md.md)], [!INCLUDE[winserver8](../build/reference/includes/winserver8_md.md)], [!INCLUDE[win7](../build/includes/win7_md.md)], [!INCLUDE[winsvr08](../build/reference/includes/winsvr08_md.md)], Windows Vista, [!INCLUDE[winxp](../build/includes/winxp_md.md)] Service Pack 3 (SP3) para x86, [!INCLUDE[winxp](../build/includes/winxp_md.md)] Service Pack 2 (SP2) para x64 e [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] Service Pack 2 (SP2) para x86 e x64. Todos esses sistemas operacionais dão suporte à Interface de programação de aplicativo (API) do Windows e oferecem suporte a Unicode. Além disso, todos os aplicativos Win32 podem usar um conjunto de caracteres multibyte (MBCS).  
  
> [!NOTE]
>  [!INCLUDE[vs_dev11_long](../build/includes/vs_dev11_long_md.md)] não dá suporte [!INCLUDE[winxp](../build/includes/winxp_md.md)] e [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] desenvolvimento. Você deve instalar [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] para adicionar suporte para [!INCLUDE[winxp](../build/includes/winxp_md.md)] e [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)].
