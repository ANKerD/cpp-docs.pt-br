# <a name="null-directive"></a>Diretiva nula
A diretiva de pré-processador nulo é um único sinal numérico (**#**) sozinho em uma linha. Ele não tem efeito.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
#  
  
```
