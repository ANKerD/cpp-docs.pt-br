# <a name="null-directive"></a>Diretiva nula
A diretiva de pré-processador nula é um único sinal numérico (**#**) em apenas uma linha. Ele não tem efeito.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
#  
```
