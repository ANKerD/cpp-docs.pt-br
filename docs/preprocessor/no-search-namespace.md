# no_search_namespace
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Específico do C\+\+**  
  
 Tem a mesma funcionalidade do atributo [no\_namespace](../Topic/no_namespace.md), mas é usado em bibliotecas de tipos que você usa a política \#import com o atributo [auto\_search](../preprocessor/auto-search.md).  
  
## Sintaxe  
  
```  
no_search_namespace  
```
