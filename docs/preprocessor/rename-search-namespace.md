# rename_search_namespace
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Específico do C\+\+**  
  
 Tem a mesma funcionalidade do atributo [rename\_namespace](../preprocessor/rename-namespace.md), mas é usado em bibliotecas de tipos que você usa a política \#import com o atributo [auto\_search](../preprocessor/auto-search.md).  
  
## Sintaxe  
  
```  
rename_search_namespace("NewName")  
```  
  
#### Parâmetros  
 `NewName`  
 O nome do novo namespace.
