# new
Descreve brevemente os usos da `new` palavra\-chave e links para mais informações.  
  
|Título|Descrição|  
|------------|---------------|  
|[operator new](../c-runtime-library/operator-new-crt.md)|Aloca um bloco de memória do heap.|  
|[operator new](../Topic/operator%20new%20\(%3Cnew%3E\).md)|A função que é chamada por uma nova expressão para alocar armazenamento para objetos individuais.|  
|[Operador new](../cpp/new-operator-cpp.md)|Aloca memória para um objeto ou matriz de objetos do armazenamento livre e retorna um ponteiro adequadamente tipado, diferente de zero para o objeto.|  
|[new \(new slot in vtable\)](../windows/new-new-slot-in-vtable-cpp-component-extensions.md)|No C \+ \+ \/ projeto CLI, indica que um membro virtual obtém um novo slot em vtable. Em outras palavras, a função não substitui um método da classe base.|  
|[ref new, gcnew](../windows/ref-new-gcnew-cpp-component-extensions.md)|Em um [!INCLUDE[cppwrt_short](../Token/cppwrt_short_md.md)] \(`ref new`\) ou C \+ \+ \/ CLI \(`gcnew`\) projeto, essa palavra\-chave composta cria um objeto de contagem de referência.|
