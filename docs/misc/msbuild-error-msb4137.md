# Erro MSB4137 (MSBuild)
**MSB4137: Valor inválido especificado no arquivo de configuração em “\<x\>”.  Nem o nome de propriedade ou ToolsVersion pode ser uma cadeia de caracteres vazia.**  
  
 Uma definição do kit de ferramentas contém nomes ou valores de propriedade inválidos.  
  
### Para corrigir este erro  
  
-   Certifique\-se de que a definição do kit de ferramentas não contém nomes ou valores de propriedade inválidos ou vazios.
