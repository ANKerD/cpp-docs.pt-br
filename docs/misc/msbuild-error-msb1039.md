# Erro MSB1039 (MSBuild)
**MSB1039: Especificar a versão do kit de ferramentas.**  
  
 Nenhum valor foi especificado quando a opção de **\/ToolsVersion** foi usado.  
  
### Para corrigir este erro  
  
-   Quando você usa a opção de **\/ToolsVersion** , certifique\-se que você especifica um valor válido, não vazio para `ToolsVersion`.
