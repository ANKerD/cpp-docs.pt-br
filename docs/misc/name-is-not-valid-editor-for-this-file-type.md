# &lt; nome &gt; n&#227;o &#233; um editor v&#225;lido para esse tipo de arquivo.
Esse erro geralmente ocorre quando um valor incorreto for inserido para a opção \/e:*editorname* argumento o **comando** janela ou **Localizar\/comando** caixa.  
  
### Para corrigir este erro  
  
1.  Verifique a documentação para a opção \/e: correto*editorname* sintaxe e digite novamente o comando.
