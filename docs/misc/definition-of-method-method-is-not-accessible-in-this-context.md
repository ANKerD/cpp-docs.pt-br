# Defini&#231;&#227;o do m&#233;todo &lt; m&#233;todo &gt; n&#227;o est&#225; acess&#237;vel neste contexto
Uma função de agregação que não existe foi incluída com o `Into` operador.  
  
 **ID do erro:** BC36594  
  
### Para corrigir este erro  
  
-   Verifique a ortografia do nome da função de agregação e faça as correções necessárias.  
  
-   Adicione o método como uma função de agregação personalizada \(método de extensão\). Para obter mais informações sobre como criar uma função de agregação personalizada, consulte [Cláusula Aggregate](../Topic/Aggregate%20Clause%20\(Visual%20Basic\).md).
