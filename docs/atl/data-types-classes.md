# <a name="data-types-classes"></a>Classes de tipos de dados

As seguintes classes encapsulam os tipos de dados C++:

- [CComBSTR](../atl/reference/ccombstr-class.md) encapsula a `BSTR` tipo de dados.

- [CComVariant](../atl/reference/ccomvariant-class.md) encapsula a `VARIANT` tipo de dados.

- [CComCurrency](../atl/reference/ccomcurrency-class.md) inclui métodos e operadores para criar e gerenciar um `CURRENCY` objeto.
