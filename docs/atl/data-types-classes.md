# <a name="data-types-classes"></a>Classes de tipos de dados
As seguintes classes de encapsular os tipos de dados C++:  
  
-   [CComBSTR](../atl/reference/ccombstr-class.md) encapsula o `BSTR` tipo de dados.  
  
-   [CComVariant](../atl/reference/ccomvariant-class.md) encapsula o **VARIANT** tipo de dados.  
  
-   [CComCurrency](../atl/reference/ccomcurrency-class.md) inclui métodos e operadores para criar e gerenciar uma `CURRENCY` objeto.
