# Classes de tipos de dados
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

As seguintes classes envolvem tipos de dados C\+\+:  
  
-   [CComBSTR](../atl/reference/ccombstr-class.md) envolve o tipo de dados de `BSTR` .  
  
-   [CComVariant](../atl/reference/ccomvariant-class.md) envolve o tipo de dados de **VARIANT** .  
  
-   [CComCurrency](../atl/reference/ccomcurrency-class.md) inclui métodos e operadores para criar e gerenciar um objeto de `CURRENCY` .
