# Inserindo um controle composto
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

a caixa de diálogo de **Adicionar Classe** permite que você insira um objeto de ATL em um projeto.  Acessar essa caixa de diálogo clique com o botão direito do mouse no nome do projeto no solution Explorer, **Adicionar**apontando para e em seguida, clicando em **Adicionar Classe**.  
  
 Em a caixa de diálogo de **Adicionar Classe** , escolha **Controle do ATL**.  Isso irá iniciar [Assistente de controle de ATL](../atl/reference/atl-control-wizard.md).  Para criar um controle composto, selecione a guia de **Opções** , e clique na caixa de seleção de **Composite control** .  
  
 Uma página HTML padrão será criado exibindo o controle.
