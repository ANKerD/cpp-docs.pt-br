# Classes de informa&#231;&#245;es do site
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

As seguintes classes permitem que um objeto se comunicar com o site:  
  
-   [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) recupera e defina um ponteiro para o site de um objeto.  Usado para objetos que não são controles.  
  
-   [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) recupera e defina um ponteiro para o site de um objeto.  Usado para controles.
