# Classes do snap-in do MMC
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

As seguintes classes oferecem suporte para desenvolver componentes do snap\-in do Microsoft Management Console \(MMC\):  
  
-   [CSnapInItemImpl](../atl/reference/csnapinitemimpl-class.md) implementa um objeto do snap\-in, como adicionar itens de menu e barras de ferramentas, e encaminhar comandos para o nó do snap\-in a função apropriada do manipulador.  
  
-   [CSnapInPropertyPageImpl](../atl/reference/csnapinpropertypageimpl-class.md) implementa um objeto da página de propriedades do snap\-in.
