# <a name="error-information-classes"></a>Classes de informações de erro
A classe a seguir indica como as informações de erro são tratadas:  
  
-   [ISupportErrorInfoImpl](../atl/reference/isupporterrorinfoimpl-class.md) determina se o objeto é compatível com o [IErrorInfo](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-ierrorinfo) interface. `IErrorInfo` permite que as informações de erro sejam propagadas de volta ao cliente.
