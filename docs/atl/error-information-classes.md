# <a name="error-information-classes"></a>Classes de informações de erro
A classe a seguir indica como as informações de erro são tratadas:  
  
-   [ISupportErrorInfoImpl](../atl/reference/isupporterrorinfoimpl-class.md) determina se o objeto oferece suporte a [IErrorInfo](http://msdn.microsoft.com/en-us/4dda6909-2d9a-4727-ae0c-b5f90dcfa447) interface. **IErrorInfo** permite que as informações de erro ser propagadas de volta para o cliente.
