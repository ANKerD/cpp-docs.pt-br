# O que &#233; AtlAxWin100?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

`AtlAxWin100` é o nome de uma classe de janela que ajuda a fornecer a funcionalidade de hospedagem de ATL.  Quando você cria uma instância da classe, o procedimento de janela usará automaticamente API de hospedagem para criar um objeto host associado com a janela e para carregá\-lo com o controle que você especifica como o título da janela.
