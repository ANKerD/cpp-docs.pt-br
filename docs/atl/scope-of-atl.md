# Escopo de ATL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ATL permite que você facilmente crie servidores de objetos COM, de automação, e controles ActiveX.  ATL fornece suporte interno para muitas das interfaces fundamentais COM.  
  
 ATL é enviado como código\-fonte que você incluir no seu aplicativo.  ATL também torna disponível \(atl90.dll uma DLL\), que contém o código que pode ser compartilhado entre os componentes.  Em o entanto, essa DLL não é necessário.
