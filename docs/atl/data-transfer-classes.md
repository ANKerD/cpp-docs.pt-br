# Classes de transfer&#234;ncia de dados
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

As seguintes classes oferecem suporte a vários tipos de transferência de dados:  
  
-   Transferência de dados de suporte uniforme de[IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) usando formatos padrão para definir e recuperar dados.  Notificações de alteração de dados handles gerenciar conexões para recomendar dissipadores.  
  
-   [CBindStatusCallback](../Topic/CBindStatusCallback%20Class.md) permite que um apelido assíncrona enviar e receber informações sobre a transferência de dados assíncrona a e do objeto.
