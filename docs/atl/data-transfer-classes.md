# <a name="data-transfer-classes"></a>Classes de transferência de dados
As classes a seguir oferecem suporte a vários tipos de transferência de dados:  
  
-   [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) suporta uniforme de transferência de dados usando os formatos padrão para recuperar e definir os dados. Lida com dados de notificações de alteração por meio do gerenciamento de conexões para informar coletores.  
  
-   [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md) permite que um moniker assíncrona enviar e receber informações sobre a transferência de dados assíncrona para e de seu objeto.
