# Classes de ponteiros de interface
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

As seguintes classes gerenciam um ponteiro determinado a interface:  
  
-   [CComPtr](../atl/reference/ccomptr-class.md) executa a contagem de referência automática.  
  
-   [CComQIPtr](../atl/reference/ccomqiptr-class.md) semelhante a `CComPtr`, mas também executa consulte automática de interfaces.  
  
-   [CInterfaceArray](../atl/reference/cinterfacearray-class.md) fornece os métodos úteis quando construir uma matriz de ponteiros da interface COM.  
  
-   [CInterfaceList](../Topic/CInterfaceList%20Class.md) fornece os métodos úteis quando construir uma lista de ponteiros da interface COM.  
  
-   [CComGITPtr](../Topic/CComGITPtr%20Class.md) fornece métodos para manipular os ponteiros de interface e a tabela global \(GIT\) da interface.
