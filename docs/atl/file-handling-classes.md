# Arquivo que trata classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Essas classes fornecem métodos para manipular arquivos, arquivos temporários, e arquivos memória mapeados.  
  
-   [CAtlFile](../atl/reference/catlfile-class.md) essa classe fornece melhor um wrapper em torno de arquivo tratamento API do windows.  
  
-   [CAtlFileMapping](../Topic/CAtlFileMapping%20Class.md) esta classe representa um arquivo memória mapeado, adicionando um operador cast métodos de [CAtlFileMappingBase](../atl/reference/catlfilemappingbase-class.md).  
  
-   [CAtlFileMappingBase](../atl/reference/catlfilemappingbase-class.md) esta classe representa um arquivo memória mapeado.  
  
-   [CAtlTemporaryFile](../atl/reference/catltemporaryfile-class.md) essa classe fornece métodos para a criação e uso de um arquivo temporário.
