# Criando um controle de ATL DHTML
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

O assistente de controle de ATL automatiza o processo de criar um controle DHTML.  Gera arquivos de recursos necessários, incluindo um arquivo HTML que contém código de exemplo.  
  
#### Para criar um controle de ATL DHTML  
  
1.  Siga as etapas em [criando um projeto de ATL](../atl/reference/creating-an-atl-project.md).  
  
2.  Em o modo da classe, clique com o botão direito do mouse no nó de projeto, aponte a **Adicionar**, clique **Adicionar Classe** do menu de atalho.  Em a caixa de diálogo de **Adicionar Classe** , clique duas vezes [Assistente de controle de ATL](../atl/reference/atl-control-wizard.md).  Em o assistente de controle de ATL, clique na guia de **Opções** e selecione **DHTML control**.  
  
 Agora você pode [testar o controle padrão](../atl/testing-the-atl-dhtml-control.md).
