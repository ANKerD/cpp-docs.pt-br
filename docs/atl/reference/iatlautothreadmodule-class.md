# Classe de IAtlAutoThreadModule
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Esta classe representa uma interface para um método de `CreateInstance` .  
  
> [!IMPORTANT]
>  Essa classe e seus membros não podem ser usados em aplicativos executados em [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
## Sintaxe  
  
```  
  
__interface IAtlAutoThreadModule  
  
```  
  
## Comentários  
 A classe deriva de [CAtlAutoThreadModuleT](../../atl/reference/catlautothreadmodulet-class.md)`IAtlAutoThreadModule`, usando o para fornecer código para criar um objeto e recuperar um ponteiro da interface.  
  
## Requisitos  
 **Cabeçalho:** atlbase.h
