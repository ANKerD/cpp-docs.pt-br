# <a name="adding-an-atl-com-10-component"></a>Adicionando um componente ATL COM+ 1.0
Este assistente adiciona um objeto ao seu projeto que ofereça suporte a serviços COM+ 1.0, incluindo transações.  
  
### <a name="to-add-an-atl-com-10-component-to-your-project"></a>Para adicionar um componente COM+ 1.0 do ATL ao seu projeto  
  
1.  No **Solution Explorer** ou [Class View](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), clique no nome do projeto ao qual você deseja adicionar o componente COM+ 1.0 do ATL.  
  
2.  No menu de atalho, clique em **adicionar**e, em seguida, clique em **Adicionar classe**.  
  
3.  No [Add Class](../../ide/add-class-dialog-box.md) caixa de diálogo, no painel de modelos, clique em **componente ATL COM+ 1.0**e, em seguida, clique em **abrir** para exibir o [Assistente componente ATL COM+ 1.0](../../atl/reference/atl-com-plus-1-0-component-wizard.md).
