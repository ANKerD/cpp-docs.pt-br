# <a name="adding-an-atl-com-10-component"></a>Adicionando um componente ATL COM+ 1.0
Este assistente adiciona um objeto ao seu projeto que suporte serviços COM+ 1.0, incluindo as transações.  
  
### <a name="to-add-an-atl-com-10-component-to-your-project"></a>Para adicionar um componente de ATL COM+ 1.0 ao seu projeto  
  
1.  No **Solution Explorer** ou [exibição de classe](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), clique no nome do projeto ao qual você deseja adicionar o componente ATL COM+ 1.0.  
  
2.  No menu de atalho, clique em **adicionar**e, em seguida, clique em **Adicionar classe**.  
  
3.  No [Adicionar classe](../../ide/add-class-dialog-box.md) caixa de diálogo, no painel modelos, clique em **componente ATL COM+ 1.0**e, em seguida, clique em **abrir** para exibir o [Assistente componente ATL COM+ 1.0 ](../../atl/reference/atl-com-plus-1-0-component-wizard.md).
