# <a name="default-atl-project-configurations"></a>Configurações padrão do projeto ATL
O Assistente de projeto de ATL cria duas configurações de projeto por padrão:  
  
|Configuração|Conjunto de caracteres|Uso de ATL|  
|-------------------|-------------------|----------------|  
|Versão|MBCS|DLL|  
|Depurar|MBCS|DLL|  
  
 **Conjunto de caracteres**, **uso de ATL** e pode ser alterada no **configurações de projeto** diálogo sob o **geral** guia. Você também pode adicionar suas próprias configurações usando o Configuration Manager. Para obter detalhes, consulte [Build Configurations](/visualstudio/ide/understanding-build-configurations).
