# Criando um projeto ATL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

A maneira mais fácil de criar um projeto de ATL é usar o assistente de projeto de ATL, localizado na pasta de projeto Win32 de **Novo caixa de diálogo de projeto**.  
  
### Para criar um projeto de ATL que usa o ATL projeta o assistente  
  
1.  Siga as instruções no tópico [Criando um projeto com um assistente de aplicativo Visual C\+\+](../../ide/creating-desktop-projects-by-using-application-wizards.md).  
  
2.  Selecione o ícone de **Projeto do ATL** no painel de modelos para abrir o assistente de projeto de ATL.  
  
3.  defina suas configurações de aplicativo usando a página de [configurações de aplicativo](../Topic/Application%20Settings,%20ATL%20Project%20Wizard.md) de `ATL Project Wizard`.  
  
    > [!NOTE]
    >  Ignore esta etapa para manter as configurações padrão do assistente.  
  
4.  **Concluir** Clique em para fechar o assistente e abrir o novo projeto no ambiente de desenvolvimento.  
  
 Uma vez que o projeto é criado, você pode exibir os arquivos criados em **Gerenciador de Soluções**.  Para obter mais informações sobre arquivos que o assistente cria para seu projeto, consulte o arquivo ReadMe.txt projeto gerado.  Para obter mais informações sobre tipos de arquivo, consulte [Tipos de arquivo criados para projetos Visual C\+\+](../../ide/file-types-created-for-visual-cpp-projects.md).  Para obter mais informações sobre configurações para o novo projeto de ATL, e como alterar\-las, consulte [Usar como padrão configurações do projeto de ATL](../../atl/reference/default-atl-project-configurations.md).
