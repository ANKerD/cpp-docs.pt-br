# Fun&#231;&#245;es de convers&#227;o globais de Pixel/HIMETRIC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Essas funções oferecem suporte para converter e de pixel e unidades de HIMETRIC.  
  
> [!IMPORTANT]
>  As funções listadas na tabela a seguir não podem ser usadas em aplicativos executados em [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
|||  
|-|-|  
|[AtlHiMetricToPixel](../Topic/AtlHiMetricToPixel.md)|Unidades converte HIMETRIC \(cada unidade é 0,01 mm\) para pixels.|  
|[AtlPixelToHiMetric](../Topic/AtlPixelToHiMetric.md)|Pixels converte às unidades de HIMETRIC \(cada unidade é 0,01 mm\).|
