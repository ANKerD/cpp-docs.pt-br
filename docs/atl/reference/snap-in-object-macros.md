# Macros do snap-in
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Esses macros oferecem suporte para extensões do snap\-in.  
  
|||  
|-|-|  
|[BEGIN\_EXTENSION\_SNAPIN\_NODEINFO\_MAP](../Topic/BEGIN_EXTENSION_SNAPIN_NODEINFO_MAP.md)|Marca o início do mapa da classe de dados de extensão do snap\-in do MMC snap\-in para um objeto.|  
|[BEGIN\_SNAPINTOOLBARID\_MAP](../Topic/BEGIN_SNAPINTOOLBARID_MAP.md)|Marca o início do mapa da barra de ferramentas para um objeto do snap\-in.|  
|[END\_EXTENSION\_SNAPIN\_NODEINFO\_MAP](../Topic/END_EXTENSION_SNAPIN_NODEINFO_MAP.md)|A marca final do mapa da classe de dados de extensão do snap\-in do MMC snap\-in para um objeto.|  
|[END\_SNAPINTOOLBARID\_MAP](../Topic/END_SNAPINTOOLBARID_MAP.md)|A marca final do mapa da barra de ferramentas para um objeto do snap\-in.|  
|[EXTENSION\_SNAPIN\_DATACLASS](../Topic/EXTENSION_SNAPIN_DATACLASS.md)|Cria um membro de dados para a classe de dados de extensão do snap\-in.|  
|[EXTENSION\_SNAPIN\_NODEINFO\_ENTRY](../Topic/EXTENSION_SNAPIN_NODEINFO_ENTRY.md)|Insere uma classe de dados de extensão do snap\-in no mapa da classe de dados de extensão do snap\-in do MMC snap\-in.|  
|[SNAPINMENUID](../Topic/SNAPINMENUID.md)|Declara a identificação do menu de contexto usado pelo snap\-in do objeto.|  
|[SNAPINTOOLBARID\_ENTRY](../Topic/SNAPINTOOLBARID_ENTRY.md)|Entra em uma barra de ferramentas no mapa da barra de ferramentas do snap\-in.|
