# Macros de classe de janela
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Esses macros definem utilitários da classe da janela.  
  
|||  
|-|-|  
|[DECLARE\_WND\_CLASS](../Topic/DECLARE_WND_CLASS.md)|Permite que você especifique o nome de uma classe de nova janela.|  
|[DECLARE\_WND\_SUPERCLASS](../Topic/DECLARE_WND_SUPERCLASS.md)|Permite que você especifique o nome de uma classe existente da janela até que uma classe de nova janela será base.|  
|[DECLARE\_WND\_CLASS\_EX](../Topic/DECLARE_WND_CLASS_EX.md)|Permite que você especifique os parâmetros de uma classe.|
