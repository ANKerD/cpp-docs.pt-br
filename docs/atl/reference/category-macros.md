# Macros de categoria
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Esses macros definem mapas de categoria.  
  
|||  
|-|-|  
|[BEGIN\_CATEGORY\_MAP](../Topic/BEGIN_CATEGORY_MAP.md)|Marca o início do mapa de categoria.|  
|[END\_CATEGORY\_MAP](../Topic/END_CATEGORY_MAP.md)|A marca final do mapa de categoria.|  
|[IMPLEMENTED\_CATEGORY](../Topic/IMPLEMENTED_CATEGORY.md)|Indica as categorias que são implementadas pelo objeto COM.|  
|[REQUIRED\_CATEGORY](../Topic/REQUIRED_CATEGORY.md)|Indica as categorias que são necessárias do recipiente pelo objeto COM.|
