# <a name="adding-an-atl-control"></a>Adicionar um controle ATL
Use este assistente para adicionar um objeto de interface de usuário a um projeto que oferece suporte a interfaces para todos os contêineres potenciais. Para dar suporte a essas interfaces, o projeto deve ter sido criado como um aplicativo ATL ou como um aplicativo MFC que contém suporte ATL. Você pode usar o [Assistente de projeto de ATL](../../atl/reference/atl-project-wizard.md) para criar um aplicativo ATL, ou [adicionar um objeto ATL ao seu aplicativo do MFC](../../mfc/reference/adding-atl-support-to-your-mfc-project.md) para implementar o suporte ATL para um aplicativo do MFC.  
  
### <a name="to-add-an-atl-control-to-your-project"></a>Para adicionar um controle ATL ao seu projeto  
  
1.  No **Solution Explorer** ou [exibição de classe](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), clique no nome do projeto ao qual você deseja adicionar o objeto ATL simples.  
  
2.  Clique em **adicionar** do menu de atalho e clique **Adicionar classe**.  
  
3.  No [Adicionar classe](../../ide/add-class-dialog-box.md) caixa de diálogo, no painel modelos, clique em **controle ATL**e, em seguida, clique em **adicionar** para exibir o [Assistente de controle ATL](../../atl/reference/atl-control-wizard.md).  
  
 Usando o **Assistente de controle ATL**, você pode criar um dos três tipos de controles:  
  
-   Um controle padrão  
  
-   Um controle composto  
  
-   Um controle DHTML  
  
 Além disso, você pode reduzir o tamanho do controle e remover interfaces que não são usados pela maioria dos contêineres selecionando **controle mínimo** no **opções** página do assistente.
