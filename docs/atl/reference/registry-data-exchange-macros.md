# Macros troca de dados do Registro
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Esses macros executam operações de troca de dados do Registro.  
  
|||  
|-|-|  
|[BEGIN\_RDX\_MAP](../Topic/BEGIN_RDX_MAP.md)|Marca o início do mapa de troca de dados do Registro.|  
|[END\_RDX\_MAP](../Topic/END_RDX_MAP.md)|A marca final do mapa de troca de dados do Registro.|  
|[RDX\_BINARY](../Topic/RDX_BINARY.md)|Associa a entrada de Registro especificada com um variável de membro especificado de BYTE do tipo.|  
|[RDX\_CSTRING\_TEXT](../Topic/RDX_CSTRING_TEXT.md)|Associa a entrada de Registro especificada com um variável de membro de tipo especificado CString.|  
|[RDX\_DWORD](../Topic/RDX_DWORD.md)|Associa a entrada de Registro especificada com um variável de membro especificado DWORD do tipo.|  
|[RDX\_TEXT](../Topic/RDX_TEXT.md)|Associa a entrada de Registro especificada com um variável de membro de tipo especificado TCHAR.|
