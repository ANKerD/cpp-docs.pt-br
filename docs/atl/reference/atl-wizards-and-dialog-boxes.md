# <a name="atl-wizards-and-dialog-boxes"></a>Caixas de diálogo e assistentes da ATL
Os assistentes da biblioteca ATL (Active Template) geram código clichê para vários tipos de objetos COM. Você pode executar os assistentes, abrindo o menu de atalho para um projeto no **Solution Explorer** e escolhendo **adicionar**, **classe**.
