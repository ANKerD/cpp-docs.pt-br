# Classe de CAtlDebugInterfacesModule
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Essa classe fornece suporte para depurar interfaces.  
  
## Sintaxe  
  
```  
  
class CAtlDebugInterfacesModule  
  
```  
  
## Comentários  
 `CAtlDebugInterfacesModule` fornece suporte necessário depurando interfaces.  É incluído em qualquer projeto que definir o símbolo [\_ATL\_DEBUG\_QI](../Topic/_ATL_DEBUG_QI.md).  
  
## Requisitos  
 **Cabeçalho:** atlbase.h
