# Classes de pool de segmentos
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Seguinte pool de segmentos de suporte de classes:  
  
-   [CComAutoThreadModule](../atl/reference/ccomautothreadmodule-class.md) implementa um módulo de EXE, com suporte a construção agrupados com múltiplos.  
  
-   [CComApartment](../atl/reference/ccomapartment-class.md) gerencia um compartimento com agrupados em um módulo de EXE.  
  
-   [CComSimpleThreadAllocator](../atl/reference/ccomsimplethreadallocator-class.md) gerencia a seleção da thread para um módulo de EXE.
