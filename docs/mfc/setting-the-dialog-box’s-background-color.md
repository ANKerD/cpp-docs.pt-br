# Definindo a cor do plano de fundo da caixa de di&#225;logo
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Você pode definir a cor do plano de fundo das caixas de diálogo tratando mensagens de `WM_CTLCOLOR` para a janela da caixa de diálogo.  A cor você define é usado somente para a caixa de diálogo especificada.
