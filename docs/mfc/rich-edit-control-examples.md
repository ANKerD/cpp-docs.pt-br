# <a name="rich-edit-control-examples"></a>Exemplos de controle edição avançada

Por extensão, ele usa o [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md). Para obter uma descrição rápida dessas três classes, consulte [Classes relacionadas aos controles de Rich editar](../mfc/classes-related-to-rich-edit-controls.md).
