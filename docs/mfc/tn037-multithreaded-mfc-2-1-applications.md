# TN037: aplicativos MFC 2.1 multithread
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Observe que essa técnica descritos originalmente as restrições de programas de vários threads com MFC 2,1, fornecidas originalmente com Visual C\+\+ 1,0 para o Windows NT.  O MFC 3,0 da suporte a multithreading diretamente e é documentado.  Consulte a referência para obter mais informações.
