# Editando dados bin&#225;rios
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Para editar um recurso no editor binário  
  
1.  Selecione o byte que você quer editar.  
  
     A tecla TAB move o foco entre a seções de ASCII do editor binário e hexadecimal.  Você pode usar as teclas PAGE UP e PAGE DOWN para percorrer a tela de um recurso por vez.  
  
2.  Digite o novo valor.  
  
     O valor é alterado imediatamente em hexadecimal e seções de ASCII e o foco é deslocado para o próximo valor na linha.  
  
    > [!NOTE]
    >  O editor binário aceita alterações automaticamente quando você fechar o editor.
