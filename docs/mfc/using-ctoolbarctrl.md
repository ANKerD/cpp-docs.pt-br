# Usando CToolBarCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Você pode usar o controle da barra de ferramentas \([CToolBarCtrl](../mfc/reference/ctoolbarctrl-class.md)\) para criar uma janela de controle que contém botões e espaços opcionais.  Cada botão na janela de controle da barra de ferramentas envia uma mensagem de comando para a janela pai como o usuário a escolher.  Normalmente, os botões em uma barra de ferramentas correspondem aos itens no menu do aplicativo, fornecendo adicional e maneira mais direta para que o usuário acesse os comandos de um aplicativo.
