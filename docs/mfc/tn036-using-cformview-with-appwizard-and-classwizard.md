# TN036: usando CFormView com AppWizard e ClassWizard
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Observe que essa técnica descrita como modificar um AppWizard gerou o aplicativo de modo que usa `CFormView` em vez de `CView` padrão como sua classe principal da exibição.  Isso tem suporte diretamente com esta versão do Visual C\+\+.
