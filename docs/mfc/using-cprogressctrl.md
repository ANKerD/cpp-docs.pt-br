# Usando CProgressCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Você pode usar o controle do progresso para indicar o progresso de uma operação demorada.  É um retângulo que seja preenchido gradativamente com a cor do realce do sistema como a operação andamento.  
  
 O controle do progresso é representado em MFC pela classe [CProgressCtrl](../mfc/reference/cprogressctrl-class.md).  
  
 Quando você cria inicialmente o controle do progresso, você especifica sua tamanho e posição, a janela pai \(normalmente uma caixa de diálogo\), e ID  Usando o parâmetro de `dwStyle` , você também pode especificar vários estilos da janela para o controle e estilos para como preenchimento.
