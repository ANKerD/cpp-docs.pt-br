# Classes de banco de dados OLE
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

O suporte do OLE DB em MFC está sendo [COLEDBRecordView](../mfc/reference/coledbrecordview-class.md)na classe.  `COleDBRecordView` exibe registros de base de dados dos controles, com uma exibição de formulário conectada diretamente a um objeto de [CRowset](../Topic/CRowset%20Class.md) .  Para obter mais informações sobre modelos do consumidor OLE DB, consulte [Lista de modelos do consumidor OLE DB](../data/oledb/ole-db-consumer-templates-reference.md).
