# <a name="ole-db-classes"></a>Classes de banco de dados OLE
O suporte de OLE DB em MFC consiste no momento a classe [COLEDBRecordView](../mfc/reference/coledbrecordview-class.md). `COleDBRecordView`Exibe os registros do banco de dados em controles, através de uma exibição de formulário conectado diretamente a um [CRowset](../data/oledb/crowset-class.md) objeto. Para obter mais informações sobre os modelos de consumidor OLE DB, consulte [lista de modelos OLE DB consumidor](../data/oledb/ole-db-consumer-templates-reference.md).
