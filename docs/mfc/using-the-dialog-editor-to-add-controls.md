# Usando o editor da caixa de di&#225;logos para adicionar controles
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Quando você cria o recurso de diálogo\- modelo com [editor da caixa de diálogo](../mfc/dialog-editor.md), você arrasta controles de uma paleta de controles e solte\-os na caixa de diálogo.  Isso adiciona as especificações para esse tipo de controle ao recurso de diálogo\- modelo.  Quando você cria um objeto da caixa de diálogo e chama a função de membro de **Criar** ou de `DoModal` , a estrutura cria um controle do windows e o coloca na janela da caixa de diálogo na tela.  
  
 Você pode então [crie controla a mão](../mfc/adding-controls-by-hand.md) se você deseja.  Este é mais trabalho.
