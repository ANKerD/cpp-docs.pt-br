##  <a name="setdefaultwinxpcolors"></a>CMFCVisualManagerOffice2003::SetDefaultWinXPColors  
 Especifica se o Gerenciador visual deve usar cores de tema nativas do Windows XP ou cores obtido [GetSysColor](http://msdn.microsoft.com/library/windows/desktop/ms724371).  
  
```  
static void SetDefaultWinXPColors(BOOL bDefaultWinXPColors = TRUE);
```  
  
### <a name="parameters"></a>Parâmetros  
 [in] `bDefaultWinXPColors`  
 Especifica se o Gerenciador visual irá usar cores nativas do Windows XP.  
  
### <a name="remarks"></a>Comentários  
 Se `bDefaultWinXPColors` é `TRUE`, o Gerenciador visual usará cores nativas do Windows XP como azul, verde-oliva ou prata. Caso contrário, o Gerenciador visual usará as cores obtidas `GetSysColor`. O Gerenciador visual usa elementos visuais como `COLOR_3DFACE`, `COLOR_3DSHADOW`, `COLOR_3DHIGHLIGHT`, `COLOR_3DDKSHADOW`, e `COLOR_3DLIGHT`.  
  
 Por padrão, um `CMFCVisualManagerOffice2003` objeto usa cores de tema nativas do Windows XP.
