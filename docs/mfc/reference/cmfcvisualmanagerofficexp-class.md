# <a name="cmfcvisualmanagerofficexp-class"></a>Classe CMFCVisualManagerOfficeXP

O `CMFCVisualManagerOfficeXP` dá um aplicativo a aparência do Microsoft Office XP.

## <a name="syntax"></a>Sintaxe

```
class CMFCVisualManagerOfficeXP : public CMFCVisualManager
```

## <a name="members"></a>Membros

Todos os membros dessa classe são funções virtuais que são derivadas do ancestral dessa classe, [classe CMFCVisualManager](../../mfc/reference/cmfcvisualmanager-class.md).

## <a name="remarks"></a>Comentários

Você usa o `CMFCVisualManagerOfficeXP` classe para alterar a aparência visual do seu aplicativo para ser semelhante do Microsoft Office XP.
