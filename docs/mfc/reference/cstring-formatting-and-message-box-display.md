# Formata&#231;&#227;o CString e exibi&#231;&#227;o da caixa de mensagem
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Várias funções são fornecidas para formatar e analisar objetos de `CString` .  Você pode usar essas funções sempre que você tiver que manipular objetos de `CString` , mas são particularmente úteis para formatar as cadeias de caracteres que aparecerão no texto da caixa de mensagem.  
  
 Esse grupo de funções também inclui uma rotina global para exibir uma caixa de mensagem.  
  
### Funções de CString  
  
|||  
|-|-|  
|[AfxExtractSubString](../Topic/AfxExtractSubString.md)|A subcadeia de caracteres de extrações separados por um único caractere de uma cadeia de caracteres de origem especificada.|  
|[AfxFormatString1](../Topic/AfxFormatString1.md)|Substitui uma cadeia de caracteres fornecida para os caracteres de formato “%1 " em uma cadeia de caracteres contida na tabela de cadeia de caracteres.|  
|[AfxFormatString2](../Topic/AfxFormatString2.md)|Substitutas duas cadeias de caracteres para os caracteres de formato “%1 " e “%2 " em uma cadeia de caracteres contida na tabela de cadeia de caracteres.|  
|[AfxMessageBox](../Topic/AfxMessageBox.md)|Exibe uma caixa de mensagem.|
