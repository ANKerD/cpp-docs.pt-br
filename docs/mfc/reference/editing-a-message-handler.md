# Editando um manipulador de mensagens
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Uma vez que você definiu um manipulador de mensagens, você pode ir para a definição de função de membro adicionar ou alterar o código.  
  
 Para ignorar a uma definição de função de membro de [editor da caixa de diálogo](../../mfc/dialog-editor.md), clique duas vezes em um controle para um manipulador já está definido.  Isso navegar em\-lo ao arquivo em que o manipulador de mensagens do controle selecionado é definido.  
  
 Para ignorar a uma definição de função de membro de [Exibição da classe](http://msdn.microsoft.com/pt-br/8d7430a9-3e33-454c-a9e1-a85e3d2db925), clique duas vezes no nome da função na exibição da classe.
