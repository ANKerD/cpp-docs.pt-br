# Fun&#231;&#245;es de bitmap cinza e pontilhado
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Funções em cinza de bitmap**  
  
 MFC O fornece duas funções para conceder a um bitmap a aparência de um controle desabilitado.  
  
 ![Vs originais Cin Interface](../../mfc/reference/media/vcgraybitmap.png "vcGrayBitmap")  
  
|||  
|-|-|  
|[AfxDrawGrayBitmap](../Topic/AfxDrawGrayBitmap.md)|Desenha uma versão cinza de um bitmap.|  
|[AfxGetGrayBitmap](../Topic/AfxGetGrayBitmap.md)|Copia uma versão cinza de um bitmap.|  
  
 **Funções hesitadas de bitmap**  
  
 MFC O também fornece duas funções para substituir o plano de fundo de um bitmap com um padrão hesitado.  
  
 ![Exemplo de Bitmap pontilhado](../../mfc/reference/media/vcditheredbitmap.png "vcDitheredBitmap")  
  
|||  
|-|-|  
|[AfxDrawDitheredBitmap](../Topic/AfxDrawDitheredBitmap.md)|Desenha um bitmap com um plano de fundo hesitado.|  
|[AfxGetDitheredBitmap](../Topic/AfxGetDitheredBitmap.md)|Copia um bitmap com um plano de fundo hesitado.|
