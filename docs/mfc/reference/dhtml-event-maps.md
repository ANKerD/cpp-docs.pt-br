# Mapas de evento DHTML
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Os seguintes macros podem ser usados para tratar eventos do DHTML.  
  
## Macros do mapa de evento do DHTML  
 Os seguintes macros podem ser usados para tratar eventos em [CDHtmlDialog](../Topic/CDHtmlDialog%20Class.md)\- classes derivadas do DHTML.  
  
|||  
|-|-|  
|[BEGIN\_DHTML\_EVENT\_MAP](../Topic/BEGIN_DHTML_EVENT_MAP.md)|Marca o início do mapa de evento do DHTML.|

|[DECLARE\_DHTML\_EVENT\_MAP](../Topic/DECLARE_DHTML_EVENT_MAP.md)|Declara o mapa de evento do DHTML.|  
|[DHTML\_EVENT](../Topic/DHTML_EVENT.md)|Usado para tratar um evento no nível do documento em para um único elemento HTML.|  
|[DHTML\_EVENT\_AXCONTROL](../Topic/DHTML_EVENT_AXCONTROL.md)|Usado para tratar um evento acionado por um controle ActiveX.|  
|[DHTML\_EVENT\_CLASS](../Topic/DHTML_EVENT_CLASS.md)|Usado para tratar um evento no nível do documento em para todos os elementos HTML com uma classe específica de CSS.|  
|[DHTML\_EVENT\_ELEMENT](../Topic/DHTML_EVENT_ELEMENT.md)|Usado para tratar um evento no nível do elemento.|  
|[DHTML\_EVENT\_ONAFTERUPDATE](../Topic/DHTML_EVENT_ONAFTERUPDATE.md)|Usado para tratar o evento de **onafterupdate** de um elemento HTML.|  
|[DHTML\_EVENT\_ONBEFOREUPDATE](../Topic/DHTML_EVENT_ONBEFOREUPDATE.md)|Usado para tratar o evento de **onbeforeupdate** de um elemento HTML.|

|[DHTML\_EVENT\_ONDATASETCOMPLETE](../Topic/DHTML_EVENT_ONDATASETCOMPLETE.md)|Usado para tratar o evento de **ondatasetcomplete** de um elemento HTML.|

## Macros do mapa de evento de URL  
 Os seguintes macros podem ser usados para tratar eventos em [CMultiPageDHtmlDialog](../../mfc/reference/cmultipagedhtmldialog-class.md)\- classes derivadas do DHTML.  
  
|||  
|-|-|  
|[BEGIN\_DHTML\_URL\_EVENT\_MAP](../Topic/BEGIN_DHTML_URL_EVENT_MAP.md)|Marca o início mapa de evento de várias páginas do DHTML e de URL.|  
|[BEGIN\_EMBED\_DHTML\_EVENT\_MAP](../Topic/BEGIN_EMBED_DHTML_EVENT_MAP.md)|Marca o início de um mapa inserido de evento do DHTML.|  
|[BEGIN\_URL\_ENTRIES](../Topic/BEGIN_URL_ENTRIES.md)|Marca o início de um mapa de entrada do evento de URL.|  
|[DECLARE\_DHTML\_URL\_EVENT\_MAP](../Topic/DECLARE_DHTML_URL_EVENT_MAP.md)|Declara o mapa de evento de várias páginas do DHTML e de URL.|  
|[END\_DHTML\_URL\_EVENT\_MAP](../Topic/END_DHTML_URL_EVENT_MAP.md)|Marca o término mapa de evento de várias páginas do DHTML e de URL.|  
|[END\_EMBED\_DHTML\_EVENT\_MAP](../Topic/END_EMBED_DHTML_EVENT_MAP.md)|Marca o fim de um mapa inserido de evento do DHTML.|  
|[END\_URL\_ENTRIES](../Topic/END_URL_ENTRIES.md)|Marca o fim de um mapa de entrada do evento de URL.|  
|[URL\_EVENT\_ENTRY](../Topic/URL_EVENT_ENTRY.md)|Mapeia um recurso de URL ou de HTML para uma página em uma caixa de diálogo com várias páginas.|
