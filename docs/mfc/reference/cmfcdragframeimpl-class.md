# <a name="cmfcdragframeimpl-class"></a>Classe CMFCDragFrameImpl
O `CMFCDragFrameImpl` classe desenha arrastar retângulo que aparece quando o usuário arrasta um painel no modo padrão de encaixe.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
class CMFCDragFrameImpl  
```  
  
## <a name="remarks"></a>Comentários  
 Um objeto dessa classe é incorporado em cada [CPane classe](../../mfc/reference/cpane-class.md) objeto. Portanto, cada painel que usa o `CanFloat` método exibe um retângulo de arrastar quando o usuário arrasta-lo.  
  
 Você pode controlar a espessura do retângulo de arrastar usando [AFX_GLOBAL_DATA::m_nDragFrameThicknessFloat]--brokenlink--(afx-global-data-structure.md#m_ndragframethicknessfloat) e [AFX_GLOBAL_DATA::m_nDragFrameThicknessDock](afx-global-data-structure.md#m_ndragframethicknessdock).
