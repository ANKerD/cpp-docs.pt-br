[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

## Sintaxe  
  

## Comentários  
 Use `CMFCVisualManagerOffice2007` para alterar a aparência visual do seu aplicativo parecido com o Microsoft Office que 2007.  O gerenciador visual requer que você defina o estilo antes de usá\-la.  Antes que você passe o gerenciador visual a `CMFCVisualManager::SetDefaultManager`, você deve chamar a função estática `CMFCVisualManagerOffice2007::SetStyle`.
