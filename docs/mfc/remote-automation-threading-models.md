# Modelos de thread de automa&#231;&#227;o remota
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A automação remoto oferece suporte aos modelos de thread único e apartamento\- rosqueados COM.  Em plataformas de 32 bits e de 64 bits, que são tudo o que oferece suporte ao Visual C\+\+, o modelo de threading deve ser definido como “.” STA. Para obter mais informações sobre como e de onde fazer isso, consulte [Criando programas que usam a automação remoto](../mfc/creating-programs-that-use-remote-automation.md).
