# Depurando classes de suporte
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC O fornece as seguintes classes para ajudar a depurar problemas dinâmicos de alocação de memória.  
  
 [CDumpContext](../mfc/reference/cdumpcontext-class.md)  
 Fornece um destino para despejos de diagnóstico.  
  
 [CMemoryState](../mfc/reference/cmemorystate-structure.md)  
 Estrutura que fornece instantâneos o uso da memória.  Também usado para comparar instantâneos anteriores e posteriores de memória.
