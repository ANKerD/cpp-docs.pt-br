# <a name="ready-to-use-list-classes"></a>Classes de lista prontas para serem usadas
As classes a seguir são classes de lista prontas para uso.  
  
 [CObList](../mfc/reference/coblist-class.md)  
 Armazena ponteiros para objetos da classe `CObject` ou objetos de classes derivadas de `CObject` em uma lista vinculada.  
  
 [CPtrList](../mfc/reference/cptrlist-class.md)  
 Armazena ponteiros para `void` (ponteiros genéricos) em uma lista vinculada.  
  
 [CStringList](../mfc/reference/cstringlist-class.md)  
 Repositórios `CString` objetos em uma lista vinculada.
