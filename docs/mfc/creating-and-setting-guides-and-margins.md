# Criando e definindo guias e margens
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Se você estiver movendo controles, adicionando controles, ou reorganizar um layout atual, guias podem ajudar a alinhar os controles com precisão dentro de uma caixa de diálogo.  As guias aparecem como linhas pontilhadas azuis em toda a caixa de diálogo exibida no editor e as setas correspondentes nas réguas \(na parte superior e no lado esquerdo do editor de diálogo\).  
  
 Quando você cria uma caixa de diálogo, quatro margens são fornecidas.  As margens são as guias modificadas, aparecendo como linhas pontilhadas azuis.  
  
### Para criar uma guia  
  
1.  Dentro da régua, clique uma vez para criar uma guia.  \(Um clique cria um novo guia; clicar duas vezes em lançamentos a  [Caixa de diálogo de configurações do guia de](../Topic/Guide%20Settings%20Dialog%20Box.md) na qual você pode especificar configurações do guia.\)  
  
### Para definir um guia  
  
1.  Na caixa de diálogo, clique na guia e arraste\-o para uma nova posição.  \(Você também pode clicar na seta na régua para arrastar a guia associada\).  
  
     As coordenadas do guia são exibidas na barra de status na parte inferior da janela e na régua.  Mova o ponteiro sobre a seta na régua para exibir a posição exata do guia.  
  
### Para excluir uma guia  
  
1.  Arraste a guia para fora da caixa de diálogo.  
  
 \- ou \-  
  
-   Arraste a seta correspondente fora da régua.  
  
#### Para mover as margens  
  
1.  Arraste a margem para a nova posição.  
  
     Para tornar uma margem desaparecem, mova a margem para uma posição zero.  Para trazer de volta a margem, posicione o ponteiro sobre a margem a posição zero e mover a margem para a posição.
