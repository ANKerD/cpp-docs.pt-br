# Classes de desenho e impress&#227;o
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

No windows, todas as saídas gráficos são desenhados em uma área de desenho virtual chamada um contexto \(DC\) do dispositivo.  MFC O fornece classes para encapsular os vários tipos de DCS, bem como encapsulamento das ferramentas de desenho do windows como bitmaps, escovas, paletas, e penalidades.
