# Barra de ferramentas (editor de imagens para &#237;cones)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

O  **Editor de imagens** barra de ferramentas contém ferramentas de desenho, pintura, inserindo texto, apagar e manipular os modos de exibição.  Ele também contém um seletor de opção, com o qual você pode selecionar opções para usar cada ferramenta.  Por exemplo, você pode escolher a partir de várias larguras de pincel, fatores de ampliação e estilos de linha.  
  
> [!NOTE]
>  Todas as ferramentas disponíveis na  **Editor de imagens** barra de ferramentas também estão disponíveis no  **imagem**  menu \(sob o  **Ferramentas** comando\).  
  
 ![Barra de ferramentas do Editor de imagens do Visual Studio](../mfc/media/vcimageeditortoolbar.png "vcImageEditorToolbar")  
Barra de ferramentas do Editor de imagens  
  
 Para usar o  **Editor de imagens** barra de ferramentas e  **opção** selector, clique na ferramenta ou a opção que você deseja.  
  
> [!TIP]
>  Dicas de ferramentas aparecem quando você passa o cursor sobre um botão da barra de ferramentas.  Estas dicas podem ajudá\-lo a identificar a função de cada botão.  
  
 Com o  **opção** seletor, você pode especificar a largura de uma linha, o traçado de pincel, etc.  No ícone do  **opção** seletor botão é alterado dependendo da ferramenta que você selecionou.  
  
 ![Seletor de opções da barra de ferramentas do Editor de imagens](../mfc/media/vcimageeditortoolbaroptionselector.png "vcImageEditorToolbarOptionSelector")
