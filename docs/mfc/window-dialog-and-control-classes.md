# Classes de janela, caixa de di&#225;logo e controle
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A classe `CWnd` e suas classes derivadas encapsulam `HWND`, um identificador para uma janela do windows.  `CWnd` pode ser usado sozinho ou como uma base com novas classes.  As classes derivadas fornecidas pela biblioteca de classes representam vários tipos do windows.  
  
 [CWnd](../Topic/CWnd%20Class.md)  
 A classe base para todas as janelas.  Você pode usar uma das classes derivadas de `CWnd` ou derivar suas próprias classes diretamente deles.
