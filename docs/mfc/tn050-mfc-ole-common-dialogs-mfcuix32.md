# TN050: caixas de di&#225;logo comuns MFC/OLE (MFCUIx32)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Essa observação cobrisse originalmente alguns problemas e o futuro das caixas de diálogo comum com OLE DB fornecido e MFC usado pelo.  As caixas de diálogo comum com OLE DB são fornecidas agora como um componente interno ao sistema \(OLEDLG.DLL\) e documentadas completamente na documentação do produto.
