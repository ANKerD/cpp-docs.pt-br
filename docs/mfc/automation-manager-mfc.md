# <a name="automation-manager-mfc"></a>Gerenciador de automação (MFC)
AUTMGR32. EXE deve ser copiado para o diretório de sistema do Windows de cada máquina que pretende fornecer objetos de automação remota. É geralmente C:\WINDOWS\SYSTEM32.  
  
 Se você deseja habilitar os retornos de chamada do servidor para o cliente, esse arquivo executável também deve ser copiado para o diretório do sistema de cada computador cliente.  
  
 Quando o Gerenciador de automação é executado, ele exibe uma janela pequena na máquina do servidor que contém informações de resumo de status. Se você quiser ocultá-lo, consulte o artigo Q138067 na Base de dados de Conhecimento Microsoft.
