# Classe raiz: CObject
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A maioria das classes na biblioteca de \(MFC\) da classe do Microsoft são derivadas de uma única classe base na raiz da hierarquia da classe.  `CObject` fornece vários recursos úteis para todas as classes derivadas deles, com sobrecarga muito baixa.  Para obter mais informações sobre como `CObject` e seus recursos, consulte [Usando CObject](../mfc/using-cobject.md).  
  
 [CObject](../Topic/CObject%20Class.md)  
 A classe base final da maioria das classes MFC.  O oferece suporte que são serializados dados e que obtêm informações de tempo de execução sobre uma classe.  
  
 [CRuntimeClass](../Topic/CRuntimeClass%20Structure.md)  
 Estrutura usada para determinar em tempo de execução a classe precisa de um objeto.
