# TN049: MBCS MFC/OLE para camada de transla&#231;&#227;o Unicode (MFCANS32)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Esta nota originalmente descrita como MFCANS32. DLL fornece interfaces de ANSI no mundo de Unicode principalmente de OLE de 32 bits. Essa DLL não é mais usada pelo MFC.
