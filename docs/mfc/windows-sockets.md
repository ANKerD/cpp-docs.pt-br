# Windows Sockets
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Esta família de artigos abrange a implementação de MFC de soquetes do windows.  MFC O fornece duas classes para aplicativos de rede de programação de suporte com os soquetes apis do windows.  A classe [CAsyncSocket](../Topic/CAsyncSocket%20Class.md) encapsula soquetes API do windows um para um, fornecendo aos programadores avançadas de rede a maioria de flexibilidade e capacidade.  A classe [CSocket](../mfc/reference/csocket-class.md) fornece uma interface simplificada para serializar dados para e de um objeto de [CArchive](../mfc/reference/carchive-class.md) .
