# Servidores: emiss&#245;es de interface do usu&#225;rio
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Um aplicativo de servidor tem vários recursos que devem ser adicionados à interface de usuário para fornecer itens com OLE DB para aplicativos do contêiner.  Para obter mais informações sobre os menus e recursos adicionais que precisam ser adicionados a um aplicativo de servidor, consulte [Menus e recursos: Adições de servidor](../mfc/menus-and-resources-server-additions.md).
