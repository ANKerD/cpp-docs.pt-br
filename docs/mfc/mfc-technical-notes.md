# <a name="mfc-technical-notes"></a>Observações técnicas MFC
Uma nota técnica é um documento escrito para programadores por programadores.  
  
 Cada nota técnica descreve um problema ou um recurso que está além do escopo do restante da documentação do MFC. As notas técnicas fornecidas refletem solicitações para obter informações de usuários, bem como informações especializadas que os desenvolvedores MFC antecipar desejarão que os usuários avançados.  
  
 Há duas maneiras de examinar as notas técnicas:  
  
-   [Observações técnicas por número](../mfc/technical-notes-by-number.md)  
  
-   [Observações técnicas por categoria](../mfc/technical-notes-by-category.md)
