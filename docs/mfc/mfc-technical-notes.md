# Observa&#231;&#245;es t&#233;cnicas MFC
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Uma nota técnica é um documento gravado para programadores por programadores.  
  
 Cada observação técnica descreve um problema ou recurso que estejam além do escopo do restante da documentação MFC.  As notas técnicas fornecidas refletem os aplicativos de informações de usuários, bem como informações especializada que os desenvolvedores de MFC antecipem todas usuários avançados preciso.  
  
 Há duas maneiras de navegar pelas técnicas notas:  
  
-   [Notas técnicas pelo número](../mfc/technical-notes-by-number.md)  
  
-   [Notas técnicas por categoria](../mfc/technical-notes-by-category.md)
