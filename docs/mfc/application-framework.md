# Framework do aplicativo
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

O núcleo de \(MFC\) da biblioteca de classes do Microsoft foundation é uma encapsulamento de uma grande parte da API do windows no formato C\+\+.  As classes de biblioteca representam janelas, caixas de diálogo, contextos de dispositivo, objetos comuns de GDI como escovas e penalidades, controles, e outros itens padrão do windows.  Essas classes que fornecem uma interface conveniente da função de membro C\+\+ para as estruturas nas janelas que encapsulam.  Para saber mais sobre como usar essas classes, consulte [Tópicos do objeto da janela](../mfc/window-objects.md).  
  
 Mas a biblioteca MFC também fornece uma camada de funcionalidade extra do aplicativo criada em encapsulamento C\+\+ da API do windows.  Essa camada é uma estrutura de aplicativo de trabalho para o windows que fornece a maioria da interface do usuário \(esperada de programas do windows, incluindo barras de ferramentas, barras de status, impressão, a visualização de impressão, suporte da base de dados, e suporte ActiveX.  [Usando as classes para gravar aplicativos do windows](../Topic/Using%20the%20Classes%20to%20Write%20Applications%20for%20Windows.md) explica a estrutura em detalhes.
