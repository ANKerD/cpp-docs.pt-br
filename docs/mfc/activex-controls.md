# <a name="activex-controls"></a>Controles ActiveX
No Visual C++, você pode criar controles ActiveX usando o MFC ou ATL.  

>[!IMPORTANT]
> ActiveX é uma tecnologia herdada que não deve ser usada para novos desenvolvimentos. Muitos recursos de controles ActiveX podem ser executados de forma mais simples e muito mais segura com tecnologias modernas, como HTML5 e JavaScript, extensões de navegador moderno ou WebAssembly módulos. Para obter mais informações, consulte [uma ruptura com o passado, parte 2: adeus ao ActiveX, VBScript, attachEvent...](https://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/) e [de mensagens nativo](/microsoft-edge/extensions/guides/native-messaging) e [extensões do Microsoft Edge](/microsoft-edge/extensions) e [WebAssembly](https://webassembly.org/).
