# Respondendo &#224; troca din&#226;mica de dados (DDE)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A janela do quadro pode responder a solicitações de \(DDE\) de troca dinâmicas de dados para abrir arquivos do gerenciador de arquivos \(se a extensão de arquivo é registrada ou associada ao aplicativo\).  Consulte [Registro de Shell](../mfc/special-cwinapp-services.md).
