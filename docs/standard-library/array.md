# &lt;array&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Define a classe de modelo de contêiner `array` e vários modelos de suporte.  
  
## Sintaxe  
  
```  
#include <array>  
```

|-|-|

|[tuple\_size](../standard-library/tuple-size-class-array.md)|Encapsula o tamanho de um elemento da matriz.|
