# <a name="lttypeinfogt"></a>&lt;typeinfo&gt;

Inclua o cabeçalho padrão \<typeinfo > para definir vários tipos associados com o operador de identificação de tipo [operador typeid](../cpp/typeid-operator.md), que gera informações sobre os tipos estáticos e dinâmicos.

## <a name="syntax"></a>Sintaxe

```cpp
#include <typeinfo>

```
