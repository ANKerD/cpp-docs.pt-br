# &lt;typeinfo&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Inclui o cabeçalho padrão `<typeinfo>` para definir vários tipos associados ao operador [Operador typeid](../cpp/typeid-operator.md)de tipo ID, que retorna informações sobre tipos estáticos e dinâmicos.  
  
## Sintaxe  
  
```  
  
#include <typeinfo>  
  
```  
  
## Comentários  
 Para obter mais informações sobre classes definidas \<no typeinfo\>, consulte os seguintes tópicos:  
  
-   [Exceção bad\_cast](../cpp/bad-cast-exception.md)  
  
-   [Exceção bad\_typeid](../cpp/bad-typeid-exception.md)  
  
-   [Classe type\_info](../cpp/type-info-class.md)
