# <a name="ltscopedallocatorgt"></a>&lt;scoped_allocator&gt;
Define o scoped_allocator de classe de modelo do contêiner.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
#include <scoped_allocator>  
```
