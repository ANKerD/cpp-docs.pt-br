# &lt;bitset&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Define o bitset de classe de modelo e duas funções de modelo suporte para representar e manipular as sequências de tamanho fixo de bits.  
  
## Sintaxe  
  
```  
  
#include <bitset>  
  
```  
  
### Operadores  
  
|||  
|-|-|  
|[operator&](../Topic/operator&%20\(%3Cbitset%3E\).md)|Executa um AND bit a bit entre duas bitsets.|  
|[Operador \<\<](../Topic/operator%3C%3C%20\(%3Cbitset%3E\).md)|Insere uma representação de texto da seqüência de bits no fluxo de saída padrão.|  
|[Operador \>\>](../Topic/operator%3E%3E%20\(%3Cbitset%3E\).md)|Insere uma representação de texto da seqüência de bits no fluxo de entrada padrão.|  
|[operator^](../Topic/operator%5E%20\(%3Cbitset%3E\).md)|Executa um bit a bit exclusivo OR entre duas bitsets.|  
|[operator&#124;](../Topic/operator%7C%20\(%3Cbitset%3E\).md)|Executa um OR bit a bit entre duas bitsets.|  
  
### Classes  
  
|||  
|-|-|  
|[Classe bitset](../Topic/bitset%20Class.md)|A classe de modelo descreve um tipo de objeto que armazena uma seqüência que consiste em um número fixo de bits que fornecem uma maneira compact de manter os sinalizadores para um conjunto de itens ou condições.|
