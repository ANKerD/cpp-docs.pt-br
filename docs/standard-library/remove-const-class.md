# <a name="removeconst-class"></a>Classe remove_const

Cria um tipo não const do tipo.

## <a name="syntax"></a>Sintaxe

```cpp
template <class T>
struct remove_const;
```

```cpp
template <class T>
using remove_const_t = typename remove_const<T>::type;
```

### <a name="parameters"></a>Parâmetros

*T* para modificar o tipo.

## <a name="remarks"></a>Comentários

Uma instância do `remove_const<T>` mantém um tipo modificado que é `T1` quando *T* está no formato `const T1`; caso contrário, *T*.

## <a name="example"></a>Exemplo

```cpp
#include <type_traits>
#include <iostream>

int main()
    {
    int *p = (std::remove_const_t<const int>*)0;

    p = p;  // to quiet "unused" warning
    std::cout << "remove_const_t<const int> == "
        << typeid(*p).name() << std::endl;

    return (0);
    }
```

```Output
remove_const_t<const int> == int
```

## <a name="requirements"></a>Requisitos

**Cabeçalho:** \<type_traits>

**Namespace:** std
