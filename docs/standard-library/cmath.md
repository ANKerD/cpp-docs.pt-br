# &lt;cmath&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Inclui o cabeçalho da biblioteca C padrão \< Math \> e adiciona os nomes associados ao `std` namespace.  
  
## Sintaxe  
  
```  
  
#include <cmath>  
  
```  
  
## Comentários  
 Inclusão desse cabeçalho garante que os nomes declarados usando vinculação externa no cabeçalho da biblioteca C padrão sejam declarados no `std` namespace.
