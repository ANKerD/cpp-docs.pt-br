# &lt;forward_list&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Define o forward\_list da classe do modelo do contêiner e vários modelos de suporte.  
  
## Sintaxe  
  
```  
#include <forward_list>  
```
