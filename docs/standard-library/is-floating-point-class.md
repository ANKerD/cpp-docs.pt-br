# Classe is_floating_point
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Testa se o tipo é de ponto flutuante.  
  
## Sintaxe  
  
```  
template<class Ty>  
    struct is_floating_point;  
```  
  
#### Parâmetros  
 `Ty`  
 O tipo a ser consultado.  
  
## Comentários  
 Uma instância do predicado do tipo manterá true se o tipo `Ty` é flutuante tipo ponto ou uma `cv-qualified` tipo de formulário de ponto flutuante, caso contrário, manterá false.  
  
 Um tipo de ponto flutuante é um dos `float`, `double`, ou `long double`.  
  
## Exemplo  
  
```  
// std_tr1__type_traits__is_floating_point.cpp   
// compile with: /EHsc   
#include <type_traits>   
#include <iostream>   
  
struct trivial   
    {   
    int val;   
    };   
  
int main()   
    {   
    std::cout << "is_floating_point<trivial> == " << std::boolalpha   
        << std::is_floating_point<trivial>::value << std::endl;   
    std::cout << "is_floating_point<int> == " << std::boolalpha   
        << std::is_floating_point<int>::value << std::endl;   
    std::cout << "is_floating_point<float> == " << std::boolalpha   
        << std::is_floating_point<float>::value << std::endl;   
  
    return (0);   
    }  
  
```  
  
  **is\_floating\_point \< trivial \> \= \= false**  
**is\_floating\_point \< int \> \= \= false**  
**is\_floating\_point \< float \> \= \= true**   
## Requisitos  
 **Cabeçalho:** \<type\_traits\>
