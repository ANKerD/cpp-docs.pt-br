# Classe de Cont&#234;iner::reverse_iterator
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  Este tópico está na documentação do Visual C\+\+ como um exemplo nonfunctional contêineres usados na biblioteca padrão C\+\+.  Para obter mais informações, consulte [Contêineres STL](../standard-library/stl-containers.md).  
  
 Descreve um objeto que pode servir como um iterador de cima para baixo para a sequência controlada.  
  
## Sintaxe  
  
```  
  
typedef T7 reverse_iterator;  
  
```  
  
## Comentários  
 É descrito aqui como um sinônimo para o tipo não for especificado **T7** \(normalmente `reverse_iterator` **\<**[iterador](../standard-library/container-class-iterator.md)**\>**\).
