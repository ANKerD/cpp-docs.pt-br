# Struct char_traits&lt;char&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Uma estrutura que é uma especialização da estrutura **char\_traits\<CharType\>** do modelo a um elemento do tipo `char`.  
  
## Sintaxe  
  
```  
template<> struct char_traits<char>;  
```  
  
## Comentários  
 A especialização permite que o struct aproveitamento das funções da biblioteca que manipulam objetos desse tipo `char`.  
  
## Exemplo  
 Consulte os typedefs e as funções de membro da classe [classe de char\_traits](../standard-library/char-traits-struct.md)\<**CharType**\> do modelo para exemplos que envolvem os elementos do tipo `char`.  
  
## Requisitos  
 **Cabeçalho:** \<cadeia de caracteres\>  
  
 **Namespace:** std
