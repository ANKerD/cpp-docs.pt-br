# Classe de Cont&#234;iner::const_iterator
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  Este tópico está na documentação do Visual C\+\+ como um exemplo nonfunctional contêineres usados na biblioteca padrão C\+\+.  Para obter mais informações, consulte [Contêineres STL](../standard-library/stl-containers.md).  
  
 Descreve um objeto que pode servir como um iterador constante para a sequência controlada.  
  
## Sintaxe  
  
```  
  
typedef T6 const_iterator;  
  
```  
  
## Comentários  
 É descrito aqui como um sinônimo para o tipo não for especificado **T6**.
