# <a name="ltcinttypesgt"></a>&lt;cinttypes&gt;
Inclui o cabeçalho \<inttypes.h> da biblioteca C Padrão. A inclusão desse cabeçalho também inclui \<cstdint>.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
#include <cinttypes>  
  
```  
  
## <a name="remarks"></a>Comentários  
 A inclusão desse cabeçalho garante que os nomes declarados usando vinculação externa no cabeçalho da biblioteca C Padrão sejam declarados no namespace `std`.
