# &lt;cvt/wbuffer&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

O cabeçalho `<cvt/wstring>` nas versões anteriores do Visual Studio definido a classe de modelo [Classe wbuffer\_convert](../standard-library/wbuffer-convert-class.md) no namespace stdext::cvt.  O cabeçalho é mantido para compatibilidade com versões anteriores.  Novo código deve usar a versão da classe que é definida em [\<locale\>](../standard-library/locale.md) no `std` namespace  
  
## Sintaxe  
  
```  
  
#include <cvt/wbuffer>  
  
```
