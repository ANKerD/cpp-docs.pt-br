# Onde posso encontrar artigos da Base de Dados de Conhecimento Microsoft sobre t&#243;picos de banco de dados?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Para localizar artigos da Base de Dados de Conhecimento sobre tópicos MFC ODBC, pesquise a base de conhecimento pela palavra\-chave MFCDATABASE.  
  
 Para localizar artigos da Base de Dados de Conhecimento sobre tópicos MFC DAO, pesquise a base de conhecimento pela palavra\-chave MFCDAO.
