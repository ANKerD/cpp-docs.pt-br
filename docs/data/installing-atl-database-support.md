# Instalando suporte do banco de dados ATL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A instalação instala todos os componentes necessários para oferecer suporte OLE DB que programa como parte da instalação de linha de base.  As interfaces OLE DB são instaladas como parte do MDAC SDK.  Os modelos OLE DB, OLE DB que dão suporte à programação no Visual C\+\+ .NET, são instalados como parte de biblioteca ATL.  A configuração do Visual C\+\+ .NET instala o SDK do MDAC \(versão 2,8\) e a biblioteca ATL de.
