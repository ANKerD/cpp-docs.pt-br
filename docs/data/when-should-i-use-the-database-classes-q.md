# Quando devo usar as classes de banco de dados?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Muitos aplicativos requerem armazenamento de dados em um base de dados, e muitos outros aplicativos podem se beneficiar do uso de um base de dados.  Um base de dados fornece um repositório de dados flexível que pode ser acessado, em muitos casos, por vários usuários e por vários aplicativos.  As bases de dados podem armazenar grandes quantidades de dados e fornecer acesso a dados rápida para consultas e atualizações.
