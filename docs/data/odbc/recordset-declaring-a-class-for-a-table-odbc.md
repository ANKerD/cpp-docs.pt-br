# Conjunto de registros: declarando uma classe para uma tabela (ODBC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Este tópico se aplica às classes ODBC do MFC.  
  
 A classe mais comuns do conjunto de registros abre uma única tabela.  Para declarar uma classe do conjunto de registros para uma única tabela, usar [Assistente do consumidor MFC ODBC](../../mfc/reference/adding-an-mfc-odbc-consumer.md) de **Adicionar Classe** e escolher cada coluna que você deseja nomeando um membro de dados correspondente do campo do conjunto de registros.  
  
 Outros usos para conjuntos de registros incluem:  
  
-   Adicionando duas ou mais tabelas.  
  
-   Que contém os resultados de uma consulta predefinida.
