# ODBC: configurando uma fonte de dados ODBC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Para usar [fonte de dados](../../data/odbc/data-source-odbc.md) com um aplicativo que você desenvolveu, você deve usar o administrador de ODBC para fazê\-lo.  O administrador ODBC mantém acompanha de fontes de dados disponíveis e de suas informações de conexão no Registro do Windows.  Use o administrador ODBC para adicionar, modificar, excluir e fontes de dados na caixa de diálogo de **Fontes de Dados** e para adicionar e excluir drivers ODBC.  
  
> [!NOTE]
>  Essas informações são aplicadas quando você usa classes de \(DAO\) do objeto de acesso a dados de MFC para acesso de ODBC e quando você usar classes MFC ODBC.  
  
 O administrador ODBC é instalado automaticamente com o suporte da base de dados \(MFC\) da biblioteca de classes do Microsoft.  Para obter mais informações sobre o programa do administrador de ODBC, consulte [Administrador ODBC](../../data/odbc/odbc-administrator.md) e o sistema de ajuda online de referência de ODBC API.  
  
 Para obter informações sobre como escrever programas de configuração e administração de ODBC para aplicativos de base de dados de MFC,[Observação 48 técnica](../../mfc/tn048-writing-odbc-setup-and-administration-programs.md).
