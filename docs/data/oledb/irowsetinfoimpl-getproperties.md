# <a name="irowsetinfoimplgetproperties"></a>IRowsetInfoImpl::GetProperties
Retorna as atuais configurações de propriedades no **DBPROPSET_ROWSET** grupo.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
      STDMETHOD (GetProperties )(const ULONG cPropertyIDSets,  
   const DBPROPIDSET rgPropertyIDSets[],  
   ULONG* pcPropertySets,  
   DBPROPSET** prgPropertySets);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 Consulte [irowsetinfo:: GetProperties](https://msdn.microsoft.com/en-us/library/ms719611.aspx) no *referência do programador de OLE DB*.  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** atldb.h
