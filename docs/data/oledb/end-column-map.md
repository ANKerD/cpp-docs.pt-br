# <a name="endcolumnmap"></a>END_COLUMN_MAP
Marca o fim das entradas de mapa de coluna.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
END_COLUMN_MAP()  
  
```  
  
## <a name="remarks"></a>Comentários  
 Ele é usado com um único acessador em um conjunto de linhas. O `BEGIN_COLUMN_MAP` macro é concluída com o `END_COLUMN_MAP` macro.  
  
## <a name="example"></a>Exemplo  
 Consulte [BEGIN_COLUMN_MAP](../../data/oledb/begin-column-map.md).
