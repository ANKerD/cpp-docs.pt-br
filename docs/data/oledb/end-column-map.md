# END_COLUMN_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Marca o término das entradas da coluna.  
  
## Sintaxe  
  
```  
  
END_COLUMN_MAP( )  
  
```  
  
## Comentários  
 É usado com um único acessador em um conjunto de linhas.  A macro de `BEGIN_COLUMN_MAP` é concluído com a macro de `END_COLUMN_MAP` .  
  
## Exemplo  
 Consulte [BEGIN\_COLUMN\_MAP](../Topic/BEGIN_COLUMN_MAP.md).  
  
## Requisitos  
 **Header:** atldbcli.h
