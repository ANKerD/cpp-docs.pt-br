# <a name="cdbpropidsetoperator-"></a>CDBPropIDSet::operator =

## <a name="syntax"></a>Sintaxe  
  
```  
  
      CDBPropIDSet& operator =(   
   CDBPropIDSet& propset    
) throw( );  
```
