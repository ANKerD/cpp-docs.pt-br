# <a name="cenumeratoraccessormntype"></a>CEnumeratorAccessor::m_nType
Uma variável que indica se a linha descreve uma fonte de dados ou um enumerador.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
USHORT m_nType;  
  
```
