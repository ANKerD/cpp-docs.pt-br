# Classe CMultipleResults
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Se você desejar um comando tratar vários conjuntos de resultados, use `CMultipleResults` para o argumento *TMultiple*do modelo de `CCommand` .  
  
## Sintaxe  
  
```  
class CMultipleResults  
```  
  
## Comentários  
 Para tratar vários conjuntos de resultados, [CCommand](../../data/oledb/ccommand-class.md) deve herdar dessa classe.
