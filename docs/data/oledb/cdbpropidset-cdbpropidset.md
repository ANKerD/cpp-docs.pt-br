# <a name="cdbpropidsetcdbpropidset"></a>CDBPropIDSet::CDBPropIDSet
O construtor. Inicializa o **rgProperties**, **cProperties**e (opcionalmente) **guidPropertySet** campos do [DBPROPIDSET](https://msdn.microsoft.com/en-us/library/ms717981.aspx) estrutura.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
      CDBPropIDSet(const GUID& guid);  

CDBPropIDSet(const CDBPropIDSet& propidset);  

CDBPropIDSet();  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `guid`  
 [in] Um GUID usado para inicializar o **guidPropertySet** campo.  
  
 *propidset*  
 [in] Outro `CDBPropIDSet` objeto para a construção de cópia.  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** atldbcli.h
