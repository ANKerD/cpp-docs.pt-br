# <a name="creating-a-project-for-the-provider"></a>Criando um projeto para o provedor
#### <a name="to-create-a-project-in-which-the-ole-db-provider-will-reside"></a>Para criar um projeto que reside em que o provedor OLE DB  
  
1.  Do **arquivo** menu, clique em `New`e, em seguida, clique em **projeto**.  
  
     A caixa de diálogo **Novo Projeto** é exibida.  
  
2.  No painel tipos de projeto, clique na pasta de projetos do Visual C++. No painel modelos, clique em **projeto ATL**.  
  
3.  No **nome** caixa, digite um nome para o projeto e, em seguida, clique em **Okey**.  
  
     O Assistente de projeto de ATL é exibida.  
  
4.  No Assistente de projeto de ATL, escolha **biblioteca de vínculo dinâmico (DLL)** para **tipo de servidor**.  
  
5.  Clique em **Finalizar**.
