# Criando um projeto para o provedor
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

#### Para criar um projeto do no qual o provedor OLE DB residirá  
  
1.  No menu de **Arquivo** , clique em `New`, e clique em **Projeto**.  
  
     A caixa de diálogo **Novo Projeto** é exibida.  
  
2.  No painel tipos de projeto, clique na pasta projetos do Visual C\+\+.  No painel modelos, clique **Projeto do ATL**.  
  
3.  Na caixa de **Nome** , digite um nome para o projeto e, em seguida **OK**.  
  
     O assistente de projeto de ATL é exibida.  
  
4.  No assistente de projeto de ATL, escolha **Dynamic\-Link Library \(DLL\)** para **Tipo de servidor**.  
  
5.  Clique em **Concluir**.
