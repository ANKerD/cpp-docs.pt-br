# CDynamicAccessor::GetColumnCount
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Recupera o número de colunas.  
  
## Sintaxe  
  
```  
  
DBORDINAL GetColumnCount( ) const throw( );  
  
```  
  
## Valor de retorno  
 O número de colunas recuperadas.

 **Header:** atldbcli.h
