# END_PROPERTY_SET
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Marca o fim de um conjunto de propriedades.  
  
## Sintaxe  
  
```  
  
END_PROPERTY_SET(  
guid   
)  
  
```  
  
#### Parâmetros  
 `guid`  
 \[in\] a propriedade GUID.  
  
## Exemplo  
 Consulte [BEGIN\_PROPSET\_MAP](../Topic/BEGIN_PROPSET_MAP.md).  
  
## Requisitos  
 **Header:** atldb.h
