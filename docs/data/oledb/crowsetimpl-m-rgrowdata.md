# <a name="crowsetimplmrgrowdata"></a>CRowsetImpl::m_rgRowData
Por padrão, um `CAtlArray` que templatizes no argumento de modelo de registro de usuário `CRowsetImpl`.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
ArrayType CRowsetBaseImpl::m_rgRowData;  
  
```  
  
## <a name="remarks"></a>Comentários  
 **ArrayType** é um parâmetro de modelo para `CRowsetImpl`.  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** atldb.h
