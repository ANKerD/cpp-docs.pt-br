# CEnumeratorAccessor::m_szParseName
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Cadeia de caracteres a ser passada a [IParseDisplayName](http://msdn.microsoft.com/library/windows/desktop/ms680604) para obter um moniker para a fonte de dados ou o enumerador.  
  
## Sintaxe  
  
```  
  
WCHAR m_szParseName[129];  
  
```  
  
## Comentários  
 Consulte [ISourcesRowset::GetSourcesRowset](https://msdn.microsoft.com/en-us/library/ms711200.aspx)*na referência do programador de OLE DB* para obter mais informações.
