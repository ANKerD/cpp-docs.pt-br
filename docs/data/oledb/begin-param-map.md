# <a name="beginparammap"></a>BEGIN_PARAM_MAP
Marca o início das entradas do mapa de parâmetro.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
BEGIN_PARAM_MAP(x)  
  
```  
  
#### <a name="parameters"></a>Parâmetros  
 *x*  
 [in] O nome da classe de registro de usuário.  
  
## <a name="remarks"></a>Comentários  
 Parâmetros são usados pelo [comandos](https://msdn.microsoft.com/en-us/library/ms724608.aspx).  
  
## <a name="example"></a>Exemplo  
 Consulte o exemplo para o [BEGIN_COLUMN_MAP](../../data/oledb/begin-column-map.md) macro.
