# M&#233;todo ComPtr::InternalAddRef
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Sintaxe  
  
```  
void InternalAddRef() const;  
```  
  
## Comentários  
 Esse método é protegido.
