# public (C++ Attributes)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Garante que um typedef irá para a biblioteca de tipo, mesmo que não é referenciado no dentro do arquivo. idl.  
  
## Sintaxe  
  
```  
  
[public]  
  
```  
  
## Comentários  
 O  **pública** o atributo do C\+\+ tem a mesma funcionalidade que o  [pública](http://msdn.microsoft.com/library/windows/desktop/aa367150) atributo MIDL.  
  
## Exemplo  
 O código a seguir mostra como usar o  **pública** atributo:  
  
```  
// cpp_attr_ref_public.cpp  
// compile with: /LD  
#include "unknwn.h"  
[module(name="ATLFIRELib")];  
[export, public] typedef long MEMBERID;  
  
[dispinterface, uuid(99999999-9999-9999-9999-000000000000)]  
__interface IFireTabCtrl : IDispatch  
{  
   [id(2)] long procedure ([in, optional] VARIANT i);  
};  
```
