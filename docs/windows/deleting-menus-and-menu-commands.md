# <a name="deleting-menus-and-menu-commands"></a>Excluindo menus e comandos de menu
### <a name="to-delete-a-menu-or-menu-command"></a>Para excluir um menu ou comando de menu  
  
1.  Clique no nome do menu ou comando.  
  
2.  Escolha **excluir** no menu de atalho.  
  
    > [!NOTE]
    >  Da mesma forma, você pode usar o menu de atalho para executar outras ações, como copiar, recortar, colar, novo, inserir separador de inserir, IDs de editar, exibir como pop-up, verificar mnemônico, etc.  
  
 Para obter informações sobre como adicionar recursos a projetos gerenciados, consulte [recursos em aplicativos de área de trabalho](/dotnet/framework/resources/index) no *guia do desenvolvedor do .NET Framework.*  
  
 Requisitos  
  
 Win32
