# <a name="control"></a>Controle 

Especifica que o tipo definido pelo usuário é um controle.

## <a name="syntax"></a>Sintaxe

```cpp
[control]
```

## <a name="remarks"></a>Comentários

O **controle** atributo implica a [coclass](../windows/coclass.md) atributo. O **controle** atributo C++ tem a mesma funcionalidade que o [controle](/windows/desktop/Midl/control) atributo MIDL.

## <a name="example"></a>Exemplo

```cpp
// cpp_attr_ref_control.cpp
// compile with: /LD
#include <windows.h>
[module(name="Test", control=true)];

[object, uuid("9e66a290-4365-11d2-a997-00c04fa37ddb")]
__interface ICustom {
   HRESULT Custom([in] long l, [out, retval] long *pLong);
};

[coclass, control, appobject, uuid("9e66a294-4365-11d2-a997-00c04fa37ddb")]
class CTest : public ICustom {};
```
