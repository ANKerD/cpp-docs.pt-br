# propget
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Especifica uma função de assessor de propriedade.  
  
## Sintaxe  
  
```  
  
[propget]  
  
```  
  
## Comentários  
 O  **propget** o atributo do C\+\+ tem a mesma funcionalidade que o  [propget](http://msdn.microsoft.com/library/windows/desktop/aa367145) atributo MIDL.
