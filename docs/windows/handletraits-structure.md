# <a name="handletraits-structure"></a>Estrutura HANDLETraits

```cpp
struct HANDLETraits;
```

|Nome|Descrição|
|----------|-----------------|
|`Type`|Um sinônimo de identificador.|

|Nome|Descrição|
|----------|-----------------|
|[Método HANDLETraits::Close](../windows/handletraits-close-method.md)|Fecha o identificador especificado.|
