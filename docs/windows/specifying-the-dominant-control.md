# Especificando o controle dominante
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Primeiro, o controle selecionado é o controle dominante.  
  
### Para especificar o controle dominante  
  
1.  Mantenha pressionada a  **CTRL** chave e clique no controle que você deseja usar para influenciar o tamanho ou local de outros controles  *primeiro*.  
  
     **Nota**  as alças de dimensionamento do controle dominante são sólidas, enquanto as alças dos controles subordinadas são vazios.  Tudo ainda mais o redimensionamento ou alinhamento baseia\-se no controle dominante.  
  
### Para alterar o controle dominante  
  
1.  Desmarque a seleção atual clicando fora todos os controles atualmente selecionados.  
  
2.  Repita o procedimento anterior, selecionando um controle diferente primeiro.
