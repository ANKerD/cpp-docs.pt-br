# <a name="comptrcomptr-destructor"></a>Destruidor ComPtr::~ComPtr
Realiza o desligamento de uma instância do **ComPtr**.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
WRL_NOTHROW ~ComPtr();  
```  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** client.h  
  
 **Namespace:** Microsoft::WRL
