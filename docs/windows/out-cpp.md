# <a name="out-c"></a>out (C++)
Identifica os parâmetros de ponteiro retornados do procedimento chamado para o procedimento de chamada (do servidor para o cliente).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
[out]  
  
```  
  
## <a name="remarks"></a>Comentários  
 O **out** atributo C++ tem a mesma funcionalidade que o [out](http://msdn.microsoft.com/library/windows/desktop/aa367136) atributo MIDL.  
  
## <a name="example"></a>Exemplo  
 Consulte o exemplo para [associável](../windows/bindable.md) para uso do exemplo **out**.
