# Editando arquivos de recurso gerenciados
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Arquivos de recurso gerenciado \(. resx\) são arquivos XML.  Quando você adiciona um arquivo de recurso gerenciado a seu projeto a partir do  **Add New Item** caixa de diálogo, o  **Editor de recursos gerenciados** abre por padrão.  
  
 Para obter mais informações, consulte  [recursos no.Formato de arquivo resx](http://msdn.microsoft.com/pt-br/0c476133-87e4-47e8-b0ef-4b88f4ef3dc5).
