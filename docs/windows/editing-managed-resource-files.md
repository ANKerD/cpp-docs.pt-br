# <a name="editing-managed-resource-files"></a>Editando arquivos de recurso gerenciados
Arquivos de recurso gerenciado (. resx) são arquivos XML. Quando você adiciona um arquivo de recurso gerenciado ao seu projeto do **Adicionar Novo Item** caixa de diálogo, o **Editor de recursos gerenciados** é aberta por padrão.  
  
 Para obter mais informações, consulte [recursos. Formato de arquivo resx](http://msdn.microsoft.com/en-us/0c476133-87e4-47e8-b0ef-4b88f4ef3dc5).
