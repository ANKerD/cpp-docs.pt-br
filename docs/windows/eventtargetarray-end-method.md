# M&#233;todo EventTargetArray::End
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Oferece suporte à infraestrutura de WRL e não deve ser usado diretamente no seu código.  
  
## Sintaxe  
  
```  
ComPtr<IUnknown>* End();  
```  
  
## Valor de retorno  
 O endereço do elemento o mais recente na matriz interna dos manipuladores de eventos.  
  
## Comentários  
 Obtém o endereço do elemento o mais recente na matriz interna dos manipuladores de eventos.  
  
## Requisitos  
 **Cabeçalho:** event.h  
  
 **Namespace:** Microsoft::WRL::Details
