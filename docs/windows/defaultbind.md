# defaultbind
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Indica a propriedade única e vinculável que melhor representa o objeto.  
  
## Sintaxe  
  
```  
  
[defaultbind]  
  
```  
  
## Comentários  
 O  **defaultbind** o atributo do C\+\+ tem a mesma funcionalidade que o  [defaultbind](http://msdn.microsoft.com/library/windows/desktop/aa366790) atributo MIDL.  
  
## Exemplo  
 Consulte o exemplo para  [vinculáveis](../windows/bindable.md) para obter um exemplo de como usar  **defaultbind**.
