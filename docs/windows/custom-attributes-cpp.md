# Custom Attributes (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Um atributo personalizado é uma técnica com rigidez de tipos que permite ao usuário estender os metadados.  Este tópico lida com o uso de atributos: especificando parâmetros e destinos.  
  
 Para informações sobre como definir os novos atributos personalizados, consulte,
