# Redimensionando uma imagem inteira (editor de imagens para &#237;cones)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Para redimensionar uma imagem inteira, usando a janela Propriedades  
  
1.  Abra a imagem cujas propriedades você deseja alterar.  
  
2.  No  **largura** e  **altura** caixas no  [janela de propriedades](../Topic/Properties%20Window.md), digite as dimensões desejadas.  
  
     Se você está aumentando o tamanho da imagem, o editor de imagens amplia a imagem à direita, para baixo, ou ambos e preenche a nova região com a cor de plano de fundo atual.  A imagem não é alargada.  
  
     Se você está diminuindo o tamanho da imagem, o editor de imagens corta a imagem na borda inferior ou direita ou ambos.  
  
    > [!NOTE]
    >  Você pode usar as propriedades Width e Height para redimensionar apenas a imagem inteira, não para redimensionar uma seleção parcial.
