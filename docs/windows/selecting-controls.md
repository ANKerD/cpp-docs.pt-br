# Selecionando controles
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Selecionar controles para o tamanho, alinhar, mover, copiar, ou excluí\-los e, em seguida, executar a operação desejada.  Na maioria dos casos, você precisa selecionar mais de um controle para usar as ferramentas de dimensionamento e alinhamento a  [barra de ferramentas do Editor de diálogo](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md).  
  
 Quando um controle é selecionado, ele tem uma borda sombreada ao redor dele com sólido \(ativo\) ou vazio \(inativo\) "alças," pequenos quadrados que aparecem na borda de seleção.  Quando vários controles selecionados, o controle dominante possui alças de dimensionamento sólida; todos os outros controles selecionados têm alças de dimensionamento vazio.  
  
 Quando você estiver de dimensionamento ou alinhar vários controles, o editor de diálogo usa o "controle dominante" para determinar como os outros controles são dimensionados ou alinhados.  Por padrão, o controle dominante é o primeiro controle selecionado.  
  
-   [Selecionar vários controles](../mfc/selecting-multiple-controls.md)  
  
-   [Especificando o controle dominante](../mfc/specifying-the-dominant-control.md)
