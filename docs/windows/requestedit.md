# requestedit
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Indica que a propriedade oferece suporte a  **OnRequestEdit** notificação.  
  
## Sintaxe  
  
```  
  
[requestedit]  
  
```  
  
## Comentários  
 O  **requestedit** o atributo do C\+\+ tem a mesma funcionalidade que o  [requestedit](http://msdn.microsoft.com/library/windows/desktop/aa367155) atributo MIDL.  
  
## Exemplo  
 Consulte o exemplo para  [vinculáveis](../windows/bindable.md) para uso do exemplo  **requestedit**.
