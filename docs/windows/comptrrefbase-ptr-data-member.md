# Membro de dados ComPtrRefBase::ptr_
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Oferece suporte à infraestrutura de WRL e não deve ser usado diretamente no seu código.  
  
## Sintaxe  
  
```  
T* ptr_;  
```  
  
## Comentários  
 Ponteiro para o tipo especificado pelo parâmetro atual do modelo.  
  
## Comentários  
 `ptr_` é o membro de dados protegido.  
  
## Requisitos  
 **Cabeçalho:** client.h  
  
 **Namespace:** Microsoft::WRL::Details
