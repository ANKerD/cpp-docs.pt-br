# Attribute Categories
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A referência de atributos descreve os vários tipos de atributos que você pode usar no Visual C\+\+.  
  
## Seções relacionadas  
 [Atributos COM](../Topic/COM%20Attributes.md)  
 Insere um código para dar suporte a diversas áreas de desenvolvimento COM.  
  
 [Atributos IDL](../windows/idl-attributes.md)  
 Insere um código que cria ou modifica o arquivo. idl associado a partir de um arquivo de código\-fonte sem um assistente e sem estar familiarizado com a estrutura e sintaxe desse arquivo.  
  
 [Atributos do OLE DB consumidor](../windows/ole-db-consumer-attributes.md)  
 Insere um código, com base no  [OLE DB consumidor modelos](../data/oledb/ole-db-consumer-templates-reference.md), para criar um trabalho OLE DB consumidor que realiza tarefas como, por exemplo, tabelas de abertura, execução de comandos e acesso a dados.  
  
 [Atributos do compilador](../windows/compiler-attributes.md)  
 Insere um código que executa uma variedade de tarefas comuns de programação.  
  
 [Atribuído em grande parte de programação](../windows/attributed-programming-concepts.md)  
 Fornece links para tópicos conceituais que descrevem atribuído em grande parte de programação.
