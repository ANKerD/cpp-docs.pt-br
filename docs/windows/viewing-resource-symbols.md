# <a name="viewing-resource-symbols"></a>Exibindo símbolos de recurso
### <a name="to-view-resource-symbols"></a>Para exibir os símbolos de recurso  
  
1.  Em [exibição recursos](../windows/resource-view-window.md), clique em seu arquivo. rc.  
  
    > [!NOTE]
    >  Se o projeto já não contiver um arquivo. RC, consulte [criar um novo arquivo de Script de recurso](../windows/how-to-create-a-resource-script-file.md).  
  
2.  Selecione **símbolos de recurso** no menu de atalho para exibir uma tabela de símbolos de recurso no [caixa de diálogo símbolos de recurso](../windows/resource-symbols-dialog-box.md).  
  
    > [!NOTE]
    >  Para ver os símbolos predefinidos, verifique o **Mostrar símbolos somente leitura** caixa de seleção.  
  

  
 **Requisitos**  
  
 Win32
