# <a name="hstringrelease-method"></a>Método HString::Release
Exclui o valor de cadeia de caracteres subjacente e inicializa o atual **HString** objeto para um valor vazio.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
void Release() throw()  
```  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** corewrappers. h  
  
 **Namespace:** Microsoft::WRL::Wrappers
