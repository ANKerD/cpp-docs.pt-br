# Editando tabelas de aceleradores
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

No Visual C\+\+, você pode editar uma tabela de aceleração diretamente com a edição no local no editor de acelerador.  
  
 Consultem os procedimentos a seguir o uso de páginas de propriedades padrão, no entanto, a edição no local e o método de página de propriedade têm o mesmo resultado.  As alterações feitas usando páginas de propriedades ou usando a edição in\-loco são imediatamente refletidas na tabela accelerator.  
  
-   [A edição de uma tabela de aceleração](../windows/editing-in-an-accelerator-table.md)  
  
-   [Localizar uma entrada em uma tabela de aceleração](../windows/finding-an-entry-in-an-accelerator-table.md)  
  
-   [Adicionando uma entrada para uma tabela de aceleração](../windows/adding-an-entry-to-an-accelerator-table.md)  
  
-   [Excluindo uma entrada de uma tabela de aceleração](../Topic/Deleting%20an%20Entry%20From%20an%20Accelerator%20Table.md)  
  
-   [Mover ou copiar uma entrada de tabela do acelerador](../windows/moving-or-copying-an-accelerator-table-entry-to-another-resource-script-file.md)  
  
-   [Alterando as propriedades de várias teclas de aceleração](../windows/changing-the-properties-of-multiple-accelerator-keys.md)  
  
 Para obter informações sobre como adicionar recursos a projetos gerenciados, consulte  [recursos em aplicativos](../Topic/Resources%20in%20Desktop%20Apps.md) na  *.Guia. do desenvolvedor do NET Framework* Para informações sobre como adicionar manualmente os arquivos de recursos para projetos gerenciados, acesso a recursos, exibindo recursos estáticos e a atribuição de seqüências de caracteres de recursos a propriedades, consulte  [Walkthrough: Localizing Windows Forms](http://msdn.microsoft.com/pt-br/9a96220d-a19b-4de0-9f48-01e5d82679e5) e [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
