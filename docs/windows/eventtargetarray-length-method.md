# <a name="eventtargetarraylength-method"></a>Método EventTargetArray::Length
Oferece suporte à infraestrutura WRL e não se destina a ser usado diretamente no seu código.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
size_t Length();  
```  
  
## <a name="return-value"></a>Valor de retorno  
 O número atual de elementos da matriz interna de manipuladores de eventos.  
  
## <a name="remarks"></a>Comentários  
 Obtém o número atual de elementos da matriz interna de manipuladores de eventos.  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** event.h  
  
 **Namespace:** Microsoft::WRL::Details
