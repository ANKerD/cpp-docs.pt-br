# <a name="new-custom-resource-dialog-box"></a>Caixa de diálogo Novo Recurso Personalizado
Permite que você crie um novo recurso personalizado.  
  
### <a name="resource-type"></a>Tipo de recurso  
 Fornece uma caixa de texto para que você insira o nome de um tipo de recurso personalizado. Visual C++ escreve o nome automaticamente quando você sair do **novo recurso personalizado** caixa de diálogo.  
  
## <a name="remarks"></a>Comentários  
 Não especifique um nome de recurso ou ID é reservado pelo [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]. Os nomes reservados são DESIGNINFO, HWB, e TEXTINCLUDE e o ID reservado é 255.
