# Caixa de di&#225;logo Novo Recurso Personalizado
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Permite que você criar um novo recurso personalizado.  
  
 **Tipo de recurso**  
 Fornece uma caixa de texto para você digitar o nome de um tipo de recurso personalizado.  Visual C\+\+ explora o nome automaticamente quando você sair do  **Novo recurso personalizado** caixa de diálogo.  
  
## Comentários  
 Não especificar um nome de recurso ou [!INCLUDE[TLA2#tla_id](../windows/includes/tla2sharptla_id_md.md)] que é reservado por [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  Os nomes reservados são DESIGNINFO, HWB e TEXTINCLUDE e o reservado [!INCLUDE[TLA2#tla_id](../windows/includes/tla2sharptla_id_md.md)] é 255.
