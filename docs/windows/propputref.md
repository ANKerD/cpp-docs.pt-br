# propputref
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Especifica uma função de configuração de propriedade que usa uma referência em vez de um valor.  
  
## Sintaxe  
  
```  
  
[propputref]  
  
```  
  
## Comentários  
 O  **propputref** o atributo do C\+\+ tem a mesma funcionalidade que o  [propputref](http://msdn.microsoft.com/library/windows/desktop/aa367147) atributo MIDL.  
  
## Exemplo  
 Consulte o exemplo para  [vinculáveis](../windows/bindable.md) para uso do exemplo  **propputref**.
