# Inserindo um espa&#231;o entre bot&#245;es em uma barra de ferramentas
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Em geral, para inserir um espaço entre os botões, simplesmente arraste\-os uns dos outros na barra de ferramentas.  Para remover o espaço, arraste\-os em direção ao outro.  
  
### Para inserir um espaço antes de um botão que não é seguido por um espaço.  
  
1.  Arraste o botão para a direita ou para baixo até que ele sobrepõe o botão Avançar sobre a metade.  
  
### Para inserir um espaço antes de um botão que é seguido por um espaço e para manter esse espaço à direita  
  
1.  Arraste o botão até que a borda direita ou inferior apenas está tocando no botão Avançar ou apenas se sobrepõe a ela.  
  
### Para inserir um espaço antes de um botão que é seguido por um espaço e fechar esse espaço a seguir  
  
1.  Arraste o botão para a direita ou para baixo até que ele sobrepõe o botão Avançar sobre a metade.
