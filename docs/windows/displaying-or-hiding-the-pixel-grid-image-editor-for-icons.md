# <a name="displaying-or-hiding-the-pixel-grid-image-editor-for-icons"></a>Exibindo ou ocultando a grade de pixels (editor de imagens para ícones)
Para todos os painéis do Editor de imagem com um fator de ampliação de 4 ou superior, você pode exibir uma grade que delimita os pixels individuais na imagem.  
  
> [!TIP]
>  Dicas de ferramenta exibida quando você passar o cursor sobre um botão de barra de ferramentas. Estas dicas podem ajudá-lo a identificar a função de cada botão.  
  
### <a name="to-display-or-hide-the-pixel-grid"></a>Para exibir ou ocultar a grade de pixels  
  
1.  Sobre o **imagem** menu, clique em **configurações da grade**.  
  
2.  Selecione o **grade de pixels** caixa de seleção para exibir a grade, ou desmarque a caixa para ocultar a grade.
