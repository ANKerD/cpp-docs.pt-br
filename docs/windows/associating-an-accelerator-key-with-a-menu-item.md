# Associando uma tecla de acelera&#231;&#227;o a um item de menu
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Muitas vezes, convém que um item de menu e uma combinação de teclado para emitir o mesmo comando do programa.  Para fazer isso, atribuindo o mesmo identificador de recurso \(ID\) para o item de menu e uma entrada na tabela de Aceleradores do seu aplicativo.  Você então editar a legenda do item de menu para mostrar o nome do accelerator.  Para obter mais informações sobre itens de menu e teclas de aceleração, consulte [Associando um Item de Menu uma tecla aceleradora](../windows/associating-a-menu-command-with-an-accelerator-key.md).  
  
 Para obter informações sobre como adicionar recursos a projetos gerenciados, consulte [Recursos em aplicativos](../Topic/Resources%20in%20Desktop%20Apps.md) no *Guia do desenvolvedor do .NET Framework.* Para obter informações sobre como adicionar manualmente os arquivos de recursos a projetos gerenciados, acessar recursos, exibir recursos estáticos e atribuir cadeias de caracteres de recursos para propriedades, consulte [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
