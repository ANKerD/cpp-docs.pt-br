# <a name="modulemodule-destructor"></a>Destruidor Module::~Module
Deinitializes a instância atual da classe de módulo.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
virtual ~Module();  
```  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** module.h  
  
 **Namespace:** Microsoft::WRL
