# <a name="modulemodule-destructor"></a>Destruidor Module::~Module

Realiza o desligamento da instância atual do **módulo** classe.

## <a name="syntax"></a>Sintaxe

```cpp
virtual ~Module();
```

## <a name="requirements"></a>Requisitos

**Cabeçalho:** module.h

**Namespace:** Microsoft::WRL
