# <a name="microsoftwrlwrappersdetails-namespace"></a>Namespace Microsoft::WRL::Wrappers::Details

## <a name="syntax"></a>Sintaxe  
  
```  
namespace Microsoft::WRL::Wrappers::Details;  
```
