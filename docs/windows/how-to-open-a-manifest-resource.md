# <a name="how-to-open-a-manifest-resource"></a>Como abrir um recurso de manifesto
Se você abrir um recurso de manifesto do [exibição recursos](../windows/resource-view-window.md), o recurso será aberto no formato binário. Para exibir o conteúdo de um recurso de manifesto em um formato mais visível, você deve abrir o recurso do Gerenciador de soluções.  
  
### <a name="to-open-a-manifest-resource-in-the-text-editor"></a>Para abrir um recurso de manifesto no Editor de texto  
  
1.  Com o projeto aberto no `Solution Explorer`, expanda o **arquivos de recurso** pasta.  
  
2.  Clique duas vezes o arquivo. manifest.  
  
     O recurso de manifesto é aberto no Editor de texto.  
  
### <a name="to-open-a-manifest-resource-in-another-editor"></a>Para abrir um recurso de manifesto em outro editor  
  
1.  No Gerenciador de soluções, clique no arquivo. manifest e escolha **abrir com...**  no menu de atalho.  
  
2.  No **abrir com** caixa de diálogo, especifique o editor que você deseja usar e clique em **abrir**.  
  
 Para obter informações sobre como adicionar recursos a projetos gerenciados, consulte [recursos em aplicativos de área de trabalho](/dotnet/framework/resources/index) no *guia do desenvolvedor do .NET Framework.*  
  
 Requisitos  
  
 Win32
