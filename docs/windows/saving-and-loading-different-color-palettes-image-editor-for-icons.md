# Salvando e carregando paletas de cores diferentes (editor de imagens para &#237;cones)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Você pode salvar e carregar uma paleta de cores que contém [personalizado cores](../windows/customizing-or-changing-colors-image-editor-for-icons.md).  \(Por padrão, a paleta de cores usada mais recentemente é carregada automaticamente quando você iniciar o Visual Studio.\)  
  
> [!TIP]
>  Como o editor de imagens não tem meios para restaurar a paleta de cores padrão, você deve salvar a paleta de cores padrão com um nome como standard.pal ou default.pal para que você possa restaurar facilmente as configurações padrão.  
  
### Para salvar uma paleta de cores personalizadas  
  
1.  Do **imagem** menu, escolha **Salvar paleta**.  
  
2.  Navegue até o diretório onde você deseja salvar a paleta e digite um nome para a paleta.  
  
3.  Clique em **Salvar**.  
  
### Para carregar uma paleta de cores personalizadas  
  
1.  Do **imagem** menu, escolha **Paleta carga**.  
  
2.  No [caixa de diálogo carregar a paleta de cores](../windows/load-palette-colors-dialog-box-image-editor-for-icons.md), navegue até o diretório correto e selecione a paleta que você deseja carregar.  Paletas de cores são salvas com uma extensão de arquivo. PAL.  
  
 Para obter informações sobre como adicionar recursos a projetos gerenciados, consulte [Recursos em aplicativos](../Topic/Resources%20in%20Desktop%20Apps.md) no *Guia do desenvolvedor do .NET Framework.* Para obter informações sobre como adicionar manualmente os arquivos de recursos a projetos gerenciados, acessar recursos, exibir recursos estáticos e atribuir cadeias de caracteres de recursos para propriedades, consulte [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
