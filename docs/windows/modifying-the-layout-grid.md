# Modificando a grade de layout
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Quando você estiver colocando ou Organizando controles em uma caixa de diálogo, você pode usar a grade de layout para posicionamento mais precisa.  Quando a grade é ativada, os controles aparecem para "ajustar a" as linhas pontilhadas da grade como se magnetizada.  Você pode ativar e desativar a esse recurso "snap para grade" e alterar o tamanho das células de layout de grade.  
  
### Para ativar ou desativar a grade de layout  
  
1.  Do  **formato** menu, escolha  **Configurações do guia**.  
  
2.  No  [Caixa de diálogo de configurações do guia de](../Topic/Guide%20Settings%20Dialog%20Box.md), marque ou desmarque o  **grade** botão.  
  
     Você ainda poderá controlar a grade em janelas de editor de diálogo individuais usando o  **Alternar grade** na barra do  [Barra de ferramentas do Editor de diálogo](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md).  
  
### Para alterar o tamanho da grade de layout  
  
1.  Do  **formato** menu, escolha  **Configurações do guia**.  
  
2.  No  [Caixa de diálogo de configurações do guia de](../Topic/Guide%20Settings%20Dialog%20Box.md), digite a altura e largura em DLUs para as células na grade.  A largura ou altura mínima é de 4 DLUs.  Para obter mais informações sobre DLUs, consulte  [A disposição dos controles em caixas de diálogo](../mfc/arrangement-of-controls-on-dialog-boxes.md).
