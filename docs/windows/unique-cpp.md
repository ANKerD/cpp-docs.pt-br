# unique (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Especifica um indicador único.  
  
## Sintaxe  
  
```  
  
[unique]  
  
```  
  
## Comentários  
 O  **exclusivo** o atributo do C\+\+ tem a mesma funcionalidade que o  [exclusivo](http://msdn.microsoft.com/library/windows/desktop/aa367294) atributo MIDL.
