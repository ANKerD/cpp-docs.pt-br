# <a name="text-tool-dialog-box-image-editor-for-icons"></a>Caixa de diálogo Ferramenta de Texto (editor de imagens para ícones)
Use o **ferramenta texto** caixa de diálogo para adicionar texto a um recurso de cursor, bitmap ou ícone.  
  
 Para acessar essa caixa de diálogo, abra o [Editor de imagem](../windows/window-panes-image-editor-for-icons.md). Selecione **ferramentas** do **imagem** menu e, em seguida, selecione o **ferramenta texto** comando.  
  
 **Botão fonte**  
 Abre o [caixa de diálogo fonte da ferramenta de texto](../windows/text-tool-font-dialog-box-image-editor-for-icons.md), na qual você pode alterar a fonte, estilo ou tamanho da fonte do cursor. As alterações são aplicadas para o texto exibido na área de texto.  
  
 **Área de texto**  
 Exibe o texto que aparece como parte do recurso. Inicialmente, essa área está vazia.  
  
> [!NOTE]
>  Se **plano de fundo transparente** está definida, somente o texto será colocado na imagem. Se **plano de fundo opaco** for definido, um retângulo preenchido com o [cor do plano de fundo](../windows/selecting-foreground-or-background-colors-image-editor-for-icons.md), será colocado atrás do texto. Para obter mais informações, consulte [escolhendo opaco e planos de fundo transparentes](../windows/choosing-a-transparent-or-opaque-background-image-editor-for-icons.md).  
  
 Clique duas vezes no **ferramenta texto** caixa de diálogo para acessar um menu de atalho padrão que contém uma lista de comandos do Windows padrão.  
  
## <a name="requirements"></a>Requisitos  
 Nenhum
