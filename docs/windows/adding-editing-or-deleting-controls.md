# Adicionando, editando ou excluindo controles
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Usando o editor de diálogo, você pode adicionar, redimensionar, editar e excluir os controles nas caixas de diálogo.  Você também pode editar as propriedades de um controle, como, por exemplo, sua identificação, ou se é inicialmente visível em tempo de execução.  
  
 Você pode usar o menu de atalho no editor de diálogo para adicionar rapidamente controles de ActiveX registrados para uma caixa de diálogo, e você pode adicionar controles de ActiveX à caixa de ferramentas para acesso rápido \(para obter mais informações, consulte  [Caixa de diálogo da caixa de ferramentas de personalização](http://msdn.microsoft.com/pt-br/bd07835f-18a8-433e-bccc-7141f65263bb)\).
