# <a name="grid-settings-dialog-box-image-editor-for-icons"></a>Caixa de diálogo Configurações da Grade (editor de imagens para ícones)
Permite que você especifique as configurações de grade para a imagem e exibe linhas de grade sobre a imagem editada. As linhas são úteis para a imagem de edição, mas não são salvos como parte da própria imagem.  
  
 **Grade de pixels**  
 Quando marcada, exibe uma grade em torno de cada pixel no editor de imagem. A grade aparece apenas em 4 × e resoluções mais altas.  
  
 **Grade lado a lado**  
 Quando selecionada, exibe uma grade de blocos de pixels no editor de imagem, especificado pelos valores de espaçamento da grade.  
  
 **Largura**  
 Especifica a largura de cada bloco de bloco. Isso é útil ao desenhar bitmaps que contém várias imagens que são organizadas em intervalos regulares.  
  
 **Altura**  
 Especifica a altura de cada bloco de bloco. Isso é útil ao desenhar bitmaps que contém várias imagens que são organizadas em intervalos regulares.  
  
## <a name="requirements"></a>Requisitos  
 Nenhum
