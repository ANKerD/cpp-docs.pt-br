# Caixa de di&#225;logo Configura&#231;&#245;es da Grade (editor de imagens para &#237;cones)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Permite que você especifique as configurações de grade para sua imagem e exibe as linhas de grade sobre a imagem editada.  As linhas são úteis para editar a imagem, mas não são salvas como parte da própria imagem.  
  
 **Grade de pixels**  
 Quando selecionada, exibe uma grade ao redor de cada pixel no editor de imagem.  A grade aparece apenas em 4 × e resoluções mais altas.  
  
 **Grade lado a lado**  
 Quando selecionada, exibe uma grade ao redor de blocos de pixels no editor de imagem, especificado pelos valores de espaçamento da grade.  
  
 **Width**  
 Especifica a largura de cada bloco de lado a lado.  Isso é útil ao desenhar os bitmaps que contém várias imagens que são organizadas em intervalos regulares.  
  
 **Height**  
 Especifica a altura de cada bloco de lado a lado.  Isso é útil ao desenhar os bitmaps que contém várias imagens que são organizadas em intervalos regulares.  
  
## Requisitos  
 Nenhum
