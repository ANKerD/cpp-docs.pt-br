# <a name="eventtargetarrayeventtargetarray-destructor"></a>Destruidor EventTargetArray::~EventTargetArray

## <a name="syntax"></a>Sintaxe  
  
```  
~EventTargetArray();  
```  
  
## <a name="remarks"></a>Comentários
