# Personalizando ou alterando cores (editor de imagens para &#237;cones)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

O editor de imagens [paleta de cores](../Topic/Colors%20Window%20\(Image%20Editor%20for%20Icons\).md) exibe inicialmente 16 cores padrão.  Além das cores exibidas, você pode criar suas próprias cores personalizadas.  Você pode então [Salvar e carregar uma paleta de cores personalizada](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md).  
  
### Para alterar as cores na paleta de cores  
  
1.  Do **imagem** menu, escolha **Ajustar cores**.  
  
2.  No [caixa de diálogo Seletor de cores personalizada](../windows/custom-color-selector-dialog-box-image-editor-for-icons.md), defina a cor digitando valores RGB ou HSL nas caixas de texto apropriado ou escolher uma cor no **exibição de cores de gradiente** caixa.  
  
3.  Defina a luminosidade movendo o controle deslizante **luminosidade** barra.  
  
4.  Muitas cores personalizadas são pontilhadas.  Se desejar que a cor sólida mais próxima da cor pontilhada, clique duas vezes o **cor** caixa.  
  
     Se você decidir posteriormente você deseja aplicar a cor pontilhada, mova o controle deslizante o **luminosidade** barra ou mova a mira a **exibição de cores de gradiente** novamente para restaurar o pontilhado.  
  
5.  Clique em **OK** para adicionar a nova cor.  
  
 Para obter informações sobre como adicionar recursos a projetos gerenciados, consulte [Recursos em aplicativos](../Topic/Resources%20in%20Desktop%20Apps.md) no *Guia do desenvolvedor do .NET Framework.* Para obter informações sobre como adicionar manualmente os arquivos de recursos a projetos gerenciados, acessar recursos, exibir recursos estáticos e atribuir cadeias de caracteres de recursos para propriedades, consulte [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
