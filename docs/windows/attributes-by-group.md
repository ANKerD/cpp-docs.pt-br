# Attributes by Group
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Os atributos C\+\+ são organizados nos grupos funcionais.  
  
|Atributo|Descrição|  
|--------------|---------------|  
|[Atributos COM](../Topic/COM%20Attributes.md)|Insira o código para suportar várias áreas de desenvolvimento COM e de desenvolvimento do common language runtime do .NET Framework.|  
|[Atributos IDL](../windows/idl-attributes.md)|Ativar a modificação do arquivo de .idl de um arquivo de origem sem o uso de um assistente e sem conhecimento da estrutura e a sintaxe de esse arquivo.|  
|[Atributos do consumidor do OLE DB](../windows/ole-db-consumer-attributes.md)|Inject código com base em [Modelos de consumidor do OLE DB](../data/oledb/ole-db-consumer-templates-reference.md)— para criar um consumidor trabalhando do OLE DB executando tarefas como as tabelas de abertura, executando comandos, e acessando dados.|  
|[Atributos do compilador](../windows/compiler-attributes.md)|Fornecido pelo compilador Visual C\+\+.|  
  
 Para obter informações sobre como definir atributos personalizados para seu aplicativo.NET Framework, consulte [User\-Defined Attributes](../windows/user-defined-attributes-cpp-component-extensions.md).
