# Caixa de di&#225;logo Seletor de Cor Personalizada (editor de imagens para &#237;cones)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Esta caixa de diálogo permite que você personalize as cores que você pode usar para sua imagem.  Para obter mais informações, consulte  [Personalizando ou alterando cores](../windows/customizing-or-changing-colors-image-editor-for-icons.md).  
  
 **Exibição de cores de gradiente**  
 Altera os valores de uma cor selecionada.  Posição na cruz na cor que você deseja alterar.  Em seguida, mova o controle deslizante para cima ou para baixo para alterar a luminosidade ou valores RGB da cor.  
  
 **Barra de luminosidade**  
 Define a luminosidade da cor que você seleciona na caixa de exibição de cores de gradiente.  Clique e arraste a seta branca até a barra de maior brilho ou para baixo por menos.  Observe que a caixa de cores exibe a cor que você selecionou e o efeito de luminosidade que você definir.  
  
 **Color**  
 Lista o matiz \(valor de roda de cores\) da cor que está sendo definida.  Os valores variam de 0 a 240, onde 0 é vermelho, 60 é amarelo, 120 é verde, 180 é ciano, 200 é magenta e 240 é azul.  
  
 **Matiz**  
 Lista o matiz \(valor de roda de cores\) da cor que está sendo definida.  Os valores variam de 0 a 240, onde 0 é vermelho, 60 é amarelo, 120 é verde, 180 é ciano, 200 é magenta e 240 é azul.  
  
 **Sat.**  
 Especifica o valor de saturação da cor que está sendo definida.  A saturação é a quantidade de cor em um matiz especificado.  Os valores variam de 0 a 240.  
  
 **Lum.**  
 Lista a luminosidade \(brilho\) da cor que está sendo definida.  Os valores variam de 0 a 240.  
  
 **Vermelho**  
 Especifica o valor de vermelho da cor que está sendo definida.  Os valores variam de 0 a 255.  
  
 **Verde**  
 Especifica o valor de verde da cor que está sendo definida.  Os valores variam de 0 a 255.  
  
 **Azul**  
 Especifica o valor de azul da cor que está sendo definida.  Os valores variam de 0 a 255.  
  
## Requisitos  
 Nenhum
