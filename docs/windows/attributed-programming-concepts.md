# Attributed Programming Concepts
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Visual C\+\+ inclui uma variedade de materiais para ajudá\-lo a usar atributos em seus programas.  Atributos, um novo recurso do Visual C\+\+, são projetados para simplificar a programação COM e.Desenvolvimento de tempo de execução comum em idioma NET Framework.  Quando você inclui os atributos nos arquivos de origem, o compilador funciona com bibliotecas de vínculo dinâmico \(DLL\) para inserir o código ou modificar o código nos arquivos gerados do objeto do provedor.  Há que auxiliam na criação de arquivos. idl, interfaces, bibliotecas de tipos e outros elementos COM atributos.  No ambiente de desenvolvimento integrado \(IDE\), os atributos são suportados pelos assistentes e pela janela Propriedades.  
  
 Enquanto os atributos eliminam alguns da codificação detalhada necessária para gravar objetos COM, você precisa de um plano de fundo em  [COM fundamentals](http://msdn.microsoft.com/library/windows/desktop/ms694363) usá\-los melhor.
