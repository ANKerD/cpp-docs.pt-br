Todas as versões do CRT agora oferecem suporte a várias threading com exceção de versões não bloqueio de algumas funções. Consulte [desempenho de bibliotecas multithread](../c-runtime-library/multithreaded-libraries-performance.md) para obter mais informações.  
  
 Consulte [recursos da biblioteca CRT](../c-runtime-library/crt-library-features.md) para obter mais informações sobre versões do CRT.
