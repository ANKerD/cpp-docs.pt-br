# <a name="ompgetwtick"></a>omp_get_wtick
Retorna o número de segundos entre tiques do relógio de processador.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
double omp_get_wtick( );  
```  
  
## <a name="remarks"></a>Comentários
