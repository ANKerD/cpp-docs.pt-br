# omp_get_thread_num
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

Retorna o número de threads do thread em execução dentro de sua equipe de thread.  
  
## Sintaxe  
  
```  
int omp_get_thread_num( );  
```
