# <a name="ompgetthreadnum"></a>omp_get_thread_num
Retorna o número de threads de execução de thread dentro de sua equipe de thread.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
int omp_get_thread_num( );  
```
