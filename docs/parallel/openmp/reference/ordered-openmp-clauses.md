# <a name="ordered-openmp-clauses"></a>ordered (cláusulas OpenMP)

## <a name="syntax"></a>Sintaxe  
  
```  
ordered  
```  
  
## <a name="remarks"></a>Comentários  
 `ordered` aplica-se para o [para](../../../parallel/openmp/reference/for-openmp.md) diretiva.
