# OpenMP in Visual C++
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

A interface de programa aplicativo OpenMP C e C\+\+ permite que você escrever aplicativos que efetivamente usam vários processadores.  Visual C\+\+ oferece suporte ao padrão OpenMP 2.0.
