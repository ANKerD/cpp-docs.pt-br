# 2. Directives
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

As diretivas são baseadas em  **\# pragma** as diretivas definidas nos padrões c e C\+\+.  Compiladores que suportam o OpenMP C e C\+\+ API incluirá uma opção de linha de comando que ativa e permite que a interpretação de todas as diretivas de compilador do OpenMP.
