# <a name="ml-nonfatal-error-a2078"></a>Erro não fatal A2078 (ML)
**instrução não permite EXTREMO endereçamento indireta**  
  
 Um salto condicional ou loop não pode tomar um operando de memória. Ele deve ser fornecido um endereço relativo ou um rótulo.
