# ML Nonfatal Error A2078
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**instrução não aceita EXTREMO endereçamento indireta**  
  
 Um salto condicional ou loop não pode assumir um operando de memória.  Ele deve ser fornecido um endereço relativo ou uma etiqueta.
