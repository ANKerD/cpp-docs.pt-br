# <a name="echo"></a>ECHO
Exibe *mensagem* para o dispositivo de saída padrão (por padrão, a tela).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
ECHO   
message  
  
```  
  
## <a name="remarks"></a>Comentários  
 Mesmo que [% OUT](../../assembler/masm/percent-out.md).
