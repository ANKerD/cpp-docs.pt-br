# .FARDATA
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Quando usado com  [.MODELO](../../assembler/masm/dot-model.md), inicia um segmento de dados até que ponto para dados inicializados \(nome do segmento FAR\_DATA ou  *nome*\).  
  
## Sintaxe  
  
```  
  
.FARDATA [[name]]  
```
