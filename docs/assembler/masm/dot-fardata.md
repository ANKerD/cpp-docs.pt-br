# <a name="fardata"></a>.FARDATA
Quando usado com [. MODELO](../../assembler/masm/dot-model.md), inicia um segmento de dados distante para dados inicializado (nome de segmento FAR_DATA ou *nome*).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
.FARDATA [[name]]  
```
