# <a name="ml-nonfatal-error-a2037"></a>Erro não fatal A2037 (ML)
**não é permitida dentro da definição de estrutura**  
  
 Uma definição de estrutura continha uma instrução inválida.  
  
 Uma estrutura não pode conter instruções, rótulos, procedimentos, diretivas de fluxo de controle, [. INICIALIZAÇÃO](../../assembler/masm/dot-startup.md), ou [. SAÍDA](../../assembler/masm/dot-exit.md).
