# ML Nonfatal Error A2063
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**pode ALINHAR apenas a potência de 2: expressão**  
  
 A expressão especificada com o  [ALINHAR](../../assembler/masm/align-masm.md) diretiva era inválida.  
  
 O  **ALINHAR** expressão deve ser uma potência de 2 entre 2 e 256 e deve ser menor ou igual ao alinhamento do segmento atual, estrutura ou união.
