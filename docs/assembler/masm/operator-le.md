# <a name="operator-le"></a>operador LE
Retorna true (-1) se *expression1* é menor que ou igual a *expression2*, ou retornará false (0) se não for.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
expression1 LE expression2  
```
