# .NOCREF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Suprime a listagem dos símbolos no arquivo de tabela e o navegador do símbolo.  
  
## Sintaxe  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## Comentários  
 Se os nomes forem especificados, apenas os determinados nomes são suprimidos.  Mesmo que  [.XCREF](../../assembler/masm/dot-xcref.md).
