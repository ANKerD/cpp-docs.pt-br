# <a name="errdef"></a>.ERRDEF
Gera um erro se *nome* é um rótulo definido anteriormente, a variável ou o símbolo.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
.ERRDEF name [[, message]]  
```
