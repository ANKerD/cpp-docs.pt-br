# operator GE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retorna true \(– 1\) se  *Expressão1* é maior que ou igual a  *Expressão2*, ou retorna false \(0\) se não for.  
  
## Sintaxe  
  
```  
  
expression1 GE expression2  
```
