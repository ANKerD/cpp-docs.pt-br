# <a name="operator-ge"></a>operador GE
Retorna true (-1) se *expression1* é maior que ou igual a *expression2*, ou retornará false (0) se não for.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
expression1 GE expression2  
```
