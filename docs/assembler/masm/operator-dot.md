# <a name="operator-"></a>Operador .
Retorna o primeiro operador *expressão* mais o deslocamento do *campo* dentro de sua estrutura ou união. O segundo operador retorna o valor no local apontado pela *registrar* mais o deslocamento do *campo* dentro de sua estrutura ou união.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
      expression  
      . field [[. field]]...  
[register]. field [[. field]]...  
```
