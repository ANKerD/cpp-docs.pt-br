# <a name="df"></a>DF
Aloca e, opcionalmente, inicializa 6 bytes de armazenamento para cada `initializer`. `DF` é um sinônimo do [FWORD](../../assembler/masm/fword.md).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
[[name]] DF initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Comentários  
 Também pode ser usado como um especificador de tipo em qualquer lugar, um tipo é legal.
