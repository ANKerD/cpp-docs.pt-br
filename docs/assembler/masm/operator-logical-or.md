# <a name="operator-"></a>Operador ||
Lógica `OR`. Usado somente em [. Se](../../assembler/masm/dot-if.md), [. ENQUANTO](../../assembler/masm/dot-while.md), ou [. REPITA](../../assembler/masm/dot-repeat.md) bloqueia e avaliadas em tempo de execução, não no momento do assembly.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
expression1 || expression2  
```
