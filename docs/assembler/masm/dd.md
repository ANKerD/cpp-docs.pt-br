# <a name="dd"></a>DD
Aloca e, opcionalmente, inicializa uma palavra dupla (4 bytes) de armazenamento para cada `initializer`. `DD` é um sinônimo do [DWORD](../../assembler/masm/dword.md).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
[[name]] DD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Comentários  
 Também pode ser usado como um especificador de tipo em qualquer lugar, um tipo é legal.
