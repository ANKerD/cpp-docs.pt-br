# <a name="real8"></a>REAL8
Aloca e, opcionalmente, inicializa um número de ponto flutuante de precisão dupla (8 bytes) para cada *inicializador*.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
name REAL8 initializer [[, initializer]]...  
```
