# INSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Localiza a primeira ocorrência de  *textitem2* na  *textitem1*.  
  
## Sintaxe  
  
```  
  
name  
 INSTR [[position,]] textitem1, textitem2  
```  
  
## Comentários  
 Iniciando o  *posição* é opcional.  Cada item de texto pode ser uma seqüência literal, uma constante precedida por um `%`, ou a seqüência de caracteres retornado por uma função de macro.
