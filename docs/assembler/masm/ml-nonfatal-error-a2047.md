# ML Nonfatal Error A2047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**seqüência de caracteres vazia \(nula\)**  
  
 Uma seqüência de caracteres consistiu em um par de delimitação de aspas e nenhum caractere dentro.  
  
 Uma seqüência de caracteres a ser válido, ele deve conter 1 – 255 caracteres.
