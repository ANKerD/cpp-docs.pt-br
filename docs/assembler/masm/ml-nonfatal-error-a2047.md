# <a name="ml-nonfatal-error-a2047"></a>Erro não fatal A2047 (ML)
**cadeia de caracteres vazia (null)**  
  
 Uma cadeia de caracteres consistia em um par de delimitador de aspas simples e nenhum caractere dentro.  
  
 Uma cadeia de caracteres a ser válido, ele deve conter 1 a 255 caracteres.
