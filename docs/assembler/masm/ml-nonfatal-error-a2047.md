# <a name="ml-nonfatal-error-a2047"></a>Erro não fatal A2047 (ML)

**cadeia de caracteres vazia (null)**

Uma cadeia de caracteres é formada por um par de delimitador de aspas e nenhum caractere dentro.

Uma cadeia de caracteres seja válido, ele deve conter entre 1 e 255 caracteres.
