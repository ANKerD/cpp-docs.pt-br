# <a name="exit"></a>.EXIT
Gera o código de encerramento.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
.EXIT [[expression]]  
```  
  
## <a name="remarks"></a>Comentários  
 Retorna opcional *expressão* shell.
