# <a name="operator-gt"></a>operador GT
Retorna true (-1) se *expression1* é maior do que *expression2*, ou retornará false (0) se não for.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
expression1 GT expression2  
```
