# operator GT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retorna true \(– 1\) se  *Expressão1* é maior do que  *Expressão2*, ou retorna false \(0\) se não for.  
  
## Sintaxe  
  
```  
  
expression1 GT expression2  
```
