# <a name="operator-high32"></a>operator HIGH32
Retorna os 32 bits altos de *expressão*. Expressões MASM são valores de 64 bits.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
HIGH32 expression  
```
