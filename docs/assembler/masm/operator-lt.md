# <a name="operator-lt"></a>operador LT
Retorna true (-1) se *expression1* é menor que *expression2*, ou retornará false (0) se não for.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
expression1 LT expression2  
```
