# <a name="operator-this"></a>operador THIS
Retorna um operando de especificado `type` cujos valores de deslocamento e de segmento são iguais para o valor de contador do local atual.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
THIS type  
```
