# <a name="qword"></a>QWORD
Aloca e, opcionalmente, inicializa 8 bytes de armazenamento para cada `initializer`. Também pode ser usado como um especificador de tipo em qualquer lugar, um tipo é legal.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
[[name]] QWORD initializer [[, initializer]]...  
```
