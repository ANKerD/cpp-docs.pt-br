# <a name="qword"></a>QWORD

Aloca e inicializa, opcionalmente, de 8 bytes de armazenamento para cada `initializer`. Também pode ser usado como um especificador de tipo em qualquer lugar que um tipo é legal.

## <a name="syntax"></a>Sintaxe

> [[nome]] Inicializador QWORD [[, inicializador]]...
