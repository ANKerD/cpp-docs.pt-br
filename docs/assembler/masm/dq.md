# DQ
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Aloca e opcionalmente inicializa 8 bytes de armazenamento para cada `initializer`.  Também pode ser usado como um especificador de tipo em qualquer lugar de um tipo é legal.  `DQ`é um sinônimo de  [QWORD](../../assembler/masm/qword.md).  
  
## Sintaxe  
  
```  
[[name]] DQ initializer [[, initializer]]...  
```
