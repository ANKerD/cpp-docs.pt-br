# IFIDN]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Assembly de subsídios se os itens de texto são idênticos.  
  
## Sintaxe  
  
```  
  
IFIDN[[I]] textitem1, textitem2  
```  
  
## Comentários  
 Se  **I** é fornecido, a comparação diferencia maiúsculas de minúsculas.  Consulte  [Se](../../assembler/masm/if-masm.md) para obter a sintaxe completa.
