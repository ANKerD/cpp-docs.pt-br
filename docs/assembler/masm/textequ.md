# TEXTEQU
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Atribui  *textitem* para  *nome*.  O  *textitem* pode ser uma seqüência literal, uma constante precedida por um `%`, ou a seqüência de caracteres retornado por uma função de macro.  
  
## Sintaxe  
  
```  
  
name TEXTEQU [[textitem]]  
```
