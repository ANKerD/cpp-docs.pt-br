# INCLUDELIB (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Informa o vinculador que o módulo atual deve ser vinculado com  *o nome da biblioteca*.  
  
## Sintaxe  
  
```  
  
INCLUDELIB libraryname  
```  
  
## Comentários  
 O  *o nome da biblioteca* deve ser colocado entre colchetes angulares, se ele inclui uma barra invertida, ponto e vírgula, maior\-símbolo, menos maior que\-de símbolo, aspas simples ou aspas duplas.
