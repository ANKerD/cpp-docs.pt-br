# <a name="datasize"></a>@DataSize
0 para **minúscula**, **pequeno**, **médio**, e **simples** modelos, 1 para **COMPACT** e  **GRANDES** modelos e 2 para **ENORME** modelo (numéricos serão iguais).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
@DataSize  
  
```
