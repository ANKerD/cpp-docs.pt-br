# @DataSize
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

0 para  **TINY**,  **pequenos**,  **Médio**, e  **FLAT** modela, 1 para  **COMPACT** e  **grande** modelos e 2 para  **ENORME** modelo \(equacionar numérico\).  
  
## Sintaxe  
  
```  
  
@DataSize  
  
```
