# <a name="data"></a>.DATA
Quando usado com [. MODELO](../../assembler/masm/dot-model.md), inicia um segmento de dados curto para dados inicializado (segmento nome data).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
.DATA  
```
