# <a name="word"></a>WORD
Aloca e, opcionalmente, inicializa uma palavra (2 bytes) de armazenamento para cada `initializer`. Também pode ser usado como um especificador de tipo em qualquer lugar, um tipo é legal.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
[[name]] WORD initializer [[, initializer]]...  
```
