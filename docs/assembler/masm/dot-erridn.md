# .ERRIDN]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gera um erro se os itens de texto são idênticos.  
  
## Sintaxe  
  
```  
  
.ERRIDN[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## Comentários  
 Se  **I** é fornecido, a comparação diferencia maiúsculas de minúsculas.
