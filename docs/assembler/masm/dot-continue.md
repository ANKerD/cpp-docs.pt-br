# <a name="continue"></a>.CONTINUE
Gera código para ir para a parte superior de um [. ENQUANTO](../../assembler/masm/dot-while.md) ou [. REPITA](../../assembler/masm/dot-repeat.md) bloquear se `condition` for true.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
.CONTINUE [[.IF condition]]  
```
