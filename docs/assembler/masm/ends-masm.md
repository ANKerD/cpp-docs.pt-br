# ENDS (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Marca o final do segmento, estrutura ou união  *nome* começado anteriormente com  [segmento](../../assembler/masm/segment.md),  [STRUCT](../Topic/STRUCT%20\(MASM\).md),  [união](../../assembler/masm/union.md), ou uma diretiva simplificada do segmento.  
  
## Sintaxe  
  
```  
  
name ENDS  
```
