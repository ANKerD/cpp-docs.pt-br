# <a name="fardata"></a>.FARDATA?
Quando usado com [. MODELO](../../assembler/masm/dot-model.md), inicia um segmento de dados distante para dados não inicializados (nome de segmento FAR_BSS ou *nome*).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
.FARDATA? [[name]]  
```
