# PAGE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

A primeira diretiva define a linha de  *comprimento* e caractere  *largura* da listagem de programa.  Se nenhum argumento for fornecido, gera uma quebra de página.  A segunda diretiva incrementa o número da seção e redefine o número da página como 1.  
  
## Sintaxe  
  
```  
  
      PAGE [[[[length]], width]]  
PAGE +  
```
