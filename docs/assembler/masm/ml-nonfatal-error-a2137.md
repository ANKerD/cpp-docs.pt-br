# ML Nonfatal Error A2137
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**poucos argumentos para INVOKE**  
  
 O número de argumentos passados usando o  [INVOKE](../../assembler/masm/invoke.md) diretiva foi menor do que o número de parâmetros obrigatórios especificados no protótipo para o procedimento que está sendo chamado.
