# <a name="sword"></a>SWORD
Aloca e, opcionalmente, inicializa uma palavra assinada (2 bytes) de armazenamento para cada `initializer`. Também pode ser usado como um especificador de tipo em qualquer lugar, um tipo é legal.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
name SWORD initializer [[, initializer]]...  
```
