 O driver MASM, chamado ML.exe, gerado um erro do sistema.  
  
 Observe as circunstâncias do erro e notifique a Microsoft Corporation. Serviços de suporte está disponível em [ http://support.microsoft.com/ ](http://support.microsoft.com/).
