# <a name="ml-fatal-error-a1016"></a>Erro fatal A1016 (ML)

**Erro do Assembler interno**

O driver MASM, chamado ML.exe, gerado um erro do sistema.

Observe as circunstâncias do erro e notificar a Microsoft Corporation. Suporte técnico está disponível em [ http://support.microsoft.com/ ](http://support.microsoft.com/).
