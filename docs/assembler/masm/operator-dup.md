# <a name="operator-dup"></a>operador DUP
Especifica *contagem* número de declarações de *initialvalue*.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
count DUP (initialvalue [[, initialvalue]]...)  
```
