# .386P
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Permite que o conjunto de todas as instruções \(inclusive privilegiado\) para o processador 80386; Desativa o conjunto de instruções introduzidas com processadores mais recentes.  
  
## Sintaxe  
  
```  
  
.386P  
  
```  
  
## Comentários  
 Também permite que 80387 instruções.
