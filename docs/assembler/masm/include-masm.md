# INCLUDE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Insere o código do arquivo de origem fornecido pelo fonte  *nome do arquivo* no arquivo de origem atual durante a montagem.  
  
## Sintaxe  
  
```  
  
INCLUDE filename  
```  
  
## Comentários  
 O  *nome do arquivo* deve ser colocado entre colchetes angulares, se ele inclui uma barra invertida, ponto e vírgula, maior\-símbolo, menos maior que\-de símbolo, aspas simples ou aspas duplas.
