# <a name="include-masm"></a>INCLUDE (MASM)
Inserções de código do arquivo de origem fornecido pelo *filename* no arquivo de origem durante o assembly.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
INCLUDE filename  
```  
  
## <a name="remarks"></a>Comentários  
 O *filename* devem ser colocados entre colchetes angulares se ele inclui uma barra invertida, ponto e vírgula, maior-que símbolo, menos-que o símbolo, aspas simples ou aspas duplas.
