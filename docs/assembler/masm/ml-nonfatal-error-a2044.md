# <a name="ml-nonfatal-error-a2044"></a>Erro não fatal A2044 (ML)
**caractere inválido no arquivo**  
  
 O arquivo de origem contém um caractere fora de um comentário, cadeia de caracteres, ou literal não foi reconhecido como um operador ou outro caractere válido.
