# <a name="operator-carry"></a>operador CARRY?

Status do sinalizador de realização. Usado somente em [. IF](../../assembler/masm/dot-if.md), [. Embora](../../assembler/masm/dot-while.md), ou [. REPITA](../../assembler/masm/dot-repeat.md) bloqueia e avaliado em tempo de execução, não no momento do assembly.

## <a name="syntax"></a>Sintaxe

> CARREGAR?
