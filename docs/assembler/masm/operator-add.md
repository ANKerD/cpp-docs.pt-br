# <a name="operator-"></a>operador +
Retorna o primeiro operador *expression1* mais *expression2*. O segundo operador inverte o sinal de *expressão*.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
      expression1 + expression2  
-expression  
```
