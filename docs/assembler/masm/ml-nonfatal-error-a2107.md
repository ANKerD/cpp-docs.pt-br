# <a name="ml-nonfatal-error-a2107"></a>Erro não fatal A2107 (ML)
**não é possível ter salto distante implícita ou chamada para perto de rótulo**  
  
 Foi feita uma tentativa para fazer implícita muito rápido ou ligue para um rótulo próximo em outro segmento.
