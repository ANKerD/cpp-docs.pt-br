# <a name="ml-nonfatal-error-a2107"></a>Erro não fatal A2107 (ML)

**não é possível ter salto distante implícito ou chamada quase rótulo**

Foi feita uma tentativa para tornar implícito muito pular ou chamada para um rótulo quase em outro segmento.
