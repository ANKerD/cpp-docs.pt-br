# <a name="ml-nonfatal-error-a2064"></a>Erro não fatal A2064 (ML)
**alinhamento da estrutura deve ser 1, 2, 4, 8 ou 16**  
  
 O alinhamento especificado em uma definição de estrutura era inválido.
