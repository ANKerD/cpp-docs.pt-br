# <a name="ml-fatal-error-a1008"></a>Erro fatal A1008 (ML)
**aninhamento de macro não correspondentes**  
  
 A macro não foi encerrada antes do final do arquivo ou a diretiva de encerramento [ENDM](../../assembler/masm/endm.md) foi encontrado fora de um bloco de macro.  
  
 Uma causa desse erro é omissão do ponto antes de [. REPITA](../../assembler/masm/dot-repeat.md) ou [. ENQUANTO](../../assembler/masm/dot-while.md).
