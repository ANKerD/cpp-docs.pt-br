# <a name="while"></a>.WHILE
Gera o código que executa o bloco de `statements` enquanto `condition` permaneça verdadeira.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
   .WHILE condition  
statements  
.ENDW  
```
