# <a name="ml-nonfatal-error-a2119"></a>Erro não fatal A2119 (ML)
**tipo de idioma deve ser especificado**  
  
 Uma definição de procedimento ou protótipo não foi fornecido um tipo de idioma.  
  
 Um tipo de idioma deve ser declarado em cada definição de procedimento ou protótipo se não for especificado um tipo de idioma padrão. Um tipo de idioma padrão é definido usando o [. MODELO](../../assembler/masm/dot-model.md) diretiva, **opção LANG**, ou as opções de linha de comando ML **/Gc** ou **/Gd**.
