# ML Nonfatal Error A2119
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**tipo de idioma deve ser especificado.**  
  
 Uma definição de procedimento ou o protótipo não recebeu um tipo de idioma.  
  
 Se um tipo de idioma padrão não for especificado, um tipo de idioma deve ser declarado em cada definição de procedimento ou protótipo.  Um tipo de idioma padrão é definido usando o  [.MODELO](../../assembler/masm/dot-model.md) diretiva,  **Opção LANG**, ou as opções de linha de comando do ML  **\/Gc** ou  **\/Gd**.
