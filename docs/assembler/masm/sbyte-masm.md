# <a name="sbyte-masm"></a>SBYTE (MASM)
Aloca e, opcionalmente, inicializa um byte assinado de armazenamento para cada `initializer`. Também pode ser usado como um especificador de tipo em qualquer lugar, um tipo é legal.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
name SBYTEinitializer [[, initializer]]...  
```
