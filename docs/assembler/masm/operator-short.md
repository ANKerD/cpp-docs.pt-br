# operator SHORT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Define o tipo de  *rótulo* a short.  Todos os saltos para  *rótulo* deve ser curto \(dentro do azul\-amarelo de intervalo a \+ 127 bytes da instrução de salto para  *rótulo*\).  
  
## Sintaxe  
  
```  
  
SHORT label  
```
