# <a name="operator-low32"></a>operator LOW32
Retorna os 32 bits baixos de *expressão*. Expressões MASM são valores de 64 bits.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
LOW32 expression  
```
