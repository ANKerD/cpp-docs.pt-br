# operator HIGH
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retorna os 8 bits altos de 16 bits baixos de  *expressão*.  Expressões MASM são valores de 64 bits.  
  
## Sintaxe  
  
```  
  
HIGH expression  
```
