# <a name="operator-high"></a>operador HIGH
Retorna os 8 bits alta de 16 bits baixos de *expressão*. Expressões MASM são valores de 64 bits.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
HIGH expression  
```
