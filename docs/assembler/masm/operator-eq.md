# <a name="operator-eq"></a>operador EQ
Retorna true (-1) se *expression1* é igual a *expression2*, ou retornará false (0) se não existir.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
expression1 EQ expression2  
```
