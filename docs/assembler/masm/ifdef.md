# IFDEF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Concede assembly se  *nome* é uma etiqueta definida anteriormente, a variável ou o símbolo.  
  
## Sintaxe  
  
```  
  
IFDEF name  
```
