# <a name="popcontext"></a>POPCONTEXT
Restaura todo ou parte do atual `context` (salvo pelo [PUSHCONTEXT](../../assembler/masm/pushcontext.md) diretiva). O `context` pode ser **SUPÕE**, `RADIX`, **listando**, **CPU**, ou **todos os**.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
POPCONTEXT context  
```
