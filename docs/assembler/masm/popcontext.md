# <a name="popcontext"></a>POPCONTEXT

Restaura todo ou parte do atual `context` (salvo o [PUSHCONTEXT](../../assembler/masm/pushcontext.md) diretiva). O `context` pode ser **PRESUME**, `RADIX`, **listando**, **CPU**, ou **todos os**.

## <a name="syntax"></a>Sintaxe

> Contexto POPCONTEXT
