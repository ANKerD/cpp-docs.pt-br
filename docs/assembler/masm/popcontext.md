# POPCONTEXT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Restaura a parte ou todo o atual `context` \(salvo pela  [PUSHCONTEXT](../../assembler/masm/pushcontext.md) diretiva\).  The `context` can be **ASSUMES**, `RADIX`, **LISTING**, **CPU**, or **ALL**.  
  
## Sintaxe  
  
```  
  
POPCONTEXT context  
```
