# DB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Aloca e opcionalmente inicializa um byte de armazenamento para cada `initializer`.  `DB`é um sinônimo de  [bytes](../../assembler/masm/byte-masm.md)  
  
## Sintaxe  
  
```  
[[name]] DB initializer [[, initializer]] ...  
```  
  
## Comentários  
 Também pode ser usado como um especificador de tipo em qualquer lugar de um tipo é legal.
