# IFDIF]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Assembly de subsídios se os itens de texto são diferentes.  
  
## Sintaxe  
  
```  
  
IFDIF[[I]] textitem1, textitem2  
```  
  
## Comentários  
 Se  **I** é fornecido, a comparação diferencia maiúsculas de minúsculas.  Consulte  [Se](../../assembler/masm/if-masm.md) para obter a sintaxe completa.
