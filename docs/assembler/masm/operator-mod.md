# <a name="operator-mod"></a>operador MOD
Retorna o valor inteiro do restante (módulo) durante a divisão *expression1* por *expression2*.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
expression1 MOD expression2  
```
