# <a name="endprolog"></a>.ENDPROLOG
Sinaliza o término das declarações de prólogo.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
.ENDPROLOG  
```  
  
## <a name="remarks"></a>Comentários  
 Erro ao usar qualquer uma das declarações de prólogo fora da região entre [PROC](../../assembler/masm/proc.md) quadro e. ENDPROLOG.  
  
 Para obter mais informações, consulte [MASM para x64 (ml64.exe)](../../assembler/masm/masm-for-x64-ml64-exe.md).
