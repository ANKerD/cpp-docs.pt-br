# .LISTMACRO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Inicia a listagem de instruções de expansão de macro que geram código ou dados.  
  
## Sintaxe  
  
```  
  
.LISTMACRO  
  
```  
  
## Comentários  
 Este é o padrão.  Mesmo que  [.XALL](../Topic/.XALL.md).
