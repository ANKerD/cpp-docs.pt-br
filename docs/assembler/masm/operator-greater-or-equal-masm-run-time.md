# <a name="operator-gt-masm-run-time"></a>operador &gt;= (tempo de execução MASM)
É maior que ou igual a. Usado somente em [. Se](../../assembler/masm/dot-if.md), [. ENQUANTO](../../assembler/masm/dot-while.md), ou [. REPITA](../../assembler/masm/dot-repeat.md) bloqueia e avaliadas em tempo de execução, não no momento do assembly.
