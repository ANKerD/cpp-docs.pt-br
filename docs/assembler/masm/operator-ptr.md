# <a name="operator-ptr"></a>operador PTR
O força do operador primeiro o *expressão* deve ser tratada como tendo especificado `type`. O segundo operador especifica um ponteiro para `type`.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
