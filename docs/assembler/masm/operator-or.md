# operator OR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retorna o resultado de um bit a bit `OR` operação para  *Expressão1* e  *Expressão2*.  
  
## Sintaxe  
  
```  
  
expression1 OR expression2  
```
