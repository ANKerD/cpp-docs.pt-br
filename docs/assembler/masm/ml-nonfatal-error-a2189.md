# <a name="ml-nonfatal-error-a2189"></a>Erro não fatal A2189 (ML)
**combinação inválida com o alinhamento de segmento**  
  
 O alinhamento especificado pelo **ALINHAR** ou `EVEN` diretiva era maior que o alinhamento de segmento atual conforme especificado pelo **segmento** diretiva.
