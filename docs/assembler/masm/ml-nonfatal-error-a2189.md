# ML Nonfatal Error A2189
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**combinação inválida com o alinhamento do segmento**  
  
 O alinhamento especificado pelo  **ALINHAR** ou `EVEN` diretiva era maior do que o alinhamento do segmento atual conforme especificado pelo  **segmento** diretiva.
