# <a name="fword"></a>FWORD
Aloca e, opcionalmente, inicializa 6 bytes de armazenamento para cada `initializer`.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
[[name]] FWORD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Comentários  
 Também pode ser usado como um especificador de tipo em qualquer lugar, um tipo é legal.
