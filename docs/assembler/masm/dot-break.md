# <a name="break"></a>.BREAK
Gera código para encerrar um [. ENQUANTO](../../assembler/masm/dot-while.md) ou [. REPITA](../../assembler/masm/dot-repeat.md) bloquear se *condição* é verdadeiro.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
.BREAK [[.IF condition]]   
```
