# <a name="operator-xor"></a>operador XOR
Retorna o resultado de um bit a bit **XOR** operação para *expression1* e *expression2*.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
expression1 XOR expression2  
```
