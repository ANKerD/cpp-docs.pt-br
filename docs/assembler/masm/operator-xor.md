# operator XOR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retorna o resultado de um bit a bit  **XOR** operação para  *Expressão1* e  *Expressão2*.  
  
## Sintaxe  
  
```  
  
expression1 XOR expression2  
```
