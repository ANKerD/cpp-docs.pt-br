# ML Nonfatal Error A2019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**operando deve ser o tipo de registro ou campo**  
  
 O seguinte operando a  [largura](../../assembler/masm/operator-width.md) ou  [MÁSCARA](../../assembler/masm/operator-mask.md) operador não era válida.  
  
 O  [largura](../../assembler/masm/operator-width.md) operador obtém um operando que é o nome de um campo ou um registro.  O  [MÁSCARA](../../assembler/masm/operator-mask.md) operador obtém um operando que é o nome de um campo ou um tipo de registro.
