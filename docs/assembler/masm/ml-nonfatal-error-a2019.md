# <a name="ml-nonfatal-error-a2019"></a>Erro não fatal A2019 (ML)
**operando deve ser tipo de registro ou do campo**  
  
 O seguinte operando a [largura](../../assembler/masm/operator-width.md) ou [MÁSCARA](../../assembler/masm/operator-mask.md) operador não era válido.  
  
 O [largura](../../assembler/masm/operator-width.md) operador receba um operando que é o nome de um campo ou um registro. O [MÁSCARA](../../assembler/masm/operator-mask.md) operador receba um operando que é o nome de um campo ou um tipo de registro.
