# <a name="forc"></a>FORC
Marca um bloco que será repetido uma vez para cada caractere em `string`, com o caractere atual substituindo `parameter` em cada repetição.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
   FORC  
parameter, <string> statements  
ENDM  
```  
  
## <a name="remarks"></a>Comentários  
 Mesmo que [IRPC](../../assembler/masm/irpc.md).
