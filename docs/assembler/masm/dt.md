# DT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Aloca e opcionalmente inicializa 10 bytes de armazenamento para cada `initializer`.  Também pode ser usado como um especificador de tipo em qualquer lugar de um tipo é legal.  `DT`é um sinônimo de  [TBYTE](../../assembler/masm/tbyte.md).  
  
## Sintaxe  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
