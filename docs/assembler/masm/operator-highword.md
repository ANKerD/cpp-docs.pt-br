# operator HIGHWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retorna os primeiros 16 bits de 32 bits baixos de  *expressão*.  Expressões MASM são valores de 64 bits.  
  
## Sintaxe  
  
```  
  
HIGHWORD expression  
```
