# <a name="operator-highword"></a>operador HIGHWORD
Retorna os primeiros 16 bits de 32 bits baixos do *expressão*. Expressões MASM são valores de 64 bits.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
HIGHWORD expression  
```
