# <a name="label-masm"></a>LABEL (MASM)
Cria um novo rótulo atribuindo o valor de contador do local atual e a determinado `type` para *nome*.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
