# LABEL (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Cria uma nova etiqueta, atribuindo o valor do contador do local atual e a determinado `type` para  *nome*.  
  
## Sintaxe  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
