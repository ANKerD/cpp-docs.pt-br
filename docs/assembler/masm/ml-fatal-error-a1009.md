# ML Fatal Error A1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**linha muito longa**  
  
 Uma linha em um arquivo de origem excedeu o limite de 512 caracteres.  
  
 Se várias linhas físicas são concatenadas com o caractere de continuação de linha \(\\\), a linha lógica resultante é ainda limitada a 512 caracteres.
