# CATSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Concatena os itens de texto.  
  
## Sintaxe  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## Comentários  
 Cada item de texto pode ser uma seqüência literal, uma constante precedida por um `%`, ou a seqüência de caracteres retornado por uma função de macro.
