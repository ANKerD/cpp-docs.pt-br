# <a name="ml-nonfatal-error-a2074"></a>Erro não fatal A2074 (ML)
**não é possível acessar o rótulo por meio de registros de segmento**  
  
 Foi feita uma tentativa para acessar um rótulo por meio de um registro de segmento que não foi considerado seu segmento ou grupo.
