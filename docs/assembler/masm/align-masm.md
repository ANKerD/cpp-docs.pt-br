# <a name="align-masm"></a>ALIGN (MASM)
Alinha a próxima variável ou instrução em um byte que é um múltiplo de `number`.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
ALIGN [[  
number  
]]  
  
```
