# DWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Aloca e opcionalmente inicializa uma palavra dupla \(4 bytes\) de armazenamento para cada `initializer`.  
  
## Sintaxe  
  
```  
[[name]] DWORD initializer [[, initializer]]...  
```  
  
## Comentários  
 Também pode ser usado como um especificador de tipo em qualquer lugar de um tipo é legal.
