# <a name="dword"></a>DWORD
Aloca e, opcionalmente, inicializa uma palavra dupla (4 bytes) de armazenamento para cada `initializer`.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
[[name]] DWORD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Comentários  
 Também pode ser usado como um especificador de tipo em qualquer lugar, um tipo é legal.
