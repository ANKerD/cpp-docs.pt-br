# <a name="operator-"></a>Operador ;;
Trata *texto* como um comentário em uma macro que aparece apenas na definição de macro. A lista não mostra *texto* onde a macro é expandida.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
;;  
text  
  
```
