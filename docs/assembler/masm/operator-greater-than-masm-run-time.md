# operator &gt; (MASM Run Time)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

É maior.  Usado somente dentro  [.IF](../Topic/.IF.md), [.WHILE](../../assembler/masm/dot-while.md), or [.Repetir](../../assembler/masm/dot-repeat.md) bloqueia e avaliado em tempo de execução, não no tempo de montagem.  
  
## Sintaxe  
  
```  
  
expression1 > expression2  
```
