# <a name="operator-ne"></a>Operator NE
Retorna true (-1) se *expression1* não é igual a *expression2*, ou retornará false (0) se ele faz.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
expression1 NE expression2  
```
