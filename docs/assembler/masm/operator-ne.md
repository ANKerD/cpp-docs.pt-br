# operator NE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retorna true \(– 1\) se  *Expressão1* não é igual a  *Expressão2*, ou retorna false \(0\) se ele faz.  
  
## Sintaxe  
  
```  
  
expression1 NE expression2  
```
