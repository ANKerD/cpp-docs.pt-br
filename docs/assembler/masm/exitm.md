# EXITM
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Finaliza a expansão do bloco de repetição ou macro atual e começa o assembly da próxima instrução fora do bloco.  
  
## Sintaxe  
  
```  
  
EXITM [[textitem]]   
```  
  
## Comentários  
 Em uma função de macro,  *textitem* é o valor retornado.
