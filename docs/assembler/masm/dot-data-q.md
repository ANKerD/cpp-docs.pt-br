# <a name="data"></a>.DATA?
Quando usado com [. MODELO](../../assembler/masm/dot-model.md), inicia um próximo segmento de dados para dados não inicializados (_BSS de nome de segmento).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
.DATA?  
  
```
