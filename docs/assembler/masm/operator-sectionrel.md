# operator SECTIONREL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retorna o deslocamento relativo da seção de expressão em relação à seção que contém o destino no executável final.  
  
## Sintaxe  
  
```  
SECTIONREL expression  
```  
  
## Comentários  
 SECTIONREL está disponível somente com emissão de objeto COFF.
