# <a name="codesize"></a>@CodeSize
0 para **minúscula**, **pequeno**, **COMPACT**, e **simples** modelos e 1 para **médio**,  **GRANDES**, e **ENORME** modelos (numéricos serão iguais).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
@CodeSize  
  
```
