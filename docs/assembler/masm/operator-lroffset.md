# <a name="operator-lroffset"></a>operador LROFFSET

Retorna o deslocamento do *expressão*. Mesmo que **deslocamento**, mas gera um deslocamento de carregador de resolvido, que permite que o Windows realocar os segmentos de código.

## <a name="syntax"></a>Sintaxe

> Expressão LROFFSET
