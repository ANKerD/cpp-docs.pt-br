# DW
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Aloca e opcionalmente inicializa uma palavra \(2 bytes\) de armazenamento para cada `initializer`.  Também pode ser usado como um especificador de tipo em qualquer lugar de um tipo é legal.  `DW`é um sinônimo de  [WORD](../../assembler/masm/word.md).  
  
## Sintaxe  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
