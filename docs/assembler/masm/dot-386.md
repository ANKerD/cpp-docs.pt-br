# <a name="386"></a>.386
Permite que o conjunto de instruções sem privilégios para o processador 80386; Desabilita o conjunto de instruções introduzidas com processadores mais recente.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
.386  
  
```  
  
## <a name="remarks"></a>Comentários  
 Também permite que 80387 instruções.
