# <a name="data"></a>@data
O nome do grupo de dados padrão. Avalia a DGROUP para todos os modelos, exceto **simples**. Avalia a **simples** sob o **simples** modelo de memória (macro de texto).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
@data  
  
```
