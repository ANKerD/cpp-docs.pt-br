# Conjunto de instru&#231;&#245;es MMX da Intel
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

## Específico da Microsoft  
 O compilador do Visual C\+\+ permite que você use a instrução de MMX Intel \(extensão de multimédios\) definida em assembler embutido.  As instruções MMX também têm suporte pela desmontagem do depurador.  O compilador gerenciar uma mensagem de aviso se a função contém as instruções MMX mas não contém uma instrução de EMMS para esvaziar o estado de multimédios.  Para obter mais informações, consulte o site Intel.
