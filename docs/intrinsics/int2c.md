# <a name="int2c"></a>__int2c
**Seção específica da Microsoft**  
  
 Gera o `int 2c` instrução, que aciona o `2c` interrupção.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
void __int2c(void);  
```
