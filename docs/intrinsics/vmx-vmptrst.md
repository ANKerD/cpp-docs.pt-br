# <a name="vmxvmptrst"></a>__vmx_vmptrst
**Seção específica da Microsoft**  
  
 Armazena o ponteiro para a estrutura atual do controle de máquina virtual (VMCS) no endereço especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
void __vmx_vmptrst(   
   unsigned __int64 *VmcsPhysicalAddress   
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 [in] *`VmcsPhysicalAddress`  
 O endereço onde o ponteiro VMCS atual está armazenado.  
  
## <a name="remarks"></a>Comentários  
 O ponteiro VMCS é um endereço físico de 64 bits.  
  
 O `__vmx_vmptrst` função é equivalente a `VMPTRST` instrução da máquina. Essa função oferece suporte a interação do monitor de máquina virtual do host com um sistema operacional e seus aplicativos de convidado. Para obter mais informações, pesquise o documento, "Intel Virtualization Technical especificação para a arquitetura IA-32 Intel," documento número C97063-002, no [Intel Corporation](http://go.microsoft.com/fwlink/?LinkId=127) site.
