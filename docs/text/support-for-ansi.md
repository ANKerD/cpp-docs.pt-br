# Suporte para ANSI
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A maioria das classes e métodos de MFC dão suporte ao conjunto de caracteres ANSI, embora a estrutura de MFC no cluster esteja evoluindo gradativamente para oferecer suporte somente ao conjunto de caracteres Unicode.  Devido aos aprimoramentos em andamento em [!INCLUDE[windowsver](../Token/windowsver_md.md)] e na versão 6,1 dos controles comum do windows, o suporte para várias classes ANSI os métodos e são substituídos.  Para obter mais informações, consulte [APIs ANSI preteridas](../mfc/deprecated-ansi-apis.md) e [Suporte para Unicode](../text/support-for-unicode.md).
