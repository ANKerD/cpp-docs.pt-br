# Posi&#231;&#227;o do arquivo no modo de acr&#233;scimo
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** Independente se o indicador da posição do arquivo de um fluxo de modo anexo for posicionado inicialmente no início ou no final do arquivo  
  
 Quando um arquivo é aberto no modo de anexação, o indicador de posição do arquivo aponta inicialmente para o final do arquivo.
