# Declaradores: n&#250;mero m&#225;ximo
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.5.4** o número máximo de declaradores que podem alterar uma aritmética, estrutura ou tipo de união  
  
 Microsoft C não limita o número de declaradores. O número é limitado apenas pela memória disponível.
