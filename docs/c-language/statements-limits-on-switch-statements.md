# Instru&#231;&#245;es: limites em instru&#231;&#245;es switch
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.6.4.2** O número máximo de valores **case** em uma instrução `switch`  
  
 Microsoft C não limita o número de valores **case**  em uma instrução **switch**.  O número é limitado somente pela memória disponível.
