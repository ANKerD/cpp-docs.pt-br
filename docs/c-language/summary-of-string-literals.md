# <a name="summary-of-string-literals"></a>Resumo de literais da cadeia de caracteres
*string-literal*:  
 **'** *s-char-sequence* opt**'**  
  
 **L'** *s-char-sequence* opt**'**  
  
 *s-char-sequence*:  
 *s-char*  
  
 *s-char-sequence s-char*  
  
 *s-char*:  
 qualquer membro do conjunto de caracteres de origem, exceto a marca de aspas duplas ("), a barra invertida (\\) ou *escape-sequence* de caractere de nova linha
