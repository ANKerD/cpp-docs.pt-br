# Pendentes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.3.5** O sinal do resto na divisão de um inteiro  
  
 O sinal do restante é o mesmo sinal do dividendo.  Por exemplo,  
  
```  
 50 / -6 == -8  
 50 % -6 ==  2  
-50 /  6 == -8  
-50 %  6 == -2  
```
