# Fun&#231;&#227;o fmod
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.5.6.4** Se um erro de domínio ocorrer ou zero for retornado quando a função `fmod` tiver um segundo argumento de zero  
  
 Quando a função `fmod` tem um segundo argumento de zero, a função retorna zero.
