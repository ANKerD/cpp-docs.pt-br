# <a name="c-primary-expressions"></a>Expressões primárias C
Os operandos em expressões são chamados de “expressões primárias”.  
  
## <a name="syntax"></a>Sintaxe  
 *primary-expression*:  
 *identifier*  
  
 *constant*  
  
 *string-literal*  
  
 **(**  *expression*  **)**  
  
 *expression*:  
 *assignment-expression*  
  
 *expression*  **,**  *assignment-expression*
