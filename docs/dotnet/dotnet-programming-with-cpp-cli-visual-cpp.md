# <a name="net-programming-with-ccli-visual-c"></a>Programação do .NET com C + + CLI (Visual C++)
Os tópicos a seguir descrevem como o Visual C++ pode direcionar a plataforma .NET.  
  
 Por padrão, os projetos CLR criados com o Visual Studio 2015 destinam do .NET Framework 4.5.2. Destino .NET Framework 4.6 quando você cria um novo projeto, no **novo projeto** caixa de diálogo, altere o framework de destino na lista suspensa no meio superior da caixa de diálogo. Para alterar a estrutura de destino para um projeto existente, fechar o projeto, edite o arquivo de projeto (. vcxproj) e altere o valor da versão do Framework de destino para 4.6. Próxima vez que abrir o projeto, as configurações tenham efeito.
