# <a name="fill-stlclr"></a>fill (STL/CLR)
Atribui o mesmo novo valor para cada elemento em um intervalo especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
template<class _FwdIt, class _Ty> inline  
    void fill(_FwdIt _First, _FwdIt _Last, const _Ty% _Val);  
```  
  
## <a name="remarks"></a>Comentários  
 Essa função se comporta como a função de biblioteca padrão C++ `fill`. Para obter mais informações, consulte [preenchimento](../standard-library/algorithm-functions.md#fill).  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** \<cliext/algoritmo >  
  
 **Namespace:** cliext
