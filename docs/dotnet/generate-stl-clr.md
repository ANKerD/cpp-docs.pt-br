# generate (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Atribui os valores gerados por um objeto da função a cada elemento em um intervalo.  
  
## Sintaxe  
  
```  
template<class _FwdIt, class _Fn0> inline  
    void generate(_FwdIt _First, _FwdIt _Last, _Fn0 _Func);  
```  
  
## Comentários  
 Essa função se comportará da mesma forma como a função `generate`STL.  Para obter mais informações, consulte [generate](../Topic/generate.md).
