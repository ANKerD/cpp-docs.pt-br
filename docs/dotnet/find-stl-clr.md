# find (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Localiza a posição da primeira ocorrência de um elemento em um intervalo que tem um valor especificado.  
  
## Sintaxe  
  
```  
template<class _InIt, class _Ty> inline  
    _InIt find(_InIt _First, _InIt _Last, const _Ty% _Val);  
```  
  
## Comentários  
 Essa função se comportará da mesma forma como a função `find`STL.  Para obter mais informações, consulte [find](../Topic/find%20\(STL\).md).
