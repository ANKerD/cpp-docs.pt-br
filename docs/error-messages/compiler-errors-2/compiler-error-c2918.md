# Erro do Compilador C2918
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'name': não não possível usar propriedades indexadas a superfície publicada de um tipo WinRT  
  
 Não há suporte para propriedades indexadas na superfície de publicada de um tipo WinRT.
