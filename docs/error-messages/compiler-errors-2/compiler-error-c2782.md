# <a name="compiler-error-c2782"></a>C2782 de erro do compilador
'declaração de ': o parâmetro de modelo 'identifier' é ambíguo  
  
 O compilador não pode determinar o tipo de um argumento de modelo.  
  
 O exemplo a seguir gera C2782:  
  
```  
// C2782.cpp  
template<typename T>  
void f(T, T) {}  
  
int main() {  
   f(1, 'c');   // C2782  
   // try the following line instead  
   // f<int>(1, 'c');  
}  
```  
  
 C2782 também pode ocorrer ao usar genéricos:  
  
```  
// C2782b.cpp  
// compile with: /clr  
generic<typename T> void gf(T, T) { }  
  
int main() {  
   gf(1, 'c'); // C2782  
   // try the following line instead  
   // gf<int>(1, 'c');  
}  
```
