# <a name="compiler-error-c3832"></a>C3832 de erro do compilador
'biblioteca de tipos ': biblioteca de tipos pareça ter sido compilada para ponteiros 32-bit; Altere o qualificador 'ptrsize'  
  
 Informações explícitas fornecidos com o `ptrsize` atributo o [#import](../../preprocessor/hash-import-directive-cpp.md) diretiva não concorda com o que o compilador encontrado na biblioteca de tipos.
