# <a name="compiler-error-c2693"></a>C2693 de erro do compilador
'operator': comparação ilegal para referências a gerenciada ou uma matriz do WinRT  
  
 Não é possível testar gerenciada ou matriz WinRT para qualquer tipo de desigualdade. Por exemplo, você pode testar para ver se gerenciado matrizes são iguais, mas você não pode testar para ver se uma matriz é maior ou menor que outro conjunto.
