# <a name="compiler-error-c3798"></a>C3798 de erro do compilador
'especificador de ': declaração de propriedade não pode ter o especificador de substituição (deve ser colocado na propriedade get/set métodos em vez disso)  
  
 Uma propriedade foi declarada incorretamente. Para saber mais, veja  
  
-   [propriedade](../../windows/property-cpp-component-extensions.md)  
  
-   [abstract](../../windows/abstract-cpp-component-extensions.md)  
  
-   [sealed](../../windows/sealed-cpp-component-extensions.md)  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera C3798  
  
```  
// C3798.cpp  
// compile with: /clr /c  
ref struct A {  
   property int Prop_1 abstract;   // C3798  
   property int Prop_2 sealed;   // C3798  
  
   // OK  
   property int Prop_3 {  
      virtual int get() abstract;  
      virtual void set(int i) abstract;  
   }  
  
   property int Prop_4 {  
      virtual int get() sealed;  
      virtual void set(int i) sealed;  
   }  
};  
```
