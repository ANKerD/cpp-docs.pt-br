# Erro do Compilador C2645
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nenhum nome qualificado do ponteiro para o membro \(encontrado “:: \*"\)  
  
 A declaração de um ponteiro para um membro não especificar uma classe.  
  
 O seguinte exemplo gera C2645:  
  
```  
// C2645.cpp  
class A {};  
int main() {  
   int B::* bp;   // C2645 B not defined  
   int A::* ap;   // OK  
}  
```
