# Erro do Compilador C2839
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tipo de retorno inválido “tipo” para o operador sobrecarregado “\-\>”  
  
 O operador de `->` deve retornar uma classe, struct, ou união, ou uma referência a um.
