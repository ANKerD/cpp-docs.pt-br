# Erro do Compilador C3855
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“classes: o parâmetro de tipo “param” é incompatível com a declaração  
  
 O compilador encontrou o modelo de nontype ou parâmetros genéricas com nomes diferentes.  Isso pode ocorrer quando um parâmetro especificado do modelo na definição de uma especialização de modelo é incompatível com sua declaração.  
  
 O seguinte exemplo gera C3855:  
  
```  
// C3855.cpp  
template <int N>  
struct C {  
   void f();  
};  
  
template <char N>  
void C<N>::f() {}   // C3855  
```  
  
 Solução possível:  
  
```  
// C3855b.cpp  
// compile with: /c  
template <int N>  
struct C {  
   void f();  
};  
  
template <int N>  
void C<N>::f() {}  
```  
  
 C3855 também pode ocorrer ao usar produtos genéricas:  
  
```  
// C3855c.cpp  
// compile with: /clr  
generic <class T>  
ref struct GC1 {  
   generic <class U>  
   ref struct GC2;  
};  
  
generic <class T>  
generic <class U>  
generic <class V>  
ref struct GC1<T>::GC2 { };   // C3855  
```  
  
 Solução possível:  
  
```  
// C3855d.cpp  
// compile with: /clr /c  
generic <class T>  
ref struct GC1 {  
   generic <class U>  
   ref struct GC2;  
};  
  
generic <class T>  
generic <class U>  
ref struct GC1<T>::GC2 { };  
```
