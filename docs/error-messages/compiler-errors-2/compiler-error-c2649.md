# C2649 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': não é um 'class\-key'  
  
 Uma classe, estrutura ou declaração de união usa uma marca incorreta.
