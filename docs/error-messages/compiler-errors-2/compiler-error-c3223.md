# <a name="compiler-error-c3223"></a>C3223 de erro do compilador
'property': não é possível aplicar 'typeid' para uma propriedade  
  
 Não é possível aplicar [typeid](../../windows/typeid-cpp-component-extensions.md) a uma propriedade.  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera C3223.  
  
```  
// C3223.cpp  
// compile with: /clr  
ref class R {  
public:  
   property int myprop;  
};  
  
int main() {  
   System::Type^ type2 = R::myprop::typeid;   // C3223  
}  
```
