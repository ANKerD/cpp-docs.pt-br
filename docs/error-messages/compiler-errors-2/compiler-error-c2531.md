# <a name="compiler-error-c2531"></a>C2531 de erro do compilador
'identifier': referência a um pouco ilegal de campo  
  
 Não são permitidas referências a campos de bits.  
  
 O exemplo a seguir gera C2531:  
  
```  
// C2531.cpp  
// compile with: /c  
class P {  
   int &b1 : 10;   // C2531  
   int b2 : 10;   // OK  
};  
```
