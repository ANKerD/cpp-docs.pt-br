# <a name="compiler-error-c2806"></a>C2806 de erro do compilador
'operador' tem muitos parâmetros formais  
  
 Um operador sobrecarregado tem muitos parâmetros.  
  
 O exemplo a seguir gera C2806:  
  
```  
// C2806.cpp  
// compile with: /c  
class X {  
public:  
   X operator++ ( int, int );   // C2806 more than 1 parameter  
   X operator++ ( int );   // OK  
} ;  
```
