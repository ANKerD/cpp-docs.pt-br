# <a name="compiler-error-c2971"></a>C2971 de erro do compilador
'class': o parâmetro de modelo 'param': 'arg': uma variável local não pode ser usada como um argumento de tipo não  
  
 Você não pode usar o nome ou endereço de uma variável local como um argumento de modelo.  
  
 O exemplo a seguir gera C2971:  
  
```  
// C2971.cpp  
template <int *pi>   
class Y {};  
  
int global_var = 0;  
  
int main() {  
   int local_var = 0;  
   Y<&local_var> aY;   // C2971  
   // try the following line instead  
   // Y<&global_var> aY;  
}  
```
