# <a name="compiler-error-c3015"></a>C3015 de erro do compilador
inicialização em instrução de OpenMP 'for' possui forma inadequada  
  
 Um `for` loop em uma instrução OpenMP deve ser totalmente e explicitamente especificado.  
  
 O exemplo a seguir gera C3015:  
  
```  
// C3015.cpp  
// compile with: /openmp  
int main()  
{  
   int i = 0, j = 10;  
  
   #pragma omp parallel  
   {  
      #pragma omp for  
      for (; i < 0; i += j)   // C3015  
      // Try the following line instead:  
      // for (i = 0; i < 0; i++)   
         --j;  
   }  
}  
```
