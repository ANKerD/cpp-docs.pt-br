# <a name="compiler-error-c3142"></a>C3142 de erro do compilador
'property_name': você não pode obter o endereço de uma propriedade  
  
 O endereço de uma propriedade não está disponível para o desenvolvedor.  
  
 O exemplo a seguir gera C3142:  
  
```  
// C3142_2.cpp  
// compile with: /clr  
using namespace System;  
ref class CSize {  
private:  
   property int Size {  
      int get();  
   }  
};  
  
int main() {  
    &CSize::Size; // C3142  
}  
```
