# Erro do Compilador C3276
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“palavra\-chave”: ignore do bloco de \_\_finally\/finally tem comportamento indefinido durante a manipulação de término  
  
 Esse erro é o mesmo que o aviso de [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md) .  No entanto, quando você estiver usando \/clr, essa condição não pode ser desabilitada com o pragma de [aviso](../../preprocessor/warning.md) .
