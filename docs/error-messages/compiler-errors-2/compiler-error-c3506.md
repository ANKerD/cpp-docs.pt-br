# <a name="compiler-error-c3506"></a>C3506 de erro do compilador
Não há nenhum typelib registrada para a ID da biblioteca 'id'  
  
 Um typelib não foi registrado corretamente. Use regsvr32.exe para registrar uma biblioteca de tipos.
