# Erro do Compilador C2739
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'number': gerenciado explícita ou WinRT dimensões da matriz devem ser entre 1 e 32  
  
 Uma dimensão de matriz não era entre 1 e 32.  
  
 O exemplo a seguir gera C2739 e mostra como corrigi\-lo:  
  
```  
// C2739.cpp  
// compile with: /clr  
int main() {  
   array<int, -1>^a;   // C2739  
   // try the following line instead  
   // array<int, 2>^a;  
}  
```
