# <a name="compiler-error-c3017"></a>C3017 de erro do compilador
teste de encerramento em instrução de OpenMP 'for' possui forma inadequada  
  
 Um `for` loop em uma instrução OpenMP deve ser totalmente e explicitamente especificado.  
  
 O exemplo a seguir gera C3017:  
  
```  
// C3017.cpp  
// compile with: /openmp  
int main()  
{  
   int i = 0, j = 10;  
  
   #pragma omp parallel  
   {  
      #pragma omp for  
      for (i = 0; i; ++i)   // C3017  
      // Try the following line instead:  
      // for (i = 0; i < 10; ++i)  
         ;  
   }  
}  
```
