# <a name="compiler-error-c2507"></a>C2507 de erro do compilador
'identifier': muitos modificadores virtual na classe base  
  
 Uma classe ou estrutura Ã© declarada como `virtual` mais de uma vez. Apenas um `virtual` modificador pode aparecer para cada classe de uma lista de classes base.  
  
 O exemplo a seguir gera C2507:  
  
```  
// C2507.cpp  
// compile with: /c

class B : virtual virtual public A {};   // C2507
```
