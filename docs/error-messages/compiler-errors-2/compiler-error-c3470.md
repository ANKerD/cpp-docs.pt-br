# <a name="compiler-error-c3470"></a>C3470 de erro do compilador
'type': uma classe não pode ter dois um indexador (propriedade indexada padrão) e um operador]  
  
 Um tipo não é possível definir um indexador padrão e um operador [].  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera C3470  
  
```  
// C3470.cpp  
// compile with: /clr  
using namespace System;  
  
ref class R {  
public:  
   property int default[int] {  
      int get(int i) {  
         return i+1;  
      }  
   }  
  
   int operator[](String^ s) { return Convert::ToInt32(s); }   // C3470  
};  
  
int main() {  
   R ^ r = gcnew R;  
   // return r[9] + r["32"] - 42;  
}  
```
