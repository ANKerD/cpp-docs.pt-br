# <a name="compiler-error-c3603"></a>C3603 de erro do compilador
'Symbol': tipo 'Type' ainda não tem suporte  
  
 Você tentou usar um tipo de dados que ainda não é suportado pelo runtime do .NET em um objeto gerenciado.
