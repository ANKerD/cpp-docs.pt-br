# Compiler Error C3456
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\[source\_annotation\_attribute\] não permitida em gerenciados ou declaração de classe WinRT  
  
 source\_annotation\_attribute é usado para definir atributos personalizados a serem usados pela análise de código.  
  
 Os atributos personalizados são úteis apenas se há um plug\-in correspondente que compreende o atributo.
