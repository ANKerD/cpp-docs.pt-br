# <a name="compiler-error-c3456"></a>C3456 de erro do compilador
[source_annotation_attribute] não permitida em gerenciados ou declaração de classe do WinRT  
  
 source_annotation_attribute é usado para definir atributos personalizados para ser usado pela análise de código.  
  
 Os atributos personalizados são úteis apenas se há um plug-in correspondente que compreende o atributo.
