# <a name="compiler-error-c2526"></a>C2526 de erro do compilador
'identifier1': função de vínculo a C não pode retornar classe C++ 'identifier2'  
  
 Uma função definida com vínculo a C não pode retornar um tipo definido pelo usuário.
