# Erro do Compilador C2526
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier1”: A função de vinculação C não pode retornar o identifier2 das classes C\+\+”  
  
 Uma função definida com esse tipo de C não pode retornar um tipo definido pelo usuário.
