# <a name="compiler-error-c3130"></a>C3130 de erro do compilador
Erro do compilador interno: Falha ao gravar bloco de código injetado em PDB  
  
 Esse erro ocorre se o compilador não pôde gravar um bloco de código injetado no arquivo. PDB. O motivo mais comum para a falha é a falta de espaço em disco.
