# Erro do Compilador C3130
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erro interno do compilador: falha na gravação injetou o bloco de código ao PDB  
  
 Esse erro ocorre se o compilador não gravou um bloco injetado de código ao arquivo .pdb.  O motivo mais comum para a falha é falta espaço em disco.
