# <a name="compiler-error-c3130"></a>Erro do compilador C3130

Erro do compilador interno: Falha ao gravar bloco de código injetado em PDB

Esse erro ocorre se o compilador não conseguiu gravar um bloco de código injetado para o arquivo. PDB. O motivo mais comum para a falha é a falta de espaço em disco.
