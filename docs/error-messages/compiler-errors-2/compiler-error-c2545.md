# <a name="compiler-error-c2545"></a>C2545 de erro do compilador
'operator': não é possível encontrar operador sobrecarregado  
  
 O operador não pode ser usado com os operandos fornecidos. Você deve fornecer um operador sobrecarregado com os operandos necessários.  
  
 Esse erro pode ser causado se operandos tem um tipo incorreto.  
  
 Esse erro pode ser corrigido, se você definir um operador de conversão ou o construtor que usa um único parâmetro.
