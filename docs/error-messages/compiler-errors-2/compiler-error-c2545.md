# <a name="compiler-error-c2545"></a>C2545 de erro do compilador
'operator': não é possível localizar sobrecarregado operador  
  
 O operador não pode ser usado com os operandos fornecidos. Você deve fornecer um operador sobrecarregado com os operandos necessários.  
  
 Esse erro pode ser causado se operandos tiverem o tipo incorreto.  
  
 Esse erro pode ser corrigido, se você definir um operador de conversão ou um construtor que aceita um único parâmetro.
