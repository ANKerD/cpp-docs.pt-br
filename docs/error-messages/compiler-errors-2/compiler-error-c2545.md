# Erro do Compilador C2545
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

operador “”: não é possível encontrar o operador sobrecarregado  
  
 O operador não pode ser usado com os operandos fornecidos.  Você deve fornecer um operador sobrecarregado com operandos necessários.  
  
 Esse erro pode ser causado se os operandos tiverem o tipo incorreto.  
  
 Esse erro pode ser corrigido se você definir um operador ou um construtor de conversão que usem um único parâmetro.
