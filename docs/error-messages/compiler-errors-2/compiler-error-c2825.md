# Erro do Compilador C2825
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

var: deve ser uma classe ou um namespace quando seguido por “::”  
  
 Uma tentativa foi feita para formar um nome qualificado.  
  
 Por exemplo, verifique se seu código não contém uma declaração de função no nome da função começa com o ::.  
  
## Exemplo  
 O seguinte exemplo gera C2825:  
  
```  
// C2825.cpp  
typedef int i;  
int main() {  
   int* p = new int;  
   p->i::i();   // C2825  
   // try the following line instead  
   // p->i::~i();  
}  
```
