# <a name="compiler-error-c3800"></a>C3800 de erro do compilador
'declaração de ': não é possível misturar propriedades e eventos  
  
 Você não pode declarar uma construção de uma propriedade e um evento.  
  
 C3800 só está acessível usando a opção de compilador obsoleto **/CLR: oldSyntax**.
