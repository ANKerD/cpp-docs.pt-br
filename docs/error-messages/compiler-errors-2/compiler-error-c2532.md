# Erro do Compilador C2532
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identificador”: modificador ilegal para referência  
  
 A referência foi alterada.  As referências não podem ser alteradas para fazer referência a outro objeto.  Use um ponteiro em vez disso.
