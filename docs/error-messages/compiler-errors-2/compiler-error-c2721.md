# Erro do Compilador C2721
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“especificador”: especificador de armazenamento classe ilegal entre a palavra\-chave do operador e o tipo  
  
 As conversões de tipos definidos pelo usuário se aplicam a todas as classes de armazenamento, portanto você não pode especificar uma classe de armazenamento em uma conversão de tipos.
