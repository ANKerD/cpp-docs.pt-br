# Erro do Compilador C2974
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

argumento inválido “número” do tipo, tipo esperado  
  
 O argumento genérico ou modelo não corresponde à declaração genérico ou modelo.  Um tipo deve aparecer dentro dos colchetes angulares.  Verifique se a definição genérico ou de modelo para encontrar os tipos corretos.  
  
 O seguinte exemplo gera C2974:  
  
```  
// C2974.cpp  
// C2974 expected  
template <class T>  
struct TC {};  
  
template <typename T>  
void tf(T){}  
  
int main() {  
   // Delete the following 2 lines to resolve  
   TC<1>* tc;  
   tf<"abc">("abc");  
  
   TC<int>* tc;  
   tf<const char *>("abc");  
}  
```  
  
 C2974 também pode ocorrer ao usar produtos genéricas:  
  
```  
// C2974b.cpp  
// compile with: /clr  
// C2974 expected  
using namespace System;  
generic <class T>  
ref struct GCtype {};  
  
generic <typename T>  
void gf(T){}  
  
int main() {  
   // Delete the following 2 lines to resolve  
   GCtype<"a">^ gc;  
   gf<"a">("abc");  
  
   // OK  
   GCtype<int>^ gc;  
   gf<String ^>("abc");  
}  
```
