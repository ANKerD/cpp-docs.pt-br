# C2656 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function': função não pode ser um campo de bits  
  
 Uma função é declarada como um membro de um campo de bits.  
  
 Esse erro pode ser causado por um erro de sintaxe em uma lista do inicializador de construtor.
