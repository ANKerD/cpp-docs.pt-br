# Erro do Compilador C2855
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opção de linha de comando “default” inconsistente com cabeçalho pré\-compilado  
  
 A opção de linha de comando determinada difere da opção usada para criar o cabeçalho pré\-compilado.  Gerencia o cabeçalho pré\-compilado com a opção de linha de comando determinada.
