# <a name="compiler-error-c2855"></a>C2855 de erro do compilador
opção de linha de comando 'option' inconsistente com o cabeçalho pré-compilado  
  
 A opção de linha de comando fornecida difere da opção usada para criar o cabeçalho pré-compilado. Regenere o cabeçalho pré-compilado com a opção de linha de comando fornecida.
