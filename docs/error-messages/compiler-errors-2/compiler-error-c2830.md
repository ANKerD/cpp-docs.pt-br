# <a name="compiler-error-c2830"></a>Erro do compilador C2830

somente os parâmetros de posicionamento para 'operator new' podem ter valores padrão

Os parâmetros formais padrão para [operador new](../../standard-library/new-operators.md#op_new) não pode ter valores padrão. Apenas parâmetros de posicionamento definidas pelo usuário podem especificar os padrões.
