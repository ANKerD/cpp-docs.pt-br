# <a name="compiler-error-c2830"></a>C2830 de erro do compilador
apenas parâmetros de posicionamento para 'operator new' podem possuir valores padrão  
  
Os parâmetros padrão formais para [operador new](../../standard-library/new-operators.md#operator_new) não podem ter valores padrão. Somente os parâmetros de posicionamento definidos pelo usuário podem especificar os padrões.
