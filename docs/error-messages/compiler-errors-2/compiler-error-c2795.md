# Erro do Compilador C2795
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o “super::function” não é uma função de membro  
  
 Essa mensagem de erro aparece sempre que você tentar usar [super](../../cpp/super.md) para acessar um membro diferente de uma função de membro.
