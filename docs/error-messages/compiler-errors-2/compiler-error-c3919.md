# Erro do Compilador C3919
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“event\_method”: a função deve ter o tipo “tipo”  
  
 Um método de acessador de evento não foi declarado corretamente.  
  
 Para obter mais informações sobre os eventos, consulte [evento](../../windows/event-cpp-component-extensions.md).  
  
 O seguinte exemplo gera C3919:  
  
```  
// C3919.cpp  
// compile with: /clr /c  
using namespace System;  
delegate void D(String^);  
ref class R {  
   event D^ e {  
      int add(int);   // C3919  
      int remove(int);   // C3919  
  
      void add(D^);   // OK  
      void remove(D^);   // OK  
   }  
};  
```
