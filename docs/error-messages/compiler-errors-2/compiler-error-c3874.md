# <a name="compiler-error-c3874"></a>C3874 de erro do compilador
tipo de retorno de 'function' deve ser 'int' em vez de 'type'  
  
 Uma função não tem o tipo de retorno esperado pelo compilador.  
  
 O exemplo a seguir gera C3874:  
  
```  
// C3874b.cpp  
double main()  
{   // C3874  
}  
```
