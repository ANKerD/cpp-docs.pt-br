# <a name="compiler-error-c2689"></a>C2689 de erro do compilador
'function': uma função friend não pode ser definida em uma classe de local  
  
 Você pode declarar, mas não definir uma função friend em uma classe local.  
  
 O exemplo a seguir gera C2689:  
  
```  
// C2689.cpp  
// compile with: /c  
void g() {  
   void f2();  
   class X {  
      friend void f2(){}   // C2689  
      friend void f2();   // OK  
   };  
}  
```
