# <a name="compiler-error-c2904"></a>C2904 de erro do compilador
'identifier': nome já foi usado para um modelo no escopo atual  
  
 Verifique o código para nomes duplicados.  
  
 O exemplo a seguir gera C2904:  
  
```  
// C2904.cpp  
// compile with: /c  
void X();  // X is declared as a function  
template<class T> class X{};  // C2904  
```
