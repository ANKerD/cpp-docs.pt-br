# <a name="compiler-error-c3159"></a>C3159 de erro do compilador
'ponteiro': matriz de ponteiros para tipo de valor não pode ser declarado  
  
 Uma matriz de ponteiros para um tipo de valor não pode ser declarada.  
  
 C3159 só está acessível usando a opção de compilador obsoleto **/CLR: oldSyntax**.
