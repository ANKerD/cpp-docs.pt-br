# <a name="compiler-error-c3135"></a>C3135 de erro do compilador
'property': uma propriedade não pode ter uma 'const' ou 'volatile' de tipo  
  
 O [const](../../cpp/const-cpp.md) e [volátil](../../cpp/volatile-cpp.md) palavras-chave não são permitidas em Propriedades.
