# <a name="compiler-error-c2903"></a>Erro do compilador C2903

'identifier': símbolo não é um modelo de classe nem um modelo de função

Código de tentativas de instanciação explícita de algo que não é um modelo.

O exemplo a seguir gera C2903:

```
// C2903.cpp
// compile with: /c
namespace N {
   template<class T> class X {};
   class Y {};
}
void g() {
   N::template Y y;   // C2903
   N::X<N::Y> y;   // OK
}
```

C2903 também podem ocorrer ao usar genéricos:

```
// C2903b.cpp
// compile with: /clr /c
namespace N {
   class Y {};
   generic<class T> ref class Z {};
}

void f() {
   N::generic Y y;   // C2903
   N:: generic Z<int>^ z;   // OK
}
```
