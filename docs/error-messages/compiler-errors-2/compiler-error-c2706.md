# Erro do Compilador C2706
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\_\_except ilegal sem corresponder \_\_try \(sem “}” no bloco \_\_try?\)  
  
 O compilador não encontrou uma chave de fechamento de um bloco de `__try` .  
  
 O seguinte exemplo gera C2706:  
  
```  
// C2706.cpp  
int main() {  
   __try {  
      void f();  
   // C2706  } missing here  
   __except(GetExceptionCode() == 0x0) {  
   }  
}  
```
