# <a name="compiler-error-c2692"></a>C2692 de erro do compilador
'nome_da_função': protótipo totalmente funções necessárias no compilador C com o ' / clr' opção  
  
 Quando a compilação para .NET de código gerenciado, o compilador C requer declarações de função ANSI. Além disso, se uma função não usa nenhum parâmetro, ele deve declarar explicitamente `void` como o tipo de parâmetro.
