# C3303 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'attribute': atributo só pode ser usado em 'uso'  
  
 Foi feita uma tentativa para usar um atributo que não é válido.
