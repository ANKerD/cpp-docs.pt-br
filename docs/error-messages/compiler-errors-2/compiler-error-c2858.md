# Erro do Compilador C2858
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opção de linha de comando “\/Yc \(\/Fdfilename\)” inconsistente com cabeçalho pré\-compilado, que usou “\/Fdfilename”  
  
 O base de dados do programa especificado pela opção pré\-compilado de cabeçalho de uso \([\/Yu](../../build/reference/yu-use-precompiled-header-file.md)\) não é aquele especificado por anterior cria a opção pré\-compilado de cabeçalho \([\/Yc](../../build/reference/yc-create-precompiled-header-file.md)\).
