# <a name="compiler-error-c2858"></a>C2858 de erro do compilador
opção de linha de comando ' /Yc (/ Fdfilename)' inconsistente com o cabeçalho pré-compilado, que usado ' / Fdfilename'  
  
 O banco de dados do programa especificado pelo cabeçalho pré-compilado Use ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) opção não é especificado pelo anterior criar pré-compilados cabeçalho ([/Yc](../../build/reference/yc-create-precompiled-header-file.md)) opção.
