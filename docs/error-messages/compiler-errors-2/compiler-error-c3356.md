# <a name="compiler-error-c3356"></a>C3356 de erro do compilador
'attribute': não é possível chamar um atributo de multicast com um nome totalmente qualificado  
  
 Um atributo que é processado por mais de um processo, por exemplo, o compilador e o provedor ATL, foi especificado incorretamente.
