# Erro do Compilador C2713
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

somente um formulário de manipulação de exceção permitido por função  
  
 Você não pode usar a manipulação de exceção estruturado \(`__try`\/`__except`\) e a manipulação de exceção C\+\+ \(`try`\/`catch`\) no mesmo funções.
