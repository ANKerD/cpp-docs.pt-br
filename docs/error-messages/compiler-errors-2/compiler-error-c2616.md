# Erro do Compilador C2616
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“conversão”: não podem converter implicitamente um tipo de non\-lvalue “\- 1 " um “do tipo 2 " que não é const  
  
 Uma referência não pode ser inicializada de um non\-lvalue.  
  
 Este é um erro na compatibilidade ANSI \([\/Za](../../build/reference/za-ze-disable-language-extensions.md)\) e um aviso em extensões da Microsoft \(**\/Ze**\).
