# Erro do Compilador C3180
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“nome do tipo”: o nome exceder o limite de metadados de caracteres de “limite”  
  
 O compilador truncados o nome para um gerenciado em metadados.  O truncamento fará o tipo inutilizável pela política de `#using` \(ou o equivalente em outro idioma.\)  
  
 O limite de classificação nome inclui todas as qualificações de namespace.
