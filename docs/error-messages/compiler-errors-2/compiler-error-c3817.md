# <a name="compiler-error-c3817"></a>C3817 de erro do compilador
'declaração de ': propriedade só pode ser aplicada a uma função  
  
 O `property` palavra-chave só pode ser aplicado a uma definição de função.  
  
 C3817 só está acessível usando a opção de compilador obsoleto **/CLR: oldSyntax**.
