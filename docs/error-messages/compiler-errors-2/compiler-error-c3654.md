# Erro do Compilador C3654
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“texto”: erro de sintaxe na substituição explícita  
  
 Uma cadeia de caracteres de forma inesperada estava em uma substituição explícita.  Para obter mais informações, consulte [Substituições explícitas](../../windows/explicit-overrides-cpp-component-extensions.md).  
  
 O seguinte exemplo gera C3654:  
  
```  
// C3654.cpp  
// compile with: /clr /c  
public ref struct B {  
   virtual void f() = 0;  
   virtual void g() = 0;  
   virtual void h() = 0;  
};  
  
public ref struct Q : B {  
   virtual void f() = B::f, 3 {}   // C3654  
   // try the following line instead  
   // virtual void g() = B::g, B::h {}  
};  
```
