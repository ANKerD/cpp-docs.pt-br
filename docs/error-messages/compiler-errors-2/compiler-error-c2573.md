# C2573 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class': não é possível excluir ponteiros para objetos desse tipo; a classe não tem nenhuma sobrecarga de posicionamento não 'operador Delete'.  
  
 A classe tem um operador de exclusão não posicionamento.
