# Erro do Compilador C2651
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“tipo de dados”: a esquerda do operador “” deve ser uma classe, uma estrutura ou uma união  
  
 Para usar um parâmetro do modelo como se for uma classe, especialize o modelo da classe com uma classe em vez de um tipo integral.
