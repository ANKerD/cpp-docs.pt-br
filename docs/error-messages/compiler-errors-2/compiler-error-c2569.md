# Erro do Compilador C2569
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“EnumOrUnion”: o enum\/união não pode ser usado como uma classe base  
  
 Se você deve derivar um tipo de união ou de enumeração especificada, altere a união ou enumeração a uma classe ou estrutura.  
  
 O seguinte exemplo gera C2569:  
  
```  
// C2569.cpp  
// compile with: /c  
union ubase {};  
class cHasPubUBase : public ubase {};   // C2569  
// OK  
struct sbase {};  
class cHasPubUBase : public sbase {};  
```
