# Erro do Compilador C3721
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“assinatura”: assinatura incompatível do evento  
  
 Um evento foi declarado incorretamente.  Para obter mais informações, consulte [\_\_event](../../cpp/event.md).  
  
 C3721 é somente **\/clr:oldSyntax**de utilização possível acessá\-lo.  
  
## Exemplo  
 O exemplo a seguir produz C3721.  
  
```  
// C3721.cpp  
// compile with: /clr:oldSyntax /c  
using namespace System;  
  
public __delegate void MyDel();  
  
public __gc class X {  
   __event void add_E1();      // C3721  
   // try the following line instead  
   // __event void add_E1(MyDel * p);  
  
   __event void remove_E1();   // C3721  
   // __event void remove_E1(MyDel * p);  
   // try the following line instead  
  
   __event void raise_E1();  
};  
```
