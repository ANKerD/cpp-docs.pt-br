# Erro do Compilador C2667
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

função “”: nenhuma de sobrecargas do número têm uma melhor conversão  
  
 Uma chamada de função sobrecarregada é ambígua e não pode ser resolvida.  
  
 A conversão necessária para corresponder aos parâmetros reais na chamada de função a uma das funções sobrecarregadas deve ser restrita melhorar das conversões necessárias por todas as funções sobrecarregadas restantes.  
  
 Consulte o artigo da Base de Dados de Conhecimento Q240869 para obter mais informações sobre como ordenar parcial de modelos da função.
