# <a name="compiler-error-c3096"></a>C3096 de erro do compilador
'attribute': atributo é permitido em membros de classes de atributo apenas dados  
  
 Um atributo foi aplicado incorretamente.  
  
 Para obter mais informações, consulte [atributos definidos pelo usuário](../../windows/user-defined-attributes-cpp-component-extensions.md).
