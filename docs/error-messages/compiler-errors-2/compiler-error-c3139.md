# <a name="compiler-error-c3139"></a>C3139 de erro do compilador
'struct': não é possível exportar um UDT sem membros  
  
 Você tentou aplicar o [exportar](../../windows/export.md) atributo a uma UDT vazio (tipo definido pelo usuário). Por exemplo:  
  
```  
// C3139.cpp  
#include "unknwn.h"  
[emitidl];  
[module(name=xx)];  
  
[export] struct MyStruct {   // C3139 empty type  
};  
int main(){}  
```
