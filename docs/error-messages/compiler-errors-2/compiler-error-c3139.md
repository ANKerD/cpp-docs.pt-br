# Erro do Compilador C3139
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“struct”: não é possível exportar um UDT sem membros  
  
 Você tentou aplique o atributo de [exportação](../../windows/export.md) vazio a um UDT \(tipo definido pelo usuário\).  Por exemplo:  
  
```  
// C3139.cpp  
#include "unknwn.h"  
[emitidl];  
[module(name=xx)];  
  
[export] struct MyStruct {   // C3139 empty type  
};  
int main(){}  
```
