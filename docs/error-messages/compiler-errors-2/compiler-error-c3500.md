# Erro do Compilador C3500
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ProgID inválido “progid”  
  
 Um progid inválido foi especificado na instrução de `#import` .  Verifique no Registro do Windows para garantir que você está especificando um progid válido.
