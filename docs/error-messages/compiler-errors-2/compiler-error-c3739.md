# Erro do Compilador C3739
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“classes: a sintaxe tem suporte apenas quando o parâmetro “layout\_dependent” de event\_receiver é true  
  
 Você tentou conectar uma interface inteira de eventos mas `layout_dependent` no atributo de [event\_receiver](../../windows/event-receiver.md) não é válido; você deve conectar um único evento de cada vez.  
  
 O seguinte exemplo gera C3739:  
  
```  
// C3739.cpp  
struct A  
{  
   __event void e();  
};  
  
// event_receiver is implied  
// [ event_receiver(layout_dependent=false)]  
  
// use the following line instead  
// [event_receiver(com, layout_dependent=true), coclass ]  
struct B  
{  
   void f();  
   B(A* a)  
   {  
      __hook(A, a, &B::f);   // C3739  
      // use the following line instead to hook a single event  
      // __hook(&A::e, a, &B::f);  
   }  
};  
  
int main()  
{  
}  
```
