# Erro do Compilador C2791
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

uso ilegal de “super”: as classes “não têm nenhuma classes base  
  
 A palavra\-chave [super](../../cpp/super.md) foi usado dentro do contexto de uma função de membro de uma classe que não tenha classes base.  
  
 O seguinte exemplo gera C2791:  
  
```  
// C2791.cpp  
struct D {  
   void mf() {  
      __super::mf();   // C2791  
   }  
};  
```
