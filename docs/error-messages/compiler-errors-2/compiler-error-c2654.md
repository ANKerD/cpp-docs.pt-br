# Erro do Compilador C2654
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identificador”: tentativa de acessar o membro fora de uma função de membro  
  
 Um membro é acessado em uma declaração.  Os dados de membro podem ser acessados somente em funções de membro.  
  
 Esse erro pode ser causado ao tentar inicializar variáveis em uma instrução.  Use um construtor para essa finalidade.
