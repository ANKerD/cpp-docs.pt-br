# <a name="compiler-error-c2654"></a>C2654 de erro do compilador
'identifier': tentativa de acesso de membro fora de uma função de membro  
  
 Um membro é acessado em uma declaração. Dados de membro podem ser acessados somente funções de membro.  
  
 Esse erro pode ocorrer ao tentar inicializar variáveis em uma declaração. Use um construtor para essa finalidade.
