# Erro do Compilador C2948
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

instanciação explícita; especificador “” especificador da classe de armazenamento não permitido na especialização  
  
 Você não pode usar especificadores de armazenamento classe \(como `extern`\) em uma especialização de uma classe do modelo que anteriormente seja criada uma instância explicitamente.
