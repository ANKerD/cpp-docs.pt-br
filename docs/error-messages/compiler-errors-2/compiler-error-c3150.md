# <a name="compiler-error-c3150"></a>C3150 de erro do compilador
'element': 'attribute' só pode ser aplicado a classe, interface, array ou ponteiro  
  
`__gc`só pode ser usado em uma classe, interface ou matriz.  
  
 C3150 só está acessível usando a opção de compilador obsoleto **/CLR: oldSyntax**.
