# <a name="compiler-error-c3150"></a>Erro do compilador C3150

'element': 'attribute' só pode ser aplicada a classe, interface, array ou ponteiro

`__gc` só pode ser usado em uma classe, interface ou matriz.

C3150 só está acessível usando a opção de compilador obsoletos **/CLR: oldSyntax**.
