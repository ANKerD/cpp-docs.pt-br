# <a name="compiler-error-c2996"></a>C2996 de erro do compilador
'function': definição de modelo de função recursiva  
  
 Uma definição de função tenta criar uma instância de sua classe de modelo de raiz. Instanciações do modelo recursivas não são permitidas.
