# <a name="compiler-error-c2932"></a>C2932 de erro do compilador
'class': tipo de id de classe redefinido como um membro de dados de 'Identificador'  
  
 Você não pode usar uma classe genérica ou o modelo como um membro de dados.  
  
 O exemplo a seguir gera C2932:  
  
```  
// C2932.cpp  
// compile with: /c  
template<class T>   
struct TC {};   
  
struct MyStruct {  
   int TC<int>;   // C2932  
   int TC;   // OK  
};  
```  
  
 C2932 também pode ocorrer ao usar genéricos:  
  
```  
// C2932b.cpp  
// compile with: /clr /c  
generic<class T>  
ref struct GC {};  
  
struct MyStruct {  
   int GC<int>;   // C2932  
   int GC;   // OK  
};  
```
