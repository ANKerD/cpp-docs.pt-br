# C2700 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': não pode ser gerada \(\/W4 de uso para obter mais informações\)  
  
 O objeto não pode ser lançado. Compile com [\/W4](../../build/reference/compiler-option-warning-level.md) para mais informações de diagnóstico.
