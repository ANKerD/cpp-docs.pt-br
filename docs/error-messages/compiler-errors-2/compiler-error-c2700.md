# <a name="compiler-error-c2700"></a>C2700 de erro do compilador
'Identificador': não pode ser lançado (use /W4 para obter mais informações)  
  
 O objeto não pode ser gerado. Compilar com [/W4](../../build/reference/compiler-option-warning-level.md) para mais informações de diagnóstico.
