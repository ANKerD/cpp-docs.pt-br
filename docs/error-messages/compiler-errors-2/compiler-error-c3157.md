# Erro do Compilador C3157
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

O atributo de ParamArray só pode ser aplicado ao parâmetro do último  
  
 O atributo de [ParamArray](https://msdn.microsoft.com/en-us/library/system.paramarrayattribute.aspx) ter sido aplicado ao parâmetro incorreto.
