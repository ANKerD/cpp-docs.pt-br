# <a name="compiler-error-c3886"></a>C3886 de erro do compilador
'var': um membro de dados literais deve ser inicializado  
  
 A [literal](../../windows/literal-cpp-component-extensions.md) variável deve ser inicializada quando é declaraed.  
  
 O exemplo a seguir gera C3886:  
  
```  
// C3886.cpp  
// compile with: /clr /c  
ref struct Y1 {  
   literal int staticConst;   // C3886  
   literal int staticConst2 = 0;   // OK  
};  
```
