# <a name="compiler-error-c3815"></a>C3815 de erro do compilador
tipo de retorno do método 'get_accessor' deve corresponder o tipo do último parâmetro de um setter  
  
 Ao declarar propriedades, o valor de retorno de `get_accessor` método deve coincidir com o último parâmetro na declaração do método de acessador de conjunto.  
  
 C3815 só está acessível usando a opção de compilador obsoleto **/CLR: oldSyntax**.
