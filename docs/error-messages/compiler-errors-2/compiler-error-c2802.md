# <a name="compiler-error-c2802"></a>C2802 de erro do compilador
membro estático 'operador' não tem nenhum parâmetro formal  
  
 Um operador declarado por um `static` a função de membro deve ter pelo menos um parâmetro.  
  
 O exemplo a seguir gera C2802:  
  
```  
// C2802.cpp  
// compile with: /clr /c  
ref class A {  
   static operator+ ();   // C2802  
   static operator+ (A^, A^);   // OK  
};  
```
