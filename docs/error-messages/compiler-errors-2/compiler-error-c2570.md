# <a name="compiler-error-c2570"></a>C2570 de erro do compilador
'identifier': union não pode ter classes base  
  
 Uma união deriva de uma classe, estrutura ou união. Isso não é permitido. Em vez disso, declare o tipo derivado como uma classe ou estrutura.  
  
 O exemplo a seguir gera C2570:  
  
```  
// C2570.cpp  
// compile with: /c  
class base {};  
union hasPubBase : public base {};   // C2570  
union hasNoBase {};   // OK  
```
