# Erro do Compilador C2568
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier1”: conseguir resolver a sobrecarga da função  
  
 O compilador não pode determinar quais sobrecarregada função a ser chamada.  Os parâmetros reais passados para a função devem ser convertidos para corresponder aos parâmetros formais para uma função de sobrecarregada, mas nenhuma correspondência é melhor maneira do que qualquer outro.
