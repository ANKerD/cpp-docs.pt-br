# Erro do Compilador C3501
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não há nenhum typelib registrado para ProgID “progid”  
  
 A ID da classe para um progid fornecido não tem uma biblioteca associado do tipo.  Consequentemente, você não pode passar esse progid à instrução de `#import` .
