# Erro do Compilador C3197
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“palavra\-chave”: pode ser usado apenas nas definições  
  
 Uma palavra\-chave foi usado em uma declaração mas só é válida em uma definição.  
  
 O seguinte exemplo gera C3197:  
  
```  
// C3197.cpp  
// compile with: /clr /c  
ref struct R abstract;   // C3197  
ref struct R abstract {};   // OK  
  
public ref class MyObject;   // C3197  
ref class MyObject;   // OK  
public ref class MyObject {};   // OK  
```
