# <a name="compiler-error-c2829"></a>C2829 de erro do compilador
'operador' não pode ter uma lista de parâmetros variáveis  
  
 Apenas dois operadores podem usar listas de parâmetros de variável:  
  
-   Chamada de função`()`  
  
-   `new`
