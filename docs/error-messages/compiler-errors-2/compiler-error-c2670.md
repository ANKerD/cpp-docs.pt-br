# <a name="compiler-error-c2670"></a>C2670 de erro do compilador
'identifier': o modelo de função não é possível converter o número do parâmetro de tipo 'type'  
  
 Um parâmetro não pôde ser convertido para o tipo solicitado.  
  
 Esse erro pode ser corrigido, se você criar uma conversão explícita.
