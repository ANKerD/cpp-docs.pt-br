# Erro do Compilador C2670
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identificador”: o modelo da função não é possível converter o número do parâmetro do tipo “tipo”  
  
 Um parâmetro não pode ser convertido no tipo necessário.  
  
 Esse erro pode ser corrigido se você criar uma conversão explícita.
