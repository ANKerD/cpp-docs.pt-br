# <a name="compiler-error-c3213"></a>C3213 de erro do compilador
classe base 'base_type' é menos acessível que 'derived_type'  
  
 Um tipo que serão visível de um assembly deve usar classes base publicamente visíveis.  
  
 O exemplo a seguir gera C3213:  
  
```  
// C3213.cpp  
// compile with: /clr  
private ref struct privateG {  
public:  
   int i;  
};  
  
public ref struct publicG {  
public:  
   int i;  
};  
  
public ref struct V : public privateG {   // C3213  
public:  
   int j;  
};  
  
public ref struct W: public publicG {   // OK  
public:  
   int j;  
};  
```
