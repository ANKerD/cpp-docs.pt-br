# <a name="compiler-error-c3857"></a>C3857 de erro do compilador
'type': não são permitidas várias listas de parâmetros de tipo  
  
 Mais de um modelo ou genérico foi especificado para o mesmo tipo, que não é permitido.  
  
 O exemplo a seguir gera C3857:  
  
```  
// C3857.cpp  
template <class T, class TT>  
template <class T2>    // C3857  
struct B {};  
```  
  
 Resolução possível:  
  
```  
// C3857b.cpp  
// compile with: /c  
template <class T, class TT, class T2>   
struct B {};  
```  
  
 C3857 também pode ocorrer ao usar genéricos:  
  
```  
// C3857c.cpp  
// compile with: /clr  
generic <typename T>  
generic <typename U>  
ref class GC;   // C3857  
```  
  
 Resolução possível:  
  
```  
// C3857d.cpp  
// compile with: /clr /c  
generic <typename U>  
ref class GC;  
```
