# Erro do Compilador C2817
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o tipo de retorno de “excluir operador” deve ser “nulo”  
  
 Uma função sobrecarregada de [a exclusão do operador](../Topic/operator%20delete%20\(%3Cnew%3E\).md) não pode retornar um valor.
