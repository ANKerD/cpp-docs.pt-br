# <a name="compiler-error-c2862"></a>C2862 de erro do compilador
'interface': uma interface pode ter somente membros públicos  
  
 Protegidos e membros privados podem ser acessados somente de outras funções de membro. Esses membros são sem uso em uma interface, pois ela não pode fornecer implementações para qualquer um dos seus membros.  
  
 O exemplo a seguir gerará C2862:  
  
```  
// C2862.cpp  
// compile with: /c  
#include <unknwn.h>  
  
[object, uuid="60719E20-EF37-11D1-978D-0000F805D73B"]  
__interface IMyInterface {  
   HRESULT mf1(void);   // OK  
protected:  
   HRESULT mf2(int *b);   // C2862  
private:  
   HRESULT mf3(int *c);   // C2862  
};  
```
