# Erro do Compilador C2862
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

contadores de “interface”: uma interface só pode ter membros públicos  
  
 Os membros protegidos e particulares podem ser acessados somente de outras funções de membro.  Esses membros não há nenhum uso em uma interface, desde que não pode fornecer implementações de qualquer um de seus membros.  
  
 O exemplo a seguir gerará C2862:  
  
```  
// C2862.cpp  
// compile with: /c  
#include <unknwn.h>  
  
[object, uuid="60719E20-EF37-11D1-978D-0000F805D73B"]  
__interface IMyInterface {  
   HRESULT mf1(void);   // OK  
protected:  
   HRESULT mf2(int *b);   // C2862  
private:  
   HRESULT mf3(int *c);   // C2862  
};  
```
