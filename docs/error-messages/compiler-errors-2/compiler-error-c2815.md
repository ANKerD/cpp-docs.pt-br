# <a name="compiler-error-c2815"></a>C2815 de erro do compilador
'operator delete': primeiro parâmetro formal deve ser ' void *', mas 'param' foi usada  
  
 Qualquer definidos pelo usuário [operador delete](../../standard-library/new-operators.md#op_delete) função deve utilizar um primeiro parâmetro formal do tipo `void *`.  
  
 O exemplo a seguir gera C2815:  
  
```  
// C2815.cpp  
// compile with: /c  
class CMyClass {  
public:  
   void mf1(int *a);  
   void operator delete(CMyClass *);   // C2815  
   void operator delete(void *);   
};  
```
