# Erro do Compilador C3838
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não pode explicitamente herdar do tipo” “  
  
 `type` especificado não pode atuar como uma classe base em nenhuma classe.  
  
 O seguinte exemplo gera C3838:  
  
```  
// C3838a.cpp  
// compile with: /clr /c  
public ref class B : public System::Enum {};   // C3838  
```  
  
 O seguinte exemplo gera C3838:  
  
```  
// C3838b.cpp  
// compile with: /clr:oldSyntax /c  
public __gc class B : public System::ValueType {};   // C3838  
```
