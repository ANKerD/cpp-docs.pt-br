# Erro do Compilador C3645
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

função “”: o \_\_clrcall não pode ser usada em funções criadas em código nativo  
  
 A presença de algumas palavras\-chave em uma função fará a função a ser compilada ao nativo.  
  
## Exemplo  
 O exemplo a seguir produz C3645.  
  
```  
// C3645.cpp  
// compile with: /clr /c  
#pragma unmanaged   
int __clrcall dog() {}   // C3645  
```
