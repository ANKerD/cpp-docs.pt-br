# Erro do Compilador C2873
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

símbolo “”: o símbolo não pode ser usado em uma declaração usados  
  
 Uma política de `using` está faltando uma palavra\-chave de [namespace](../Topic/namespace%20Declaration.md) .  Isso faz com que o compilador interprete incorretamente o código como [usando a declaração](../../cpp/using-declaration.md) em vez de [usando a diretiva](../../misc/using-directive-cpp.md).
