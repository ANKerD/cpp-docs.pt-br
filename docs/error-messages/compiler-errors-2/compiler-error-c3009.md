# <a name="compiler-error-c3009"></a>C3009 de erro do compilador
'Rótulo': pular para o bloco estruturado OpenMP não permitido  
  
 Código não é possível ir para ou de um bloco OpenMP.  
  
 O exemplo a seguir gera C3009:  
  
```  
// C3009.c  
// compile with: /openmp  
int main() {  
   #pragma omp parallel   
   {  
   lbl2:;  
   }  
   goto lbl2;   // C3009  
}  
```
