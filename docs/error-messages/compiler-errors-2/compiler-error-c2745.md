# Erro do Compilador C2745
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“token”: este token não pode ser convertido em um identificador  
  
 Os identificadores devem ser compreendidos de caracteres válidos.  
  
 O seguinte exemplo gera C2745:  
  
```  
// C2745.cpp  
// compile with: /clr  
int main() {  
   int __identifier([));   // C2745  
}  
```
