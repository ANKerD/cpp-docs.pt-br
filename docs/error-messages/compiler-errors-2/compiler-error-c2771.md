# C2771 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#import permitido somente em global ou escopo de namespace  
  
 A `#import` diretiva não é permitida em, por exemplo, uma função ou estrutura.
