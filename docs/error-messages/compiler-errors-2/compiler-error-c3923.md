# <a name="compiler-error-c3923"></a>C3923 de erro do compilador
'member': local classe, estrutura ou união não são permitidas em uma função de membro de uma classe gerenciada ou WinRT  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera C3923.  
  
```  
// C3923.cpp  
// compile with: /clr /c  
ref struct x {  
   void Test() {  
      struct a {};   // C3923  
      class b {};   // C3923  
      union c {};   // C3923  
   }  
};  
```
