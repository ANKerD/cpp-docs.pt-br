# Erro do Compilador C3923
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'member': local classe, estrutura ou união não são permitidos em uma função de membro de uma classe gerenciada ou o WinRT  
  
## Exemplo  
 O exemplo a seguir gera C3923.  
  
```  
// C3923.cpp  
// compile with: /clr /c  
ref struct x {  
   void Test() {  
      struct a {};   // C3923  
      class b {};   // C3923  
      union c {};   // C3923  
   }  
};  
```
