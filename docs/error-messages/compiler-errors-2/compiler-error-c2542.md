# Erro do Compilador C2542
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identificador”: o objeto da classe não tem nenhum construtor para inicialização  
  
 Não há nenhum construtor com uma lista de parâmetros que corresponde a inicialização.  
  
 Esse erro pode ser causado por parâmetros inválidos na inicialização.
