# C2969 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erro de sintaxe: 'symbol': esperado da definição de função de membro para terminar com '}'  
  
 Uma definição de função de membro de modelo tem uma chave de fechamento não correspondente.  
  
 O exemplo a seguir gera C2969:  
  
```  
// C2969.cpp // compile with: /c class A { int i; public: A(int i) {} }; A anA(1); class B { A a; B() : a(anA);   // C2969 // try the following line instead // B() : a(anA) {} };  
```
