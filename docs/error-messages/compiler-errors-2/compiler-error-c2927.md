# <a name="compiler-error-c2927"></a>C2927 de erro do compilador
'function': um modelo de função deve ser chamado com pelo menos um argumento  
  
 Você não pode chamar uma função de modelo sem argumentos. O tipo dos argumentos de modelo determina qual versão da função para gerar.
