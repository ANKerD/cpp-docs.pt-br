# <a name="compiler-error-c3417"></a>C3417 de erro do compilador
'member': tipos de valor não podem conter funções de membro especial definido pelo usuário  
  
 Tipos de valor não podem conter funções como um construtor de instância padrão, o destruidor ou o construtor de cópia.  
  
 O exemplo a seguir gera C3517:  
  
```  
// C3417.cpp  
// compile with: /clr /c  
value class VC {  
   VC(){}   // C3417  
  
   // OK  
   static VC(){}  
   VC(int i){}  
};  
```
