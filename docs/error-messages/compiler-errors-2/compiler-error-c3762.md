# <a name="compiler-error-c3762"></a>C3762 de erro do compilador
não é possível processar o atributo 'attribute'  
  
 Um atributo definido pelo usuário que herda de `System.Security.Permissions.SecurityAttribute` está sendo usado para definir um atributo de segurança. Um atributo não pode ser usado no mesmo assembly onde ele está definido.
