# Erro do Compilador C3762
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não foi possível processar o atributo “atributo”  
  
 Um atributo definido pelo usuário que herde de `System.Security.Permissions.SecurityAttribute` está sendo usado para definir um atributo de segurança.  Esse atributo não poderá ser usado no mesmo assembly no qual é definido.
