# <a name="compiler-error-c3818"></a>C3818 de erro do compilador
declaração de propriedade de matriz 'property1' não deve sobrecarregar uma propriedade de índice 'property2'  
  
 Uma sobrecarga não é possível para propriedades quando um é um indexador e a outra é uma propriedade de matriz. 
  
 C3818 só está acessível usando a opção de compilador obsoleto **/CLR: oldSyntax**.
