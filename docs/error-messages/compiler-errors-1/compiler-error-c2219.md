# Erro do Compilador C2219
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erro de sintaxe: o qualificador de tipo deve ser depois de “\*”  
  
 O qualificador de tipo \(`const` ou `volatile`\) é exibido o local onde não é permitido.
