# <a name="fatal-error-c1098"></a>Erro fatal C1098
Incompatibilidade de versão com mecanismo Editar e Continuar  
  
 A versão do depurador que você está usando não coincide com o compilador usado para criar o executável. Se a recompilação não corrigir o problema, você precisará reinstalar o Visual C++ para certificar-se de que você tem as versões apropriadas do depurador e do compilador.
