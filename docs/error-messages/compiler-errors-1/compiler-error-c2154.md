# <a name="compiler-error-c2154"></a>C2154 de erro do compilador
'type': tipo de enumeração só é permitido como um argumento para a característica de tipo intrínseco de compilador '__underlying_type'  
  
 Você pode obter apenas o tipo subjacente de um tipo de enumeração.  
  
 Para obter mais informações, consulte [suporte a compilador para características de tipo](../../windows/compiler-support-for-type-traits-cpp-component-extensions.md).
