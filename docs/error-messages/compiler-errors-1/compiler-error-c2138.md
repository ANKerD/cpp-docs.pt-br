# Erro do Compilador C2138
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ilegal para definir uma enumeração sem alguns membros  
  
 Uma enumeração deve ter pelo menos um membro quando \/Za \(extensões da Microsoft para desabilitar\) é selecionado.
