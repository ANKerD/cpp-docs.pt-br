# <a name="compiler-error-c2451"></a>C2451 de erro do compilador
expressão condicional do tipo 'type' é inválido  
  
 Avalia a expressão condicional para um tipo inteiro.  
  
 O exemplo a seguir gera C2451:  
  
```  
// C2451.cpp  
class B {};  
  
int main () {  
   B b1;  
   int i = 0;  
   if (b1)   // C2451  
   // try the following line instead  
   // if (i)  
      ;  
}  
```
