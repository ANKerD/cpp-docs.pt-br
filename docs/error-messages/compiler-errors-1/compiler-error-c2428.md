# <a name="compiler-error-c2428"></a>C2428 de erro do compilador
'operation': não é permitido no operando do tipo 'booleano'  
  
 Não é possível aplicar um operador de decremento para objetos do tipo `bool`.  
  
 O exemplo a seguir gera C2428:  
  
```  
// C2428.cpp  
void g(bool fFlag) {  
   --fFlag;   // C2428  
   fFlag--;   // C2428  
}  
```
