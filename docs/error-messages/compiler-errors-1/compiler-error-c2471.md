# <a name="compiler-error-c2471"></a>C2471 de erro do compilador
não é possível atualizar o banco de dados do programa 'file'  
  
 O compilador não pode gravar o arquivo de banco de dados.  
  
### <a name="to-fix-the-problem"></a>Para corrigir o problema  
  
-   Certifique-se de que o disco tem espaço suficiente disponível.  
  
-   Certifique-se de que o arquivo não é somente leitura.  
  
-   Certifique-se de que não há nenhuma violação de compartilhamento.
