# Erro do Compilador C2471
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não é possível atualizar o 'arquivo' de banco de dados  
  
 O compilador não pode gravar no arquivo do banco de dados.  
  
### Para corrigir o problema  
  
-   Certifique\-se de que o disco tenha espaço disponível suficiente.  
  
-   Certifique\-se de que o arquivo é seja somente leitura.  
  
-   Certifique\-se de que não haja nenhuma violação de compartilhamento.
