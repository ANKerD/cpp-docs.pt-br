# Erro do Compilador C2104
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'&' no campo de bit ignorado  
  
 Você não pode colocar o endereço de um campo de bit.  
  
 O seguinte exemplo gera C2104:  
  
```  
// C2104.cpp  
struct X {  
   int sb : 1;  
};  
  
int main() {  
   X x;  
   &x.sb;   // C2104   
   x.sb;   // OK  
}  
```
