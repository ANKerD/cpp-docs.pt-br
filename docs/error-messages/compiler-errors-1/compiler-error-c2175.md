# <a name="compiler-error-c2175"></a>C2175 de erro do compilador
'local': localidade inválida  
  
 O local especificado não é válido. Consulte [idiomas e cadeias de caracteres de país/região](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) no *referência da biblioteca de tempo de execução* para localidades com suporte.
