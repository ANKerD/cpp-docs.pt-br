# <a name="compiler-error-c2175"></a>C2175 de erro do compilador
'local': localidade inválida  
  
 A localidade especificada não é válida. Consulte [cadeias de caracteres de país/região e idioma](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) no *referência da biblioteca de tempo de execução* para localidades com suporte.
