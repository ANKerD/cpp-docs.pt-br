# <a name="compiler-error-c2121"></a>C2121 de erro do compilador
'#': caractere inválido: possivelmente o resultado de uma expansão de macro  
  
 Um caractere # inválido ter sido inserido por uma macro incorretova que usa o operador de colagem de token (#) em vez do operador stringizing (#).
