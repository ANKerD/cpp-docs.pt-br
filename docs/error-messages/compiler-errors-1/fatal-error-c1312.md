# <a name="fatal-error-c1312"></a>Erro fatal C1312
Muitas ramificações condicionais na função. Simplifique ou refatorar o código-fonte.  
  
 O código é muito complexo para o compilador a processar sem falta de memória de pilha.  Simplificar o seu código.
