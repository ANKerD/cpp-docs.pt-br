# <a name="compiler-error-c2101"></a>C2101 de erro do compilador
' &' na constante  
  
 Operador address-of ( `&` ) deve ter um valor de l como operando.  
  
 O exemplo a seguir gera C2101:  
  
```  
// C2101.cpp  
int main() {  
   char test;  
   test = &'a';   // C2101  
   test = 'a';   // OK  
}  
```
