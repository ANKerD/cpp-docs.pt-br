# Erro do Compilador C2418
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não é possível excluir o arquivo de navegador: nome do arquivo  
  
 O compilador não puder excluir o arquivo do navegador.  
  
### Para corrigir verificando as seguintes causas possíveis  
  
1.  Arquivo é somente leitura.  
  
2.  Arquivo está sendo usado por outro processo.
