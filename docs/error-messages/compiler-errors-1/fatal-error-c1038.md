# <a name="fatal-error-c1038"></a>Erro fatal C1038
limite do compilador: função: controlar o estado de fluxo muito complexo; Simplifique a função  
  
 A função tem mais estados do fluxo de controle que o compilador pode manipular. Simplificar o fluxo de controle ou divida a função em funções menores.
