# <a name="fatal-error-c1038"></a>Erro fatal C1038
limite do compilador: função: controlar o estado de fluxo muito complexo; simplificar a função  
  
 A função tem mais estados do fluxo de controle que o compilador pode manipular. Simplifique o fluxo de controle ou dividir a função em funções menores.
