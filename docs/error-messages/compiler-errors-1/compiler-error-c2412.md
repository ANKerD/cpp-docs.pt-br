# <a name="compiler-error-c2412"></a>C2412 de erro do compilador
'Rótulo': diferenciam maiusculas de minúsculas rótulo redefinido  
  
 O rótulo é definido mais de uma vez na função atual. Altere a ortografia do rótulo e suas referências.
