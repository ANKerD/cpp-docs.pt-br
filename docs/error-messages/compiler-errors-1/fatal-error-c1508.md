# Erro fatal C1508
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

limite do compilador: 'function': mais de 65.535 bytes de argumento  
  
 Os parâmetros formais para a função excederem o limite de 65535 bytes.
