# Erro do Compilador C2205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identificador”: não é possível inicializar extern variáveis com escopo do bloco  
  
 Uma variável de `extern` não pode ser inicializado em uma função.
