# <a name="compiler-error-c2241"></a>C2241 de erro do compilador
'Identificador': acesso de membro é restrito  
  
 Código tenta acessar um membro particular ou protegido.  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>Para corrigir usando as seguintes soluções possíveis  
  
1.  Altere o nível de acesso do membro.  
  
2.  Declarar o membro de uma `friend` da função de acesso.
