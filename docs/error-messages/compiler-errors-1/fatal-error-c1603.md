# Erro fatal C1603
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

destino embutida da ramificação do assembly fora do intervalo pelo número de bytes “”  
  
 Computado a distância entre uma instrução de JCXZ ou de JECXZ e sua rótulo especificado de destino era maior que 128 bytes.  Atualizar seu código de forma que o rótulo é mais perto da instrução.
