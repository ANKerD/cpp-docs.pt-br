# <a name="fatal-error-c1603"></a>Erro fatal C1603
destino de ramificação de assembly embutido fora do intervalo de bytes 'número'  
  
 A distância computada entre uma instrução JCXZ ou JECXZ e seu nome de destino especificado era maior que 128 bytes. Atualize seu código para que o rótulo mais próximo à instrução.
