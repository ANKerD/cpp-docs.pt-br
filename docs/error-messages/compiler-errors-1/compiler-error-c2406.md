# C2406 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': nome indefinido 'contexto'  
  
 Um identificador indefinido é usado com o `SIZE`, `LENGTH`, ou seleção de membro \(.\) operador.
