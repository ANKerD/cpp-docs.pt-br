# <a name="compiler-error-c2077"></a>C2077 de erro do compilador
inicializador de campo não escalar 'Identificador'  
  
 Você tentou inicializar um campo de um não escalares (struct, union, matriz ou classe). Use um inteiro ou um valor de ponto flutuante.
