# <a name="fatal-error-c1070"></a>Erro fatal C1070
incompatível #if #endif par em File 'filename'  
  
 Um `#if`, `#ifdef`, ou `#ifndef` diretiva não tem correspondente `#endif`.  
  
 O exemplo a seguir gera C1070:  
  
```  
// C1070.cpp  
#define TEST  
  
#ifdef TEST  
  
#ifdef TEST  
#endif  
// C1070  
```  
  
 Possível solução:  
  
```  
// C1070b.cpp  
// compile with: /c  
#define TEST  
  
#ifdef TEST  
#endif  
  
#ifdef TEST  
#endif  
```
