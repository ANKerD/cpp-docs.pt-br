# Erro do Compilador C2022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“número”: muito grande para o caractere  
  
 O número octal que segue uma barra invertida \(\\\) em uma constante de caractere ou cadeia de caracteres é muito grande para representar um caractere.
