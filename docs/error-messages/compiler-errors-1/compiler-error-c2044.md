# C2044 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

continuar ilegal  
  
 Um [continuar](../../cpp/continue-statement-cpp.md) é válido somente dentro de um `do`, `for`, ou `while` instrução.
