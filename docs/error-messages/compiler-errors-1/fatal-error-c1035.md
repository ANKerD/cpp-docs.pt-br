# Erro fatal C1035
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

expressão muito complexa; Simplificar expressão  
  
 O compilador não pôde gerar o código de uma expressão complexa. Divida a expressão em expressões simples e recompilar.
