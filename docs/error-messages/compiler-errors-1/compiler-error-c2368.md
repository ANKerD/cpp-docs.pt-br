# C2368 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': redefinição; especificadores de alocação diferentes  
  
 A declaração e a definição do símbolo especificam diferentes `__declspec` atributos.
