# <a name="compiler-error-c2149"></a>C2149 de erro do compilador
'identifier': campo de bits nomeado não pode ter largura zero  
  
 Os campos de bits só podem ter largura zero se sem nome.  
  
 O exemplo a seguir gera C2149:  
  
```  
// C2149.cpp  
// compile with: /c  
struct C {  
   int i : 0;   // C2149  
   int j : 2;   // OK  
};  
```
