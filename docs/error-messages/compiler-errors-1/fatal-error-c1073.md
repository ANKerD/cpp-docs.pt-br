# Erro fatal C1073
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erro interno que envolve a compilação incremental \(arquivo “nome de arquivo” do compilador, número de linha\)  
  
 Recompilar o arquivo sem usar a compilação incremental.
