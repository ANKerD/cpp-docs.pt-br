# C2043 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

quebra ilegal  
  
 Um [quebra](../../cpp/break-statement-cpp.md) é válido somente dentro de um `do`, `for`, `while`, ou `switch` instrução.
