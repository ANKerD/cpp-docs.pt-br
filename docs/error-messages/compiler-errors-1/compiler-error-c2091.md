# Erro do Compilador C2091
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

função da função retorna  
  
 Uma função não pode retornar uma função.  Retorna um ponteiro para uma função em seu lugar.
