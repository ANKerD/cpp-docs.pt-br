# Compiler Error C2222
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tipo inesperado 'type': era esperado uma classe base ou membro  
  
 A lista de inicializadores só pode inicializar classes base ou membros de um tipo.  Para corrigir esse erro, verifique se que classes base apenas ou membros do tipo são inicializados na lista de inicializadores.
