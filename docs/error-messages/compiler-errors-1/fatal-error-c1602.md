# Erro fatal C1602
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

intrinsic sem suporte  
  
 Este erro indica uma incompatibilidade em seus arquivos do compilador .exe, que podem ter ocorrido devido a uma instalação incompleta.  Por exemplo, você pode instalar um service pack mas não o pacote de processador.  Instalar todos os produtos necessários.
