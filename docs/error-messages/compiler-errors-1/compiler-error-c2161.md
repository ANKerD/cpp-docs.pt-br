# <a name="compiler-error-c2161"></a>C2161 de erro do compilador
'##' não pode ocorrer no fim de uma definição de macro  
  
 Uma definição de macro terminou com um operador de colagem de token (#).  
  
 O exemplo a seguir gera C2161:  
  
```  
// C2161.cpp  
// compile with: /c  
#define mac(a,b) a   // OK  
#define mac(a,b) a##   // C2161  
```
