# Erro fatal C1075
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o token esquerdo foi não correspondente ao final do arquivo  
  
 O compilador esperadas *token* antes de que atingiu o final do arquivo.  
  
 Esse erro pode ser causado por um colchete sem correspondência, chave, ou outros emparelhado caracteres.
