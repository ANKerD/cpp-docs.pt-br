# C2403 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': registro deve ser base\/índice em 'context'  
  
 O registro usado em um operando indireta de memória deve ser um registro base ou o índice neste contexto.
