# Erro do Compilador C2185
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identificador”: alocação baseado ilegal  
  
 Uma variável de registro ou variável de automático \(local\) são declarados `__based`.  Somente variáveis globais podem ser declaradas `__based`.
