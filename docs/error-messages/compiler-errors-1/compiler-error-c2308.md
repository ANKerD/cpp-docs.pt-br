# <a name="compiler-error-c2308"></a>C2308 de erro do compilador
concatenando cadeias de caracteres incompatíveis  
  
 Ambas as cadeias de caracteres largos e não todo foram especificadas para concatenação. Você não pode concatenar uma cadeia de caracteres largos e uma cadeia de caracteres não.
