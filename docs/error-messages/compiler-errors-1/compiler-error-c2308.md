# Erro do Compilador C2308
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

concatenando cadeias de caracteres incompatíveis  
  
 As cadeias de caracteres amplas e não todo foram especificadas para concatenação.  Você não pode concatenar uma cadeia de caracteres ampla e a cadeia de caracteres não larga.
