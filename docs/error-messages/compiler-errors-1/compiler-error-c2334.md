# <a name="compiler-error-c2334"></a>C2334 de erro do compilador
tokens inesperado anterior ': ou {'; ignorando o corpo da função aparente  
  
 O exemplo a seguir gera C2334. Esse erro ocorre após erro C2059:  
  
```  
// C2334.cpp  
// compile with: /c  
// C2059 expected  
struct s1 {  
   s1   {}   // C2334  
   s1() {}   // OK  
};  
```
