# Erro do Compilador C2207
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'member': um membro de um modelo de classe não pode adquirir um tipo de função  
  
 O `member` da classe de modelo foi anteriormente analisado como um membro de dados não estáticos.  Ele não pode ser redefinido como uma função de membro.
