# Erro fatal C1350
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erro ao carregar a DLL “DLL”: DLL não encontrado  
  
 A DLL que oferece suporte à operação tentativa não foi encontrado.  
  
 Isso indica que um problema com seus instalação e reinstale o DLL do CD do produto.
