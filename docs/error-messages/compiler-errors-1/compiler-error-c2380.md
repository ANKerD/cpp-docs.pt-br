# <a name="compiler-error-c2380"></a>C2380 de erro do compilador
tipo (s) anterior 'identifier' (construtor com o tipo de retorno ou redefinição inválida do nome de classe atual)?  
  
 Um construtor retorna um valor ou redefine o nome da classe.  
  
 O exemplo a seguir gera C2326:  
  
```  
// C2380.cpp  
// compile with: /c  
class C {  
public:  
   int C();   // C2380, specifies an int return  
   int C;   // C2380, redefinition of i  
   C();   // OK  
};  
```
