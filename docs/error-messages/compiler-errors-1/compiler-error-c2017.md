# <a name="compiler-error-c2017"></a>C2017 de erro do compilador
sequência de escape inválida  
  
 Uma sequência de escape, como \t, aparece fora de um caractere ou uma cadeia de caracteres constante.  
  
 O exemplo a seguir gera C2017:  
  
```  
// C2017.cpp  
int main() {  
   char test1='a'\n;   // C2017  
   char test2='a\n';   // ok  
}  
```  
  
 C2017 pode ocorrer quando o operador stringize é usado com cadeias de caracteres que contêm sequências de escape.  
  
 O exemplo a seguir gera C2017:  
  
```  
// C2017b.cpp  
#define TestDfn(x) AfxMessageBox(#x)  
TestDfn(CString("\\") + CString(".h\"\n\n"));   // C2017  
```
