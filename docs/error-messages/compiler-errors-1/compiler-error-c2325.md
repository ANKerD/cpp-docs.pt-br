# <a name="compiler-error-c2325"></a>C2325 de erro do compilador
'type': tipo inesperado à direita do 'name'  
  
 Uma chamada é feita para um destruidor de tipo incorreto.  
  
 O exemplo a seguir gera C2325:  
  
```  
// C2325.cpp  
// compile with: /c  
class A {};  
typedef A* pA_t;  
void f() {  
    A** ppa = new A *;  
    ppa->~A*;   // C2325  
  
   pA_t *ppa2 = new pA_t;  
   ppa2->~pA_t();   // OK  
}  
```
