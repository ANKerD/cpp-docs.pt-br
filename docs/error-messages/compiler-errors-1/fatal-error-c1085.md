# <a name="fatal-error-c1085"></a>Erro fatal C1085
Não é possível gravar o arquivo do tipo de arquivo: 'arquivo': mensagem  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Para corrigir verificando as possíveis causas a seguir  
  
1.  Unidade é somente leitura.  
  
2.  Unidade está cheia.  
  
3.  Violação de compartilhamento.  
  
4.  Se a mensagem indicar "número de arquivo incorreto", o arquivo pode estar sendo fechado em primeiro plano durante a compilação em segundo plano.
