# <a name="compiler-error-c2446"></a>C2446 de erro do compilador
'operator': nenhuma conversão de 'type1' em 'type2'  
  
 O compilador não pode converter `type1` para `type2`. A conversão não pode fazer sentido porque ele viola a semântica de C/C++.
