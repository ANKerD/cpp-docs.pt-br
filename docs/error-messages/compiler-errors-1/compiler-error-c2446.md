# Erro do Compilador C2446
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

operador “”: nenhuma conversão do tipo “\- 1 " para o tipo “\- 2 "  
  
 O compilador não pode converter `type1` a `type2`.  A conversão não pode fazer sentido como viola a semântica C\/C\+\+.
