# <a name="compiler-error-c2170"></a>C2170 de erro do compilador
'identifier': não é declarada como uma função, não pode ser intrínseco  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Para corrigir verificando as possíveis causas a seguir  
  
1.  Pragma `intrinsic` é usado para um item que não seja uma função.  
  
2.  Pragma `intrinsic` é usado para uma função com nenhum formulário intrínseco.
