# C2069 de erro do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Converter do termo 'void' não 'void'  
  
 Tipo de `void` não pode ser convertido em qualquer outro tipo.
