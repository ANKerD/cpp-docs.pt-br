# <a name="compiler-error-c2223"></a>Erro do compilador C2223

esquerda de '-> identificador' deve apontar para struct/union

O operando à esquerda do `->` não é um ponteiro para uma classe, estrutura ou união.

Esse erro pode ser causado por um operando à esquerda que é uma variável indefinida (portanto, digite `int`).
