# Erro fatal C1506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erro irrecuperável do escopo do bloco  
  
 O bloco era muito grande de compilação.  
  
### Para corrigir verificando as seguintes causas possíveis  
  
1.  Chaves incompatíveis  
  
2.  Função ou a classe raramente grande
