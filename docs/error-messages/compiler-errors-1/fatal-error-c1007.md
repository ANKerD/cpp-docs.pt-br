# Erro fatal C1007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

cadeia de caracteres de sinalizador não reconhecido na opção  
  
 A opção de linha de comando contém uma cadeia de caracteres inválida. Verifique o **CL** variável de ambiente para erros e de linha de comando.
