# Erro do Compilador C2348
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“nome do tipo”: não é a c a agregação de estilo, não pode ser exportada inserido no \- IDL  
  
 Para colocar `struct` em um arquivo de .idl com o atributo de [exportação](../../windows/export.md) , `struct` deve conter apenas dados.  
  
 O seguinte exemplo gera C2348:  
  
```  
// C2348.cpp  
// C2348 error expected  
[ module(name="SimpleMidlTest") ];  
  
[export]  
struct Point {  
   // Delete the following two lines to resolve.  
   Point() : m_i(0), m_j(0) {}  
   Point(int i, int j) : m_i(i), m_j(j) {}  
  
   int m_i;  
   int m_j;  
};  
```
