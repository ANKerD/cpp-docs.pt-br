# <a name="fatal-error-c1206"></a>Erro fatal C1206

Dados Per-appdomain não há suporte para a versão do tempo de execução instalado

Somente há suporte para alguns recursos, por exemplo, de acordo com os dados de domínio do aplicativo, pelo common language runtime que suporta o recurso.

C1206 indica que a versão mais recente do tempo de execução não está instalada no seu computador. Instale a versão common language runtime que é destinado para uso com o compilador.

Ver [appdomain](../../cpp/appdomain.md) para obter mais informações.
