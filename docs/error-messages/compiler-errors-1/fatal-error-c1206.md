# <a name="fatal-error-c1206"></a>Erro fatal C1206
Dados Per-appdomain não há suporte para a versão do tempo de execução instalado  
  
 Somente há suporte para alguns recursos, como de acordo com os dados do domínio de aplicativo, o common language runtime que ofereça suporte ao recurso.  
  
 C1206 indica que a versão mais recente do tempo de execução não está instalada no seu computador. Instale a versão de tempo de execução de idioma comum que é destinada para uso com o compilador.  
  
 Consulte [appdomain](../../cpp/appdomain.md) para obter mais informações.
