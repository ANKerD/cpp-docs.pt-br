# <a name="fatal-error-c1008"></a>Erro fatal C1008
nenhum arquivo de entrada especificado  
  
 O compilador não foi fornecido um arquivo de código-fonte C ou C++ para compilar. Verifique o **CL** variável de ambiente para especificações de nome de arquivo e de linha de comando.
