# <a name="compiler-error-c2315"></a>C2315 de erro do compilador
'type1': referência é capturada por 'type2' no número de linha  
  
 O tipo de exceção é tratado por um manipulador anterior. A referência para o segundo problema tem o tipo do primeiro.
