# Erro fatal C1192
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#using falha no “arquivo”  
  
 Apenas um arquivo no formato de linguagem intermediária da Microsoft \(MSIL\) pode ser passado a uma política de [\#using](../../preprocessor/hash-using-directive-cpp.md) .  A opção do compilador de [\/clr](../../build/reference/clr-common-language-runtime-compilation.md) permite que você crie um arquivo de saída de MSIL.  Outros idiomas do Visual Studio também gerenciam arquivos de MSIL.
