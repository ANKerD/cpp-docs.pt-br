# <a name="compiler-error-c2410"></a>C2410 de erro do compilador
'identifier': nome de membro ambíguo 'contexto'  
  
 O identificador é um membro de mais de uma estrutura ou união neste contexto.  
  
 Use um especificador de estrutura ou união no operando que causou o erro. Um especificador de estrutura ou união é um identificador de tipo `struct` ou `union` (uma `typedef` nome ou uma variável do mesmo tipo que a estrutura ou união que está sendo referenciado). O especificador deve ser o operando esquerdo do operador de seleção de membro (.) primeiro usar o operando.
