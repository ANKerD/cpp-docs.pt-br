# Erro do Compilador C2095
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

função “”: o parâmetro real tem o tipo “nulo”: parâmetro “número”  
  
 O parâmetro passado para a função é o tipo `void`, que não é permitido.  Use um ponteiro como nulo em vez \( `void *`\).  
  
 `number` indica que o parâmetro é `void`.
