# <a name="compiler-error-c2124"></a>Erro do compilador C2124

divisão ou mod por zero

Uma expressão constante tem um denominador igual a zero. Para resolver o erro, não divida por zero.

O exemplo a seguir gera C2124:

```
// C2124.cpp
int main() {
  int i = 1 / 0;   // C2124  do not divide by zero
  int i2 = 12 / 2;   // OK
}
```
