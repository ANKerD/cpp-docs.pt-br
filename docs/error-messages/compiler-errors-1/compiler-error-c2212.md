# <a name="compiler-error-c2212"></a>C2212 de erro do compilador
'identifier': based não disponível para ponteiros para funções  
  
 Ponteiros para funções não podem ser declarados `__based`. Se precisar de dados baseada em código, use o `__declspec` palavra-chave ou o `data_seg` pragma.
