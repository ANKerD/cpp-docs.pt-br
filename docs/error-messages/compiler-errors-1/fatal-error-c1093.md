# Erro fatal C1093
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

A API chamada de função “name” falha no “local de chamada”: “texto de tempo de execução”  
  
 Uma chamada a uma função de .NET falhou.  ***text from run\-time*** pode ou não ser fornecido no tempo de execução da.  
  
 Para obter mais informações sobre mensagens de erro do sistema, consulte o arquivo do sistema winerror.h, e [FormatMessage](http://msdn.microsoft.com/library/windows/desktop/ms679351).
