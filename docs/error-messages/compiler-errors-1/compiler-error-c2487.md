# <a name="compiler-error-c2487"></a>C2487 de erro do compilador
'identifier': membro da classe de interface de dll não pode ser declarado com interface de dll  
  
 Você pode declarar uma classe inteira ou alguns membros de uma classe de interface não-DLL, com interface DLL. Você não pode declarar uma classe com interface DLL e, em seguida, declarar um membro da classe com interface DLL.
