# Erro do Compilador C2487
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identificador”: o membro da classe de interface de DLL não pode ser declarado com a interface da DLL  
  
 Você pode declarar uma classe inteira, ou certos membros de uma interface de classificação não DLL, com a interface da DLL.  Você não pode declarar uma classe com a interface da DLL e depois declarar um membro dessa classe com a interface da DLL.
