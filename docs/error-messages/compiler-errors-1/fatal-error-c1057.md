# Erro fatal C1057
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

fim de arquivo inesperado em expansão macro  
  
 O compilador um chegou ao final do arquivo de origem ao coletar argumentos de \- invocação, provavelmente devido a um parêntese direito faltando na invocação macro.
