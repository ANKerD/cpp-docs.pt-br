# <a name="compiler-error-c2324"></a>C2324 de erro do compilador
'identifier': inesperado à direita do 'name'  
  
 Um destruidor é chamado usando um identificador incorreto.  
  
 O exemplo a seguir gera C2324:  
  
```  
// C2324.cpp  
class A {};  
typedef A* pA_t;  
int i;  
  
int main() {  
   pA_t * ppa = new pA_t;  
   ppa->~i;   // C2324  
   ppa->~pA_t();   // OK  
}  
```
