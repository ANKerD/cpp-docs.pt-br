# Erro do Compilador C2224
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

a esquerda de “.identifier” deve ter o tipo da estrutura\/união  
  
 O operando à esquerda do ponto \(.\) não é uma classe, uma estrutura, ou uma união.  
  
 Esse erro pode ser causado por um operando esquerdo que é uma variável indefinido \(como consequência tipo `int`\).
