# Erro fatal C1099
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Editar e continuar compilação terminação do mecanismo  
  
 Editar e continuar carregado um arquivo de cabeçalho pré\-compilado em preparação para compilar as alterações de código, mas as ações subsequentes \(como alterações de código antes de cabeçalho pré\-compilado `#include` instrução ou parar o depurador\) impediu o editar e continuar Concluindo a compilação com esse processo. Você não precisa realizar nenhuma ação para corrigir esse erro.
