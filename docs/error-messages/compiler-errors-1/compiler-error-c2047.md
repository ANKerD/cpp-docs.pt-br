# <a name="compiler-error-c2047"></a>C2047 de erro do compilador
padrão inválido  
  
 A palavra-chave `default` pode aparecer somente em um `switch` instrução.  
  
 O exemplo a seguir gera C2047:  
  
```  
// C2047.cpp  
int main() {  
   int i = 0;  
   default:   // C2047  
   switch(i) {  
      case 0:  
      break;  
   }  
}  
```  
  
 Possível solução:  
  
```  
// C2047b.cpp  
int main() {  
   int i = 0;  
   switch(i) {  
      case 0:  
      break;  
      default:  
      break;  
   }  
}  
```
