# <a name="fatal-error-c1208"></a>Erro fatal C1208
Alocar classes de referência na pilha não é suportado pela versão de tempo de execução instalada  
  
 C1208 ocorre quando você tem um compilador para a versão atual, mas um common language runtime de uma versão anterior.  
  
 Algumas funcionalidades do compilador não podem funcionar em uma versão anterior do tempo de execução.  
  
 Instale a versão de tempo de execução de idioma comum que é destinada para uso com seu compilador.
