# <a name="fatal-error-c1208"></a>Erro fatal C1208
Não há suporte para alocar classes de referência na pilha pela versão do tempo de execução instalada  
  
 C1208 ocorre quando você tiver um compilador para a versão atual, mas um tempo de execução de linguagem comum de uma versão anterior.  
  
 Algumas funcionalidades do compilador não podem funcionar em uma versão anterior do tempo de execução.  
  
 Instale a versão de tempo de execução de idioma comum que é destinada para uso com o compilador.
