# Erro do Compilador C2411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identificador”: struct ilegal\/sindicalista em “contexto”  
  
### Para corrigir verificando as seguintes causas possíveis  
  
1.  O identificador não é um membro de uma estrutura ou de uma união visível neste contexto.  
  
2.  O identificador não é um membro da estrutura ou de união especificada com o operador membros de seleção \(.\).
