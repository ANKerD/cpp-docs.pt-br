# Erro do Compilador C2423
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“número”: escala ilegal  
  
 O código do assembly embutido usa um número diferente de 1, 2, 4, ou 8 para dimensionar um registro.  
  
 O seguinte exemplo gera C2423:  
  
```  
// C2423.cpp  
// processor: x86  
int main() {  
   _asm {  
      lea EAX, [EAX*3]   // C2423  
      lea EAX, [EAX+EAX*2]   // OK  
   }  
}  
```
