# <a name="compiler-error-c2083"></a>C2083 de erro do compilador
comparação de estrutura/união ilegal  
  
 Uma estrutura ou união é comparado diretamente com outro tipo definido pelo usuário. Isso não é permitido, a menos que um operador de comparação tenha sido definido ou existe uma conversão para um tipo escalar.
