# Erro fatal C1046
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

limite do compilador: estrutura aninhada muito abaixo  
  
 A estrutura, a união, ou a classe excederam o limite de aninhamento, que é 15 níveis.  Reescreva a definição para reduzir o nível de aninhamento.  Divida a estrutura, união, ou a classe em dois ou mais partes usando `typedef` para definir uma ou mais das estruturas aninhadas.
