# Erro do Compilador C2450
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

a expressão da opção de tipo “tipo” é ilegal  
  
 A expressão de `switch` avalia a um tipo inválido.  Deve ser avaliado como um tipo inteiro ou um tipo da classe com inequívoca a conversão do tipo inteiro.  Se for avaliado como um tipo definido pelo usuário, você deve fornecer um operador de conversão.  
  
 O seguinte exemplo gera C2450:  
  
```  
// C2450.cpp  
class X {  
public:  
   int i;  
} x;  
  
class Y {  
public:  
   int i;  
   operator int() { return i; }   // conversion operator  
} y;  
  
int main() {  
   int j = 1;  
   switch ( x ) {   // C2450, x is not type int  
   // try the following line instead  
   // switch ( y ) {  
      default:  ;  
   }  
}  
```
