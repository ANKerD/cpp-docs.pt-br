# Erro fatal C1077
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

limite do compilador: não pode ter mais do que o número de opções de linha de comando  
  
 O número de opções de linha de comando excede o limite interno.  
  
 Pode haver muitos símbolos definidos com [\/D](../../build/reference/d-preprocessor-definitions.md). \(Coloque as definições em um arquivo de cabeçalho em vez disso.\)
