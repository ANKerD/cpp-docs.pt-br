# <a name="fatal-error-c1100"></a>Erro fatal C1100
não é possível inicializar o OLE: mensagem de erro do sistema  
  
 O compilador não pode inicializar a biblioteca de modelo de objeto de componente (COM). Consulte [CoInitialize](http://msdn.microsoft.com/library/windows/desktop/ms678543).
