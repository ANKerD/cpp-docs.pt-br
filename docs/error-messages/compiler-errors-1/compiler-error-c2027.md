# Erro do Compilador C2027
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

uso de tipo indefinido “tipo”  
  
 Um tipo não pode ser usado até que seja definido.  Para resolver o erro, certifique\-se de que o tipo é totalmente definido antes de fazer referência.  
  
## Exemplo  
 O exemplo a seguir produz C2027.  
  
```  
// C2027.cpp  
class C;  
class D {  
public:  
   void func() {  
   }  
};  
  
int main() {  
   C *pC;  
   pC->func();   // C2027  
  
   D *pD;  
   pD->func();  
}  
```  
  
## Exemplo  
 É possível declarar um ponteiro para um tipo declarado mas indefinido.  Mas o Visual C\+\+ não permite uma referência a um tipo indefinido.  
  
 O exemplo a seguir produz C2027.  
  
```  
// C2027_b.cpp  
class A;  
A& CreateA();  
  
class B;  
B* CreateB();  
  
int main() {  
   CreateA();   // C2027  
   CreateB();   // OK  
}  
```
