# Erro fatal C1051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

programe o arquivo de base de dados “, pdbfile”, tem um formato obsoleto, excluí\-lo e recompilar\-lo  
  
 O compilador não pode atualizar o arquivo de base de dados do programa, que tem um número de versão anterior.  Exclua o arquivo e recompile o programa com **\/Zi** ou **\/ZI**.  Para obter mais informações, consulte [\/Z7, \/Zi, \/ZI \(depurar formato de informações\)](../Topic/-Z7,%20-Zi,%20-ZI%20\(Debug%20Information%20Format\).md)
