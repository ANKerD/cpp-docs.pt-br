# <a name="fatal-error-c1904"></a>Erro fatal C1904

interação de provedor incorreta: 'file'

Esse erro indica que a falha de um provedor de atributo.

Observe as circunstâncias do erro, tente isolar o problema e criar um caso de teste pode ser reproduzido, em seguida, entre em contato com [Fale conosco](/visualstudio/ide/talk-to-us).
