# Erro fatal C1904
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

a interação do provedor inválido: 'arquivo'  
  
 Esse erro indica a falha de um provedor de atributo.  
  
 Observe as circunstâncias do erro, tente isolar o problema e criar um caso de teste pode ser reproduzido, em seguida, entre em contato com [Fale conosco](../Topic/Talk%20to%20Us.md).
