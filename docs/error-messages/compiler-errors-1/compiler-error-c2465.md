# <a name="compiler-error-c2465"></a>C2465 de erro do compilador
não é possível definir um tipo anônimo dentro de parênteses  
  
 Uma estrutura anônima, união ou tipo enumerado é definido dentro de uma expressão entre parênteses. Isso é inválido em C++ porque a definição não faz sentida no escopo de função.
