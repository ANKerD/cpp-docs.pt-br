# <a name="compiler-error-c2459"></a>C2459 de erro do compilador
'identifier': está sendo definida; não é possível adicionar um membro anônimo  
  
 Uma classe, estrutura ou união é redefinida em seu próprio escopo por um membro da união anônima.  
  
 O exemplo a seguir gera C2459:  
  
```  
// C2459.cpp  
// compile with: /c  
class C {  
   union { int C; };   // C2459  
   union { int D; };  
};  
```
