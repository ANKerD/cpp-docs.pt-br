# Erro do Compilador C2405
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

referência antecipada curto ilegal com deslocamento  
  
 As referências antecipadas curtas devem se referir apenas um rótulo.  Um deslocamento adicional não pode ser usado.
