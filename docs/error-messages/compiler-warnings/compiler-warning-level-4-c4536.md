# <a name="compiler-warning-level-4-c4536"></a>Compilador C4536 de aviso (nível 4)
nome do tipo: nome do tipo excede o limite de metadados de caracteres 'limit'  
  
 Um nome de tipo será truncado nos metadados se fosse um tipo gerenciado. Consulte [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md) para obter mais informações.  
  
 Esse aviso é desativada por padrão. Consulte [compilador avisos que está desativado por padrão](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para obter mais informações.
