# Aviso do compilador (n&#237;vel 4) C4536
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“nome do tipo”: o nome do tipo exceder o limite de metadados de caracteres de “limite”  
  
 Um nome de tipo será truncado nos metadados se fosse um tipo gerenciado.  Consulte [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md) para obter mais informações.  
  
 Esse aviso é desativada por padrão.  Consulte [Avisos do compilador que estão por padrão](../Topic/Compiler%20Warnings%20That%20Are%20Off%20by%20Default.md) para obter mais informações.
