# <a name="compiler-warning-level-1-c4113"></a>Compilador C4113 de aviso (nível 1)
'identifier1' é diferente nas listas de parâmetro de 'identifier2'  
  
 Um ponteiro de função é atribuído a outro ponteiro de função, mas as listas de parâmetros formais das funções não estão de acordo. A atribuição é compilada sem modificação.
