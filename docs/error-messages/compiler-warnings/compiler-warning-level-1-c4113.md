# Aviso do compilador (n&#237;vel 1) C4113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o “identifier1” difere em listas de parâmetro “identifier2”  
  
 Um ponteiro de função é atribuído a outro ponteiro de função, mas as listas de parâmetro formal de funções não concordam.  A atribuição é compilada sem modificação.
