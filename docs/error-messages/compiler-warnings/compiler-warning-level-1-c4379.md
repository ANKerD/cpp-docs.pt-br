# <a name="compiler-warning-level-1-c4379"></a>Compilador C4379 de aviso (nível 1)
Não há suporte para a versão do common language runtime por esse compilador. Usando essa versão pode causar resultados inesperados.  
  
 Você tem uma versão anterior do common language runtime em seu computador, mas não a versão atual. Para resolver C4379, instale a versão do common language runtime fornecido com seu compilador.
