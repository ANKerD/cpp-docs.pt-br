# Aviso do compilador (n&#237;vel 1) C4379
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

A versão da versão do Common Language Runtime não é suportada pelo compilador.Usar essa versão pode causar resultados inesperados.  
  
 Você tiver uma versão anterior do Common Language Runtime no computador, mas não a versão atual.  Para resolver C4379, instale a versão do Common Language Runtime fornecida com seu compilador.
