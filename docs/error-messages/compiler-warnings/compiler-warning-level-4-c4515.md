# <a name="compiler-warning-level-4-c4515"></a>Compilador C4515 de aviso (nível 4)
'namespace': usa o namespace em si  
  
 Um namespace é usada repetidamente.  
  
 O exemplo a seguir gera C4515:  
  
```  
// C4515.cpp  
// compile with: /W4  
namespace A  
{  
   using namespace A; // C4515  
}  
  
int main()  
{  
}  
```
