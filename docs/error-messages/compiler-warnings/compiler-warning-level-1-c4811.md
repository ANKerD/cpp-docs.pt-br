# <a name="compiler-warning-level-1-c4811"></a>Compilador C4811 de aviso (nível 1)
valor de pragma conform (forScope, show) = = valor  
  
 Esse aviso é emitido quando você usa o **Mostrar** opção do [está de acordo com](../../preprocessor/conform.md) pragma. *valor* é o valor atual do método encaixar.
