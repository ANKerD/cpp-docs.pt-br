# <a name="compiler-warning-level-1-c4506"></a>Compilador C4506 de aviso (nível 1)
Nenhuma definição de função embutida 'function'  
  
 A função fornecida foi declarada e marcada para inlining mas não foi definida.  
  
 O compilador fez não embutir a função.  
  
 Certifique-se de que funções externas para ser embutida são declaradas com o `extern` palavra-chave.
