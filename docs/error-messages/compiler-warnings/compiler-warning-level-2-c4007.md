# <a name="compiler-warning-level-2-c4007"></a>Compilador aviso (nível 2) C4007

'function': deve ser 'atributo'

Um atributo necessário para uma função não é explicitamente declarado. Por exemplo, a função **principal** deve ter o `__cdecl` atributo. O compilador força o atributo.
