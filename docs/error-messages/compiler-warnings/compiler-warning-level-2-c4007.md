# Aviso do compilador (n&#237;vel 2) C4007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

função “”: deve ser o atributo” “  
  
 Um atributo obrigatório para uma função não for explicitamente declarado.  Por exemplo, a função **main** deve ter o atributo de `__cdecl` .  O compilador força o atributo.
