# Aviso do compilador (n&#237;vel 1) C4031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

lista de segundo parâmetro formal por mais tempo que a primeira lista  
  
 Uma função redeclarado com parâmetros formais diferentes.  O compilador usa os parâmetros formais dados na primeira instrução.
