# <a name="compiler-warning-level-1-c4555"></a>Compilador C4555 de aviso (nível 1)
expressão não tem efeito; expressão esperada com efeito colateral  
  
 Esse aviso lhe informa quando a expressão não tem nenhum efeito.  
  
 Esse aviso é desativada por padrão. Consulte [compilador avisos que está desativado por padrão](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para obter mais informações.  
  
 Por exemplo:  
  
```  
// C4555.cpp  
// compile with: /W1  
#pragma warning(default:4555)  
  
void func1()  
{  
   1;   // C4555  
}  
  
void func2()  
{  
   int x;  
   x;   // C4555  
}  
  
int main()  
{  
}  
```
