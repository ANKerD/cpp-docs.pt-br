# <a name="compiler-warning-level-1-c4045"></a>Compilador C4045 de aviso (nível 1)
'array': estouro de limites de matriz  
  
 A matriz tem um número excessivo de inicializadores. Inicializadores extras são ignorados.  
  
 Certifique-se de elementos de matriz e os inicializadores correspondem no tamanho e quantidade.
