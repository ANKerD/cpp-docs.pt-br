# Aviso do compilador (n&#237;vel 3) C4511
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“classes: o construtor de cópia não pôde ser gerado  
  
 O compilador não foi possível gerar uma cópia construtor padrão para uma classe; uma classe base pode ter um construtor cópia em particular.
