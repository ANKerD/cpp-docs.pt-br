# <a name="compiler-warning-level-3-c4511"></a>Compilador C4511 de aviso (nível 3)
'class': construtor de cópia não pôde ser gerado  
  
 O compilador não pode gerar um construtor de cópia padrão para uma classe; uma classe base pode ter um construtor de cópia particular.
