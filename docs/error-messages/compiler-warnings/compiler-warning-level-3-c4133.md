# Aviso do compilador (n&#237;vel 3) C4133
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“tipo”: incompatibilidade de tipo \- “\- 1 " para o tipo “\- 2 "  
  
 Esse aviso pode ser causado pela tentativa subtrair ponteiros de dois tipos diferentes.  
  
 Para evitar esse aviso, forneça uma conversão de tipo apropriada.
