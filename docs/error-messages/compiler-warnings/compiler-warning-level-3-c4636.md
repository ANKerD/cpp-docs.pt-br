# Compilador C4636 de aviso (n&#237;vel 3)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Comentário de documento XML aplicado para 'Criar': marca requer vazio ' atributo.  
  
 A marca, como `cref`, não possui um valor.  
  
## Exemplo  
 O exemplo a seguir gera C4636.  
  
```  
// C4636.cpp // compile with: /clr /doc /W3 /c /// <see cref=''/> // /// <see cref='System::Exception'/> ref struct A {   // C4636 void f(int); }; // OK /// <see cref='System::Exception'/> ref struct B { void f(int); };  
```
