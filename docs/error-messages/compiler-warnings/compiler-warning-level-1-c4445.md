# <a name="compiler-warning-level-1-c4445"></a>Compilador C4445 de aviso (nível 1)
'function': em um WinRT ou tipo gerenciado um método virtual não pode ser particular  
  
 Se uma função virtual for privada, ele não pode ser acessado por um tipo derivado. Para corrigir esse erro, altere a acessibilidade da função de membro virtual público ou protegido.
