# Aviso do compilador (n&#237;vel 2) C4302
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“conversão”: truncamento de tipo “\- 1 " para o tipo “\- 2 "  
  
 O compilador detectou uma conversão de um tipo maior a um tipo menor.  As informações podem ser perdidas.  
  
 Esse aviso é desativada por padrão.  Consulte [Avisos do compilador que estão por padrão](../Topic/Compiler%20Warnings%20That%20Are%20Off%20by%20Default.md) para obter mais informações.  
  
 O seguinte exemplo gera C4302:  
  
```  
// C4302.cpp  
// compile with: /W2  
#pragma warning(default : 4302)  
int main() {  
   int i;  
   char c = (char) &i;     // C4302  
   short s = (short) &i;   // C4302  
}  
```
