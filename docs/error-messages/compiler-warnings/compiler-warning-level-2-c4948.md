# <a name="compiler-warning-level-2-c4948"></a>Compilador aviso (nível 2) C4948

tipo de retorno de 'accessor' não coincide com o último tipo de parâmetro do setter correspondente

O compilador encontrou uma incompatibilidade entre a que tipo de dados está sendo obter e define para uma propriedade indexada.

C4948 só está acessível usando a opção de compilador obsoletos **/CLR: oldSyntax**.
