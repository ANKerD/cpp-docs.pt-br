# <a name="compiler-warning-level-1-c4143"></a>Compilador C4143 de aviso (nível 1)
pragma 'same_seg' não tem suportada; Use a alocação de based  
  
 O **#pragma same_seg** não é mais suportada. Use o [based](../../cpp/based-pointers-cpp.md) palavra-chave em vez disso.
