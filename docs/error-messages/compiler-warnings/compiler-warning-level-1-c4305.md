# <a name="compiler-warning-level-1-c4305"></a>Compilador C4305 de aviso (nível 1)
'identifier': truncamento de 'type1' em 'type2'  
  
 O identificador é convertido em um tipo menor, resultando na perda de informações.  
  
 Se você usou o Assistente de ponto de Conexão de ATL, consulte o artigo de Base de dados de Conhecimento:  
  
-   Q250847: Assistente de ponto de Conexão de ATL código gerado para o evento com o argumento VARIANT oferece C4305 aviso
