# Aviso do compilador (n&#237;vel 1) C4305
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identificador”: truncamento de tipo “\- 1 " para o tipo “\- 2 "  
  
 O identificador é convertido em um tipo secundário, o que resulta em perda de informações.  
  
 Se você usou o assistente do ponto de conexão de ATL, consulte o artigo da Base de Dados de Conhecimento:  
  
-   Q250847: O código gerado assistente do ponto de conexão de ATL do evento com argumento VARIANT do aviso C4305
