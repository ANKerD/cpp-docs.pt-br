# Aviso do compilador (n&#237;vel 1) C4015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identificador”: o tipo de campo de bit deve ser integral  
  
 O campo de bit não é declarado como um tipo inteiro.  O compilador assume o tipo de base do campo de bit para ser sem assinatura.  Os campos de bit deve ser declarado como tipos de número inteiro sem sinal.
