# <a name="compiler-warning-level-1-c4344"></a>Compilador C4344 de aviso (nível 1)
alteração de comportamento: uso de resultados do modelo explícito argumentos na chamada para 'function'  
  
 Uma chamada para uma função usando argumentos de modelo explícitos chama uma função diferente do que seria se não tivessem sido especificados argumentos explícitos
