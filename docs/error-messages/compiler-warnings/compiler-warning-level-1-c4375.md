# Aviso do compilador (n&#237;vel 1) C4375
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o método “method2” que não seja do utilitário do não substitui “method1”  
  
 Um tipo que implementa outro tipo definiu um método de substituição, mas a substituição não era utilitário.  Em virtude disso, o método não substitui o método do tipo de base.
