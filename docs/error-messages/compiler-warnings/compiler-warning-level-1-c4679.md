# Aviso do compilador (n&#237;vel 1) C4679
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

O compilador encontrou uma construção que não pode dar suporte, que não pode ser importado de metadados.  
  
 **'**   
 ***member* ' : could not import member**  
  
 Não tente usar a construção.
