# <a name="compiler-warning-level-1-c4679"></a>Compilador C4679 de aviso (nível 1)
O compilador encontrou uma construção que não tem suporte, que não pode ser importada de metadados.  
  
 **'**   
 ***membro* ': não foi possível importar o membro**  
  
 Não tente usar a construção.
