# <a name="compiler-warning-level-1-c4164"></a>Compilador C4164 de aviso (nível 1)
'identifier': função intrínseca não declarada  
  
 A função intrínseca especificada não é declarada; Talvez você precise #include do arquivo de cabeçalho apropriado.
