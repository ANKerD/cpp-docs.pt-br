# <a name="compiler-warning-level-1-c4541"></a>Compilador C4541 de aviso (nível 1)
'identifier' usado no tipo polimórfico 'type' /GR-; poderá ocorrer um comportamento imprevisível  
  
 Você tentou usar um recurso que requer informações do tipo de tempo de execução sem habilitar informações de tipo de tempo de execução. Recompilar com [/GR](../../build/reference/gr-enable-run-time-type-information.md).
