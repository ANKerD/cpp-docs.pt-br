# Compilador C4245 de aviso (n&#237;vel 4)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'conversion': conversão de 'type1' em 'type2', incompatibilidade assinada\/não  
  
 Você tentou converter entrou **const** que tem um valor negativo para um `unsigned`.  
  
 O exemplo a seguir gera C4245:  
  
```  
// C4245.cpp  
// compile with: /W4 /c  
const int i = -1;  
unsigned int j = i; // C4245  
  
const int k = 1;  
unsigned int l = k; // okay  
  
int m = -1;  
unsigned int n = m; // okay  
  
void Test(size_t i) {}  
  
int main() {  
   Test( -19 );   // C4245  
}  
```
