# Aviso do compilador (n&#237;vel 1) C4024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

função “”: tipos diferentes do parâmetro formal número real “e”  
  
 Os parâmetros formais e reais correspondentes têm tipos diferentes.  O compilador transmite o parâmetro real sem modificação.  A função pull converte o tipo de parâmetro para o tipo esperado.
