# <a name="compiler-warning-level-1-c4024"></a>Compilador C4024 de aviso (nível 1)
'function': diferentes tipos de parâmetro formal e real 'number'  
  
 Parâmetros formais e reais correspondentes têm tipos diferentes. O compilador passa o parâmetro real sem alteração. A função de recepção converte o tipo de parâmetro para o tipo esperado.
