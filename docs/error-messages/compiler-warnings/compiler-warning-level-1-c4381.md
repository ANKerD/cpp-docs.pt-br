# <a name="compiler-warning-level-1-c4381"></a>Compilador C4381 de aviso (nível 1)
'function1': método de interface não será implementado pelo método não público 'function2'  
  
 Uma classe deve implementar funcionam em uma interface. Uma classe pode satisfazer essa condição se uma de suas classes base implementa a função. No entanto, a função deve ser implementada como uma função pública.
