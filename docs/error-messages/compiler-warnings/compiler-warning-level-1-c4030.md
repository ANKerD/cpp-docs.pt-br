# <a name="compiler-warning-level-1-c4030"></a>Compilador C4030 de aviso (nível 1)
**lista de parâmetros formais primeiro maior que a segunda lista**  
  
 Uma função foi declarado novamente com diferentes parâmetros formais. O compilador usa parâmetros formais dado na primeira declaração.
