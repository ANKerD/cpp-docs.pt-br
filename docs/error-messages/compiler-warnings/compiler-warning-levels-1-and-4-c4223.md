# <a name="compiler-warning-levels-1-and-4-c4223"></a>Aviso do compilador (níveis 1 e 4) C4223
extensão não padrão usada: matriz não lvalue convertida em ponteiro  
  
 No padrão C, você não pode converter uma matriz não lvalue para um ponteiro. Com as extensões do Microsoft padrão ([/Ze](../../build/reference/za-ze-disable-language-extensions.md)), você pode.
