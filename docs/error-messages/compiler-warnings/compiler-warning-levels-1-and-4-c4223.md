# Aviso do compilador (n&#237;veis 1 e 4) C4223
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

extensão não padrão usada: matriz de non\-lvalue convertida para o ponteiro  
  
 Em C padrão 2.0, você não pode converter uma matriz de non\-lvalue a um ponteiro.  Com as extensões padrão da Microsoft \([\/Ze](../../build/reference/za-ze-disable-language-extensions.md)\), você pode.
