# <a name="compiler-warning-levels-1-and-4-c4223"></a>Aviso do compilador (níveis 1 e 4) C4223
extensão não padrão usada : matriz não lvalue convertida para ponteiro  
  
 Em C padrão, você não pode converter uma matriz de não-l-value para um ponteiro. Com as extensões padrão da Microsoft ([/Ze](../../build/reference/za-ze-disable-language-extensions.md)), você pode.
