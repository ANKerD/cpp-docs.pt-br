# Compilador C4051 de aviso (n&#237;vel 2)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

conversão de tipo; possível perda de dados  
  
 Uma expressão contém dois itens de dados com diferentes tipos de base. Converter um tipo faz com que o item de dados sejam truncados.  
  
 Esse aviso pode ser corrigido se você converter os itens de dados para o tipo apropriado.
