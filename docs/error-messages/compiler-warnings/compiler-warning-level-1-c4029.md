# <a name="compiler-warning-level-1-c4029"></a>Compilador C4029 de aviso (nível 1)
lista de parâmetros formais declarada diferente da definição  
  
 Tipos de parâmetro formal na declaração de função não concordo com aqueles na definição de função. O compilador usa a lista de parâmetros da definição.
