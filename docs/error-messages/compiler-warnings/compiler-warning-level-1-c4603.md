# <a name="compiler-warning-level-1-c4603"></a>Compilador aviso (nível 1) C4603

'\<identificador >': macro não está definida ou definição é diferente após uso de cabeçalho pré-compilado

A macro especificada pela *identificador* espaço reservado é diferente ou não definido depois que o cabeçalho de pré-compilador é usado.
