# <a name="compiler-warning-level-1-c4603"></a>Compilador C4603 de aviso (nível 1)
'\<identificador >': macro não está definida ou definição é diferente depois de usar o cabeçalho pré-compilado  
  
 A macro especificada pelo *identificador* espaço reservado é diferente ou não definida após o cabeçalho de pré-compilador é usado.
