# C4961 de aviso do compilador
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Nenhum dado de perfil foi mesclado 'arquivo. PGD,' otimizações guiadas por perfil desabilitadas  
  
 Nenhum dado de perfil \(arquivos .pgc\) estava disponíveis, portanto otimizações guiadas por perfil não podem ocorrer.
