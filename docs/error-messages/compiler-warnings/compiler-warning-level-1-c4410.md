# Aviso do compilador (n&#237;vel 1) C4410
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tamanho ilegal do operando  
  
 Um dos operandos na instrução tinha um tamanho incorreto.  O tamanho válido o menor para o operando foi usado.
