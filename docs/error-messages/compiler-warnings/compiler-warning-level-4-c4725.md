# Compilador C4725 de aviso (n&#237;vel 4)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

instrução pode não ser precisas em algumas Pentiums  
  
 O código contém uma instrução de assembly embutido que talvez não produza resultados precisos em alguns microprocessadores Pentium.  
  
 O exemplo a seguir gera C4725:  
  
```  
// C4725.cpp // compile with: /W4 // processor: x86 double m32fp = 2.0003e-17; void f() { __asm { FDIV m32fp   // C4725 } } int main() { }  
```
