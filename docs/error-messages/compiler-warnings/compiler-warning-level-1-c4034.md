# <a name="compiler-warning-level-1-c4034"></a>Compilador aviso (nível 1) C4034

sizeof retorna 0

O `sizeof` operador é aplicado a um operando de tamanho zero (uma estrutura vazia, união, classe ou tipo enumerado, ou tipo `void`).
