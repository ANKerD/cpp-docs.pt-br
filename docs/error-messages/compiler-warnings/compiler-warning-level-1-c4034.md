# <a name="compiler-warning-level-1-c4034"></a>Compilador C4034 de aviso (nível 1)
sizeof retorna 0  
  
 O `sizeof` operador é aplicado a um operando de tamanho zero (uma estrutura vazia, união, classe ou tipo enumerado, tipo ou `void`).
