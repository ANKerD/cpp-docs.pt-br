# <a name="compiler-warning-level-1-c4340"></a>Compilador C4340 de aviso (nível 1)
'value': valor convertido de positivo com um valor negativo  
  
 O `enum` valor é maior que o maior `enum` valor positivo que contorna a um valor negativo.
