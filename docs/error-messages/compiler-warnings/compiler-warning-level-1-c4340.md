# <a name="compiler-warning-level-1-c4340"></a>Compilador C4340 de aviso (nível 1)
'value': valor ajustado de positivo para valor negativo  
  
 O `enum` valor é maior que o maior `enum` valor positivo encapsulado ao redor de um valor negativo.
