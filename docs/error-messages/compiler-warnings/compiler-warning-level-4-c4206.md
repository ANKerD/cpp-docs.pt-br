# Aviso do compilador (n&#237;vel 4) C4206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**extensão não padrão usada: a unidade de conversão está vazia**  
  
 O arquivo estava vazia depois de pré\-processamento.  
  
 Esta extensão pode impedir que o código seja portátil para outros compiladores.  Gerencie um erro na compatibilidade ANSI \([\/Za](../../build/reference/za-ze-disable-language-extensions.md)\) e aplica apenas a 2.0 C o código\-fonte.
