# <a name="compiler-warning-level-4-c4206"></a>Compilador C4206 de aviso (nível 4)
**extensão não padrão usada: unidade de conversão está vazia**  
  
 O arquivo estava vazio após o pré-processamento.  
  
 Esta extensão pode impedir que seu código portátil para outros compiladores. Ele gera um erro em compatibilidade com ANSI ([/Za](../../build/reference/za-ze-disable-language-extensions.md)) e aplica-se somente ao código-fonte C.
