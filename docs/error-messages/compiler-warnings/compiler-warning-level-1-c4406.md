# <a name="compiler-warning-level-1-c4406"></a>Compilador C4406 de aviso (nível 1)
operando em diretiva ignorado  
  
 A diretiva não tem qualquer operandos, mas um operando foi especificado.  
  
 O compilador ignorado o operando determinado ou operandos.
