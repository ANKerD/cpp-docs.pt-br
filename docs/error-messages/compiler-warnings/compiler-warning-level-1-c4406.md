# Aviso do compilador (n&#237;vel 1) C4406
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

operando na política ignorada  
  
 A política não requer nenhum operandos, mas um operando foi especificado.  
  
 O compilador ignorado o operando ou os operandos dados.
