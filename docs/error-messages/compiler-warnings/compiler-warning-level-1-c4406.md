# <a name="compiler-warning-level-1-c4406"></a>Compilador aviso (nível 1) C4406

operando em diretiva ignorado

A diretiva não terá qualquer operandos, mas um operando foi especificado.

O compilador ignorava o determinado operando ou operandos.
