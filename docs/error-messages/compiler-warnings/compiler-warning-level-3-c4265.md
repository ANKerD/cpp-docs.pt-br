# <a name="compiler-warning-level-3-c4265"></a>Compilador C4265 de aviso (nível 3)
'class': classe tem funções virtuais, mas não o destruidor é virtual  
  
 Quando uma classe tem funções virtuais, mas um destruidor virtual, objetos do tipo podem não ser destruídos corretamente quando a classe é destruída através de um ponteiro de classe base.  
  
 Esse aviso é desativada por padrão. Consulte [compilador avisos que está desativado por padrão](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para obter mais informações.  
  
 O exemplo a seguir gera C4265:  
  
```  
// C4265.cpp  
// compile with: /W3 /c  
#pragma warning(default : 4265)  
class B  
{  
public:  
   virtual void vmf();  
  
   ~B();  
   // try the following line instead  
   // virtual ~B();  
};   // C4265  
  
int main()  
{  
   B b;  
}  
```
