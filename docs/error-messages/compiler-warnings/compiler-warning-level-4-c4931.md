# Aviso do compilador (n&#237;vel 4) C4931
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

estamos assumindo a biblioteca de tipos fomos criados para ponteiros de \- \- bit  
  
 Informações explícitas não foi fornecida com o atributo de **ptrsize** de política de [\#import](../Topic/%23import%20Directive%20\(C++\).md) ; o compilador concluiu que o tamanho do ponteiro da biblioteca de tipos é ***number***.  
  
 Esse aviso é desativada por padrão.  Consulte [Avisos do compilador que estão por padrão](../Topic/Compiler%20Warnings%20That%20Are%20Off%20by%20Default.md) para obter mais informações.
