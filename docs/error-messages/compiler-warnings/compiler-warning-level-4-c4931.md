# <a name="compiler-warning-level-4-c4931"></a>Compilador C4931 de aviso (nível 4)
estamos supondo que a biblioteca de tipos foi criada para ponteiros de bit por número  
  
 Informações explícitas não foi fornecidas com o **ptrsize** atributo o [#import](../../preprocessor/hash-import-directive-cpp.md) diretiva; o compilador concluiu que o tamanho do ponteiro da biblioteca de tipos é ***número***.  
  
 Esse aviso é desativada por padrão. Consulte [compilador avisos que está desativado por padrão](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para obter mais informações.
