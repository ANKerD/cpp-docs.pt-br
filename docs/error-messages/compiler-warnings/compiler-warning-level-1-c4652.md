# Aviso do compilador (n&#237;vel 1) C4652
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opção “default” do compilador inconsistente com cabeçalho pré\-compilado; a opção de linha de comando atual substituirá o que definida no cabeçalho pré\-compilado  
  
 A opção de linha de comando determinada diferiu do que determinada quando o cabeçalho pré\-compilado \(.pch\) foi criado.  A opção especificada na linha de comando atual foi usada.  
  
 Esse aviso pode ser evitado regenerando o cabeçalho pré\-compilado com a opção de linha de comando determinada.
