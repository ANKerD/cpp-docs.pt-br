# Aviso do compilador (n&#237;vel 1) C4116
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

definição de tipo sem nome entre parênteses  
  
 Uma estrutura, uma união, ou um tipo enumerado sem o nome são definidos em uma expressão parenthetical.  A definição de tipo é sem\-sentido.  
  
 Na chamada de função c, a definição tiver escopo global.  Na chamada de função c criando, a definição tem o mesmo escopo que a função que está sendo chamada.
