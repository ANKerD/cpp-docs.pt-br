# <a name="compiler-warning-level-1-c4116"></a>Compilador C4116 de aviso (nível 1)
definição de tipo sem-nome entre parênteses  
  
 Uma estrutura, união ou tipo enumerado sem nome é definido em uma expressão entre parênteses. A definição de tipo não faz sentida.  
  
 Em uma chamada de função C, a definição tem escopo global. Em uma chamada de função do C++, a definição tem o mesmo escopo de função que está sendo chamado.
