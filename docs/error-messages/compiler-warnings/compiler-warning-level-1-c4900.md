# <a name="compiler-warning-level-1-c4900"></a>Compilador aviso (nível 1) C4900

incompatibilidade de linguagem intermediária entre a versão "number1 'version1' e"number2 "da" versão 'version2'

A linguagem intermediária usada na *tool1* e *tool2* não correspondeu. Verifique se a versão mais atual de cada ferramenta foi instalada.
