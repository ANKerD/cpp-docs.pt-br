# Aviso do compilador (n&#237;vel 4) C4610

 A classe tem construtores não definidos pelo usuário ou padrões.  Nenhuma instanciação é executada.  O seguinte exemplo gera C4610:  
  
```  
// C4610.cpp  
// compile with: /W4  
struct A {  
   int &j;  
  
   A& A::operator=( const A& );  
};   // C4610  
  
/* use this structure definition to resolve the warning  
struct B {  
   int &k;  
  
   B(int i = 0) : k(i) {  
   }  
  
   B& B::operator=( const B& );  
} b;  
*/  
  
int main() {  
}  
```
