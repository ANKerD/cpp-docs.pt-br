# <a name="compiler-warning-level-4-c4513"></a>Compilador C4513 de aviso (nível 4)
'class': não foi possível gerar o destruidor  
  
 O compilador não pode gerar um destruidor de padrão de uma determinada classe; Nenhum destruidor foi criado. O destruidor é em uma classe base que não está acessível para a classe derivada. Se a classe base tem um destruidor particular, torne-o público ou protegido.
