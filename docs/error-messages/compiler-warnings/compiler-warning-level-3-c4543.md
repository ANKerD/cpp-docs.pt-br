# Aviso do compilador (n&#237;vel 3) C4543
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Texto SQL injetado suprimido pelo atributo “no\_injected\_text”  
  
 O atributo de [no\_injected\_text](../../windows/no-injected-text.md) foi exibido no, o que significa o compilador impedirá que os atributos injetem o código.  
  
 Esse aviso é com que os atributos não poderão injetar o código.
