# <a name="compiler-warning-level-4-c4672"></a>Compilador C4672 de aviso (nível 4)
'identifier1': ambíguo. Primeira vista como 'identifier2'  
  
 O objeto especificado a ser lançado uma **tente** bloco é ambíguo. O objeto não pode ser ambíguas se ela é gerada.
