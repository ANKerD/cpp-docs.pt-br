# <a name="compiler-warning-level-1-c4502"></a>Compilador C4502 de aviso (nível 1)
'especificação de vínculo' requer o uso da palavra-chave 'extern' e deve preceder todos os outros especificadores  
  
 Uma vinculação foi especificada sem o `extern` palavra-chave. Vinculação não é relevante para tipos não-externo.  
  
 O compilador assumido o `extern` palavra-chave.
