# <a name="compiler-warning-level-1-c4502"></a>Compilador aviso (nível 1) C4502

'especificação de vínculo' requer o uso da palavra-chave 'extern' e deve preceder todos os outros especificadores

Uma vinculação foi especificada sem o `extern` palavra-chave. Vinculação não é relevante para os tipos de não-externo.

O compilador presume-se a `extern` palavra-chave.
