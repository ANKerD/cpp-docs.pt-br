# <a name="compiler-warning-level-4-c4152"></a>Compilador C4152 de aviso (nível 4)
extensão não padrão, a conversão de ptr de dados/função na expressão  
  
 Um ponteiro de função é convertido para ou de um ponteiro de dados. Essa conversão é permitida em extensões da Microsoft (/Ze) mas não em ANSI C.
