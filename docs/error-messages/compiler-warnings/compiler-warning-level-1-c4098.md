# <a name="compiler-warning-level-1-c4098"></a>Compilador C4098 de aviso (nível 1)
'function': função void retornando um valor  
  
 Uma função declarada com o tipo de retorno [void](../../cpp/void-cpp.md) tem um `return` instrução que retorna um valor. O compilador pressupõe que a função retorna um valor do tipo `int`.
