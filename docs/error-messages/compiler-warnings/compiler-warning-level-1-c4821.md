# <a name="compiler-warning-level-1-c4821"></a>Compilador C4821 de aviso (nível 1)
Não é possível determinar tipo de codificação Unicode, salve o arquivo com assinatura (BOM)  
  
 O compilador não pôde determinar o tipo de codificação para um arquivo. Para resolver esse aviso, salve o arquivo com uma marca de ordem de byte. Consulte [gerenciar arquivos com codificação](http://msdn.microsoft.com/en-us/ab3a74da-5c8f-4680-8882-f1def4909e82) para obter mais informações.
