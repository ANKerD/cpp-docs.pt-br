# Aviso do compilador (n&#237;vel 1) C4089
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

função “”: tipos diferentes no parâmetro real “número”, “número do parâmetro formal”  
  
 Os parâmetros formais e reais correspondentes têm tipos diferentes.  O parâmetro real é passado sem modificação.  A função converte o parâmetro real o tipo especificado na definição de função.
