# <a name="compiler-warning-level-1-c4089"></a>Compilador C4089 de aviso (nível 1)
'function': tipos diferentes em parâmetro real 'número', 'number' do parâmetro formal  
  
 Os parâmetros formais e reais correspondentes têm tipos diferentes. O parâmetro real é passado sem alteração. A função converte o parâmetro real para o tipo especificado na definição de função.
