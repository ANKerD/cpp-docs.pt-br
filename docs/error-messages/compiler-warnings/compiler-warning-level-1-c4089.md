# <a name="compiler-warning-level-1-c4089"></a>Compilador C4089 de aviso (nível 1)
'function': diferentes tipos de parâmetro real 'number', 'number' do parâmetro formal  
  
 Os parâmetros formais e reais têm tipos diferentes. O parâmetro real será passado sem alteração. A função converte o parâmetro real para o tipo especificado na definição de função.
