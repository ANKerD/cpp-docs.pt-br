# <a name="compiler-warning-level-4-c4639"></a>Compilador C4639 de aviso (nível 4)
Erro do MSXML, documento XML comentários não serão processados. motivo  
  
 Esse aviso pode ocorrer por vários motivos.  
  
 Para resolver este aviso:  
  
-   Recompile.  
  
-   Reinstale o MSXML reinstalando o common language runtime.  
  
-   Edite ou remova o comentário de documentação que está causando o aviso e recompile.  
  
 Quando C4639 é emitido, todo processamento adicional de comentário XML está desabilitado e o arquivo .xdc não é gerado.
