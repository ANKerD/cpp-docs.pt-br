# Aviso do compilador (n&#237;vel 4) C4639
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

O erro MSXML, comentários do documento XML não será processado. motivo  
  
 Esse aviso pode ocorrer em qualquer número de motivos.  
  
 Para resolver esse aviso:  
  
-   Recompilação.  
  
-   Reinstale o MSXML reinstalando Common Language Runtime.  
  
-   Editar ou remover o comentário da documentação que está causando o aviso e recompilar\-lo.  
  
 Quando C4639 é emitido, todo o processamento adicional de comentário XML é desabilitado, e o arquivo de .xdc não é gerado.
