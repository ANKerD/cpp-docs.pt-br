# <a name="compiler-warning-level-1-c4179"></a>Compilador C4179 de aviso (nível 1)
' / / *': analisada como '/' e ' /\*': confusão com padrão ' / /' comentários  
  
 **//\***é um delimitador de comentário incorreta. Use **//** or **/\*** instead.
