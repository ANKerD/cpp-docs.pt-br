# <a name="compiler-warning-level-1-c4088"></a>Compilador C4088 de aviso (nível 1)
'function': incompatibilidade de ponteiro em parâmetro real 'número', 'number' do parâmetro formal  
  
 Os parâmetros formais e reais correspondentes têm diferentes níveis de indireção. O parâmetro real é passado sem alteração. A função chamada interpreta seu valor como um ponteiro.
