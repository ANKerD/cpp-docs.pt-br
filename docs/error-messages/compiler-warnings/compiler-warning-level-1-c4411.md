# Aviso do compilador (n&#237;vel 1) C4411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identificador”: resoluções do símbolo para o registro de deslocamento  
  
 O identificador é um símbolo local que resolve a um deslocamento registrem e como consequência possam ser usadas em um operando com outro símbolo.
