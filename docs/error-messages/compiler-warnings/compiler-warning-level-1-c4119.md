# <a name="compiler-warning-level-1-c4119"></a>Compilador C4119 de aviso (nível 1)
diferentes bases 'base1' e 'base2' especificado  
  
 Dois ponteiros com base são incompatíveis porque têm diferentes bases de dados. O compilador não pode converter entre eles.
