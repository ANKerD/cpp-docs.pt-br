# Aviso do compilador (n&#237;vel 1) C4409
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tamanho da instrução ilegal  
  
 A instrução não tinha um formulário com o tamanho especificado.  O tamanho válido o menor foi usado.
