# <a name="cvtres-warning-cvt4001"></a>Aviso CVTRES CVT4001
**tipo de máquina não especificado; assumido**   
 ***type***  
  
 CVTRES não encontrou uma especificação de máquina. Assume-se que o tipo de determinado computador. Se o padrão é CVTRES incorreto, execute novamente usando a opção /MACHINE.
