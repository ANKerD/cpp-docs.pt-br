# Erro fatal U1083 (NMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“destino” macro de destino expanda a nada  
  
 O destino fornecido é uma invocação de uma macro que não esteja definido ou tem um valor nulo.  NMAKE não pode processar um destino nulo.
