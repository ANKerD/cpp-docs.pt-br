# Erro fatal RC1009 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

limite do compilador: “macro muito aninhado de macros”  
  
 O arquivo exceder o limite do compilador de recursos para o aninhamento macro.  Revise o arquivo de origem especificado para diminuir a profundidade de aninhamento de seus macros.
