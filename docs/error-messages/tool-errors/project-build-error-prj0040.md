# <a name="project-build-error-prj0040"></a>Erro de Build PRJ0040 no Projeto
Erro interno na compilação. Não é possível continuar. . Recarregue o projeto e tente novamente.  
  
 O sistema do projeto não pôde processar o seu projeto.  
  
 Tente recarregar o projeto.
