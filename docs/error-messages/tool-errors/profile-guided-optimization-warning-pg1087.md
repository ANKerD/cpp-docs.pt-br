# <a name="profile-guided-optimization-warning-pg1087"></a>Aviso PG1087 (otimização guiada por perfil)
O arquivo PGC 'file' não é válido.  
  
 Um arquivo. PGC está corrompido, possivelmente devido a um término anormal a uma sessão de criação de perfil.  Execute novamente o cenário de criação de perfil e encerrado normalmente.
