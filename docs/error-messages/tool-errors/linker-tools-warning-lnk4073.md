# <a name="linker-tools-warning-lnk4073"></a>Aviso LNK4073 (Ferramentas de Vinculador)
não é possível criar o mapa para o arquivo. ilk; vinculação de maneira não incremental  
  
 Não havia um grande espaço contíguo na memória compartilhada para LINK para criar o arquivo de status incremental (. ilk). LINK realizado uma compilação não incremental.
