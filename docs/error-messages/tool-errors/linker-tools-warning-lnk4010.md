# <a name="linker-tools-warning-lnk4010"></a>Aviso LNK4010 (Ferramentas de Vinculador)
número de número de versão de subsistema inválido; versão do subsistema padrão assumido  
  
 Você pode especificar uma versão do subsistema da imagem ([/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)). Cada subsistema tem um requisito de versão mínima. Se a versão especificada é menor do que o mínimo, esse aviso ocorrerá e o vinculador apenas use o subsistema de padrão.
