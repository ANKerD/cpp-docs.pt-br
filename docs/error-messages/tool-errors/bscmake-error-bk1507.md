# Erro BK1507 (BSCMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não é possível abrir o arquivo temporário “nome de arquivo” \[: motivo\]  
  
 BSCMAKE não pode abrir um arquivo temporário.  
  
### Para corrigir verificando as seguintes causas possíveis  
  
1.  A variável de ambiente de **TMP** não é definido como uma unidade e um diretório válido.  
  
2.  Um disco cheio.
