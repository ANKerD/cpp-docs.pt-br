# Erro CXX0052 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

função de membro não atual  
  
 Uma função de membro foi especificada como um ponto de interrupção mas não pôde ser localizada.  Definir um ponto de interrupção de uma função que será embutida pode causar esse erro.  
  
 Recompilar o arquivo com inlining forçado expansão \(\/Ob0\) para definir um ponto de interrupção nesta função.  
  
 Uma expressão chamou uma função que não foi definida.  
  
 Esse erro é idêntico a CAN0052.
