# Erro RC2135 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

arquivo não encontrado: nome do arquivo  
  
 O arquivo especificado na linha de comando do compilador de recursos não foi encontrado.  Verifique se o arquivo esteve movido para outro diretório e se o nome de arquivo e caminho são digitados corretamente.  
  
 Os arquivos são pesquisados usando uma variável de ambiente de **INCLUDE** ou defina\-o do Visual C\+\+ **INCLUDE** .
