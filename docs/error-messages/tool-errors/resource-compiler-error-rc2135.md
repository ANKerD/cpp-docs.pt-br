# <a name="resource-compiler-error-rc2135"></a>Erro RC2135 (compilador de recurso)
arquivo não encontrado: filename  
  
 O arquivo especificado na linha de comando de compilador de recurso não foi encontrado. Verifique se o arquivo foi movido para outro diretório e se o nome de arquivo e caminho são digitadas corretamente.  
  
 Arquivos são pesquisados para usar o **incluir** variável de ambiente ou o Visual C++ **incluir** configuração.
