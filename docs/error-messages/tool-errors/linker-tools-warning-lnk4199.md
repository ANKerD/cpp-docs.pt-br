# Aviso LNK4199 (Ferramentas de Vinculador)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/DELAYLOAD:dllname ignorado; nenhuma importa localizadas de dllname  
  
 O vinculador ignora `dllname` porque não precisa algumas das funções que `dllname` exporta.
