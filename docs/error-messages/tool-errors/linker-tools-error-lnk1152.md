# Erro das Ferramentas de Vinculador LNK1152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não é possível resolver um ou mais undecorated símbolos  
  
 Esse erro é precedido por um aviso [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) símbolo para cada undecorated que não pôde ser resolvido e por pelo menos dois avisos [LNK4002](../../error-messages/tool-errors/linker-tools-warning-lnk4002.md) para os símbolos duplicados localizado para o símbolo undecorated.
