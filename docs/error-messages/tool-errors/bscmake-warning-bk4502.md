# Aviso BK4502 (BSCMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

arquivo truncado “nome de arquivo” de .SBR não o nome do arquivo  
  
 Um arquivo de comprimento zero de .sbr que não foi originalmente parte do arquivo de .bsc foi especificada durante uma atualização.  
  
### Para corrigir verificando as seguintes causas possíveis  
  
1.  Nome de arquivo incorreto especificado.  
  
2.  Arquivo excluída. \(Resultados de [BK1513](../../error-messages/tool-errors/bscmake-error-bk1513.md) do erro.\)  
  
3.  Arquivo corrompido, exigindo BSCMAKE fazer uma compilação completa.
