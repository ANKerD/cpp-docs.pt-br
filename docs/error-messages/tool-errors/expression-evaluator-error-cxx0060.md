# Erro CXX0060 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o registro não está disponível  
  
 Uma expressão especificou um registro que não pode ser usado.  Esse erro pode ser causado pela tentativa de acessar um registro que não existe na execução do computador.  
  
 Esse erro é idêntico a CAN0060.
