# <a name="expression-evaluator-error-cxx0060"></a>Erro CXX0060 (avaliador de expressão)
o registro não está disponível  
  
 Uma expressão especificada de um registro que não pode ser usado. Esse erro pode ser causado pela tentativa de acessar um registro que não existe na máquina executando.  
  
 Esse erro é idêntico ao CAN0060.
