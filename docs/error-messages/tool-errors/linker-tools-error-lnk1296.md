# <a name="linker-tools-error-lnk1296"></a>Erro das Ferramentas de Vinculador LNK1296
não é possível carregar o nome de arquivo  
  
 O DLL especificado não estava disponível.  
  
 Os locais possíveis para a DLL são o diretório atual, o diretório do sistema, o diretório do Windows e diretórios especificados na variável de ambiente PATH.
