# Erro das Ferramentas de Vinculador LNK1296
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não é possível carregar o nome do arquivo  
  
 A DLL determinado não estava disponível.  
  
 Os locais possíveis para a DLL no diretório atual, o diretório do sistema, o diretório do Windows, e os diretórios especificados na variável de ambiente PATH.
