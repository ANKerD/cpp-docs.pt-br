# <a name="resource-compiler-warning-rc4002"></a>Aviso RC4002 (compilador de recurso)
número excessivo de parâmetros real para 'Identificador' de macro  
  
 O número de parâmetros reais especificado com o identificador especificado era maior que o número de parâmetros formais na definição de macro do identificador.  
  
 Os parâmetros reais adicionais foram coletados mas ignorados durante a expansão da macro.
