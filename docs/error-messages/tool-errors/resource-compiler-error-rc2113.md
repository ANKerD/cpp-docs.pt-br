# Erro RC2113 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

END esperada na caixa de diálogo  
  
 A palavra\-chave de **FIM** deve ocorrer no final de uma instrução de **CAIXA DE DIÁLOGO** .  Verifique se não há citação deixada aberta da instrução anterior.
