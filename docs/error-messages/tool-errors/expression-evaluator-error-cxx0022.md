# <a name="expression-evaluator-error-cxx0022"></a>Erro CXX0022 (avaliador de expressão)
chamada de função antes de _main  
  
 O avaliador de expressão C não é possível avaliar uma função antes que o depurador entrou a função **Main**. O programa não foi inicializado corretamente até **Main** foi chamado.  
  
 Esse erro é idêntico ao CAN0022.
