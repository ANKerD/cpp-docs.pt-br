# <a name="linker-tools-warning-lnk4200"></a>Aviso LNK4200 (Ferramentas de Vinculador)
informações de número de linha corrompida no arquivo de objeto; ignorado  
  
 As informações de número de linha no arquivo de objeto estão corrompidas. Reconstrução.
