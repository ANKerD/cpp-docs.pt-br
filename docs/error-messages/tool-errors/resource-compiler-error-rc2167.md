# <a name="resource-compiler-error-rc2167"></a>Erro RC2167 (compilador de recurso)
campo VERSIONINFO não reconhecido. BEGIN ou vírgula esperada  
  
 Um campo não reconhecido foi encontrado no **fixo** faz parte de um **VERSIONINFO** declaração de estrutura. Um **VERSIONINFO** campo deve ser DWORDS separados por uma vírgula.
