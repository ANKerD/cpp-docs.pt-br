# Erro M6205 (Math)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

função “”: erro de \_TLOSS  
  
 Uma perda de significância total \(precisão\) ocorreu.  
  
 Esse erro pode ser causado fornecendo um número muito grande como o operando de sin, cos, ou bronzear\-se porque o operando deve ser reduzido a um número entre 0 e 2\*pi.
