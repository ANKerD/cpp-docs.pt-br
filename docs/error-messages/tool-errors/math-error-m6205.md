# <a name="math-error-m6205"></a>Erro M6205 (Math)
'function': erro tloss  
  
 Ocorreu uma perda total de significância (precisão).  
  
 Esse erro pode ser causado por fornecendo um número muito grande como o operando do sin, cos ou tan porque o operando deve ser reduzido a um número entre 0 e 2 * pi.
