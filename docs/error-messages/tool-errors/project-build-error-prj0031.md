# <a name="project-build-error-prj0031"></a>Erro de Build PRJ0031 no Projeto
A propriedade 'Outputs' para a compilação personalizada passo para o arquivo 'arquivo' contido 'macro' que é avaliado 'macro_expansion'.  
  
 Uma etapa de compilação personalizada em um arquivo teve saída incorreta provavelmente devido a um problema de avaliação de macro. Esse erro também pode significar que o caminho é mal formado, contendo caracteres ou combinações de caracteres são ilegais em um caminho de arquivo.  
  
 Para resolver esse erro, corrija a macro ou corrigir a especificação do caminho. O caminho avaliado é um caminho absoluto do diretório do projeto.
