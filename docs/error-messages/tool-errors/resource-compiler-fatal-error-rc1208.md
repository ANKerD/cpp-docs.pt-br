# Erro fatal RC1208 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o arquivo de entrada tem a extensão de .RES  
  
 A extensão de .RES é usada para saída do compilador do recurso.  A extensão de .RC deve ser usada para scripts do compilador do recurso.
