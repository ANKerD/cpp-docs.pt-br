# <a name="resource-compiler-fatal-error-rc1208"></a>Erro fatal RC1208 (compilador de recurso)
arquivo de entrada tem. Extensão de RES  
  
 A. Extensão de RES é usada para a saída do compilador de recurso. A. Extensão RC deve ser usado para scripts do compilador de recurso.
