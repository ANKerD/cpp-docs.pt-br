# Erro PRJ0026 (compila&#231;&#227;o de projeto)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

O arquivo “arquivo” de resposta contém o conteúdo Unicode que não puderam ser convertidos para a página de código ANSI do usuário.  
  
 ***Conteúdo UNICODE do arquivo***  
  
 O conteúdo localizados sistema Unicode do projeto em uma resposta arquivo que não pode ser convertida corretamente para a página de código ANSI atual do usuário.  
  
 A solução desse erro é atualizar o conteúdo do arquivo de resposta para usar ANSI ou para instalar a página de código no seu computador e defini\-la como a opção do sistema.
