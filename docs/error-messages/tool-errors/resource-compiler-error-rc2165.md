# Erro RC2165 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

cadeia de caracteres não encontrada na instrução de DLGINCLUDE  
  
 A instrução não especificou um válido inclui o arquivo.  
  
 A instrução de **DLGINCLUDE** deve usar a seguinte sintaxe:  
  
 **DLGINCLUDE "*filename*.h"**
