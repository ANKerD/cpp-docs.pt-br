# <a name="resource-compiler-fatal-error-rc1021"></a>Erro fatal RC1021 (compilador de recurso)

comando de pré-processador inválido 'string'

Os caracteres após o sinal de número (**#**) não formavam uma diretiva de pré-processador válida.

O sinal de número não pode ser usado como o primeiro caractere em um identificador.
