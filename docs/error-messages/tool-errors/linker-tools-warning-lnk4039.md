# Aviso LNK4039 (Ferramentas de Vinculador)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

a seção “name” especificado com a opção de \/SECTION não existe  
  
 [DUMPBIN](../../build/reference/dumpbin-reference.md) ou [EDITBIN](../Topic/EDITBIN%20Reference.md) não pôde localizar uma seção chamada `name` no arquivo de entrada.  
  
 Para ver as seções em um arquivo, DUMPBIN de execução com a opção de [\/HEADERS](../../build/reference/headers.md) .
