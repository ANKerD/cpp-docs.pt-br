# <a name="linker-tools-warning-lnk4039"></a>Aviso LNK4039 (Ferramentas de Vinculador)
a seção 'name' especificada com opção /SECTION não existe  
  
 [DUMPBIN](../../build/reference/dumpbin-reference.md) ou [EDITBIN](../../build/reference/editbin-reference.md) não foi possível encontrar uma seção chamada `name` no arquivo de entrada.  
  
 Para ver as seções em um arquivo, execute DUMPBIN com o [/HEADERS](../../build/reference/headers.md) opção.
