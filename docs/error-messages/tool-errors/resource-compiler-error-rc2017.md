# Erro RC2017 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sequência de escape ilegal  
  
 Uma sequência de escape apareceu o local onde se não foi esperado.  
  
 Uma sequência de escape — uma barra invertida \( **\\** \) seguido por um número ou uma letra — só pode ocorrer em uma constante de caractere ou cadeia de caracteres.
