# Erro CXX0034 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**tipo incompatível com o operador**  
  
 Os tipos de operando especificados não são válidos para a operação.  
  
 Por exemplo, um ponteiro não pode ser multiplicado por nenhum valor.  
  
 Você pode precisar a conversão de tipos os operandos em um tipo compatível com o operador.
