# Erro CXX0008 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**missing “no final da cadeia de caracteres**  
  
 As aspas duplas esperadas no final da cadeia literal estavam faltando.  Digite a expressão, incluindo o literal da cadeia de caracteres entre aspas duplas.  
  
 Esse erro é idêntico a CAN0008.
