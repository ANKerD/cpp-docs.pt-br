# <a name="resource-compiler-fatal-error-rw1030"></a>Erro fatal RW1030 (compilador de recurso)
Erro de saída  
  
 Esse erro pode ser causado se o compilador de recurso não pôde criar o arquivo de destino. Certifique-se de que há espaço em disco suficiente e se você tem permissão de gravação no volume.
