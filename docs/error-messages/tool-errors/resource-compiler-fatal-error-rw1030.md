# Erro fatal RW1030 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erro de saída  
  
 Esse erro pode ser causado se o compilador de recursos não foi possível criar o arquivo de destino.  Verifique se há bastante espaço em disco e se você tem permissão de gravação no volume.
