# Erro BK1505 (BSCMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não é possível ler do arquivo “nome de arquivo” \[: motivo\]  
  
 BSCMAKE não pode ler do arquivo.  
  
### Para corrigir verificando as seguintes causas possíveis  
  
1.  Dano de Arquivo.  
  
2.  O arquivo truncamento à execução do compilador sem espaço em disco ou que está sendo interrompida ao criar o arquivo de .sbr.
