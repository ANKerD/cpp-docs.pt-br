# <a name="expression-evaluator-error-cxx0007"></a>Erro CXX0007 (avaliador de expressão)
parêntese direito ausento  
  
 Parêntese foram encontrado na expressão. Digite novamente a expressão com parênteses equilibradas.  
  
 Esse erro é idêntico ao CAN0007.
