# <a name="project-build-error-prj0025"></a>Erro de Build PRJ0025 no Projeto
O arquivo de lote 'file' contém conteúdo Unicode que não pôde ser convertido na página de código ANSI do usuário.  
  
 ***Conteúdo UNICODE do arquivo.***  
  
 O sistema do projeto encontrado conteúdo Unicode em um personalizado regra de compilação ou criar um evento que não pode ser convertido corretamente na página de código ANSI atual do usuário.  
  
 A resolução para esse erro é atualizar o conteúdo da regra de compilação ou evento usar ANSI ou para instalar a página de código em seu computador e defina-o como o padrão do sistema de compilação.  
  
 Para obter mais informações sobre personalizado etapas de compilação e eventos de build, consulte [Noções básicas sobre etapas de compilação personalizadas e eventos de Build](../../ide/understanding-custom-build-steps-and-build-events.md).
