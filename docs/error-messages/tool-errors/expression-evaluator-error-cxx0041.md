# Erro CXX0041 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o elemento da classe deverá ser membro estático ou função de membro  
  
 Um membro não estático de uma classe ou estrutura ou \(union\) foi usado sem especificar que instanciação da classe para usar.  
  
 Somente os membros de dados estáticos ou funções de membro podem ser usados sem especificar uma instanciação.  
  
 Esse erro é idêntico a CAN0041.
