# Erro fatal RC1121 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Arquivo de leitura de erro de E\/S  
  
 O compilador de recursos não foi possível ler um arquivo.  Verifica se a unidade que contém o arquivo está disponível e se o arquivo é válido.
