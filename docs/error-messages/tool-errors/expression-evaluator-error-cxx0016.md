# <a name="expression-evaluator-error-cxx0016"></a>Erro CXX0016 (avaliador de expressão)
constante muito grande  
  
 O avaliador de expressão C não pode aceitar uma constante de número inteiro sem sinal maior que 4.294.967.295 (0FFFFFFFF hexadecimal) ou uma constante de ponto flutuante cuja magnitude seja maior do que aproximadamente 1.8E + 308.  
  
 Esse erro é idêntico ao CAN0016.
