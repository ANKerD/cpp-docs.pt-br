# Erro CXX0016 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

constante muito grande  
  
 O avaliador de expressão C não pode aceitar as constantes de número inteiro sem sinal maior que 4.294.967.295 \(0FFFFFFFF hexadecimal\), ou uma constante de ponto flutuante cujo valor é maior do que aproximadamente 1.8E\+308.  
  
 Esse erro é idêntico a CAN0016.
