# Erro PRJ0035 (compila&#231;&#227;o de projeto)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

O arquivo XML “arquivo” contém o conteúdo Unicode que não puderam ser convertidos para a página de código ANSI do usuário.  
  
 ***Conteúdo UNICODE do arquivo***  
  
 ***file***  é o arquivo XML criado como a linha de comando para a ferramenta de implantação de Web.  
  
 O sistema de projeto encontrou caracteres Unicode em qualquer propriedade na página de propriedades de implantação de Web que não possa ser convertida em ANSI.  
  
 A solução desse erro é atualizar o conteúdo da propriedade para usar ANSI ou para instalar a página de código no seu computador e defini\-la como a opção do sistema.
