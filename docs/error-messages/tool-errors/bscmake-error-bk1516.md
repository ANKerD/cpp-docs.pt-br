# Erro BK1516 (BSCMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

bscfile danificar; não pode atualizar incrementalmente  
  
 O arquivo de .bsc foi danificado, possivelmente devido a uma falha do sistema durante a compilação.  Exclua o arquivo de .bsc, recriar todos os arquivos de .sbr, então recriar o arquivo de .bsc.
