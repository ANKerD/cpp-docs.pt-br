# Erro das Ferramentas de Vinculador LNK1215
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

falha na operação de metadados \(HRESULT\): erro  
  
 O vinculador recebeu um erro de tempo de execução do .NET. ao tentar fazer uma atualização de metadados com o tempo de execução do. .NET.  
  
 `HRESULT` é o HRESULT do método de tempo de execução do. .NET.  `error` é o texto de .NET\-supplied.  
  
 Provavelmente terá um o vinculador incompatível e o tempo de execução do .NET;. reinstalar o Visual C\+\+.
