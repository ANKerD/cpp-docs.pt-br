# Erro CXX0029 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não ponteiro da estrutura  
  
 O operador membros de seleção \(**\-\>**\) foi aplicado a uma expressão que não é um ponteiro para uma estrutura.  
  
 Verifica se a expressão inteira seja posta entre parênteses corretamente, ou conversão de tipos a expressão de endereço no tipo apropriado do ponteiro da estrutura.  
  
 Esse erro é idêntico a CAN0029.
