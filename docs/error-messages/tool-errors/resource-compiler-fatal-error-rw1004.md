# Erro fatal RW1004 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fim de arquivo inesperado  
  
 Esse erro pode ser causado por caracteres de alimentação de linha e de retorno ausentes na última linha de um arquivo de texto.
