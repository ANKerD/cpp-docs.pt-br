# <a name="nmake-fatal-error-u1052"></a>Erro fatal U1052 (NMAKE)
arquivo 'arquivo' não encontrado  
  
 NMAKE não pôde localizar o arquivo especificado com um dos seguintes:  
  
-   **/F** opção  
  
-   **! INCLUIR** política de pré-processamento  
  
-   Sinal de arroba (**@**) especificador para um arquivo de resposta  
  
 Verifique se o arquivo existe e o nome do arquivo está correto.
