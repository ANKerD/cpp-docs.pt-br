# Erro fatal U1052 (NMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“” nome de arquivo não encontrado  
  
 NMAKE não pôde localizar o arquivo especificado com um dos seguintes:  
  
-   opção de**\/F**  
  
-   **\!INCLUDE** que eles a política  
  
-   Especificador de arroba \(**@**\) para um arquivo de resposta  
  
 Verifique se o arquivo existe e o nome de arquivo seja digitado corretamente.
