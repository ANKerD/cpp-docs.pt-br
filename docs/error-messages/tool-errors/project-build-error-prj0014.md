# Erro PRJ0014 (compila&#231;&#227;o de projeto)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

O objeto de trabalho usado para controlar os processos de reprodução falhou.  A compilação não pode continuar.  
  
 Ocorreu um erro no ambiente de desenvolvimento.  
  
 Para resolver este erro, feche e recarregar o projeto.  Se necessário, saída e reinicie o Visual Studio.
