# <a name="math-error-m6110"></a>Erro M6110 (Math)
estouro de pilha  
  
 Uma expressão de ponto flutuante causou um estouro de pilha de ponto flutuante.  
  
 Exceções de ponto flutuante de estouro de pilha são interceptadas até o limite de sete níveis Além de oito níveis coprocessador 8087/287/387 geralmente com suporte.  
  
 Programa é encerrado com código de saída 138.
