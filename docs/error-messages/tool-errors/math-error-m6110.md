# <a name="math-error-m6110"></a>Erro M6110 (Math)
estouro de pilha  
  
 Uma expressão de ponto flutuante causou um estouro de pilha de ponto flutuante.  
  
 Exceções de estouro de pilha de ponto flutuantes são interceptadas até o limite de sete níveis Além dos níveis de oito a coprocessador 287/8087/387 normalmente com suporte.  
  
 Programa será encerrado com código de saída 138.
