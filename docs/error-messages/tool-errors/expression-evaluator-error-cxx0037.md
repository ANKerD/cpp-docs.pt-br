# <a name="expression-evaluator-error-cxx0037"></a>Erro CXX0037 (avaliador de expressão)
memória insuficiente  
  
 O avaliador de expressão C ficou sem memória, avaliar a expressão.  
  
 Esse erro é idêntico ao CAN0037.
