# Erro CXX0037 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sem memória  
  
 O avaliador de expressão C ficou sem memória que avalia a expressão.  
  
 Esse erro é idêntico a CAN0037.
