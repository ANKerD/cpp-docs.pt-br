# Aviso LNK4229 (Ferramentas de Vinculador)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/directive diretivo inválido localizado; ignorado  
  
 Uma política passada pelo pragma de [comentário](../../preprocessor/comment-c-cpp.md) era inválido.  O vinculador ignora `/directive`.
