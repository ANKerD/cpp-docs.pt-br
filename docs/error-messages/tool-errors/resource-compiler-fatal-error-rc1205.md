# Erro fatal RC1205 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

página de código inválido  
  
 A página de código \(\/c\) especifique a opção for seguida por uma página de código inválido.  Consulte [Páginas de código](../../c-runtime-library/code-pages.md)*na referência da biblioteca de tempo de execução* para obter mais informações.
