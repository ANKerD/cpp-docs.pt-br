# Erro PRJ0004 (compila&#231;&#227;o de projeto)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Não foi possível gerar a linha de comando para a ferramenta de ferramenta “”.  
  
 Uma ou mais propriedades foram especificadas de modo que fazer a sintaxe de chamada a ***tool*** ilegal:  
  
-   Você pode ter especificado macros ruim formados ou desconhecidos.  
  
-   Seu computador pode ser baixo no espaço livre em disco.  
  
 Talvez você queira examinar as definições da ferramenta examinando [Página de propriedades de linha de comando](../../ide/command-line-property-pages.md).
