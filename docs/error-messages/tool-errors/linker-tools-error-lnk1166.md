# <a name="linker-tools-error-lnk1166"></a>Erro das Ferramentas de Vinculador LNK1166
não é possível ajustar o código no deslocamento = deslocamento, va = valor  
  
 LINK não pôde preencher o código conforme necessário.  
  
 Determinadas instruções não são permitidas para cruzar os limites de página em alguns processadores. LINK tenta adicionar painéis para corrigir essa situação. Nesse caso, o LINK não funcionaria resolver o problema.
