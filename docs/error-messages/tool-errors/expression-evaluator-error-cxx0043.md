# Erro CXX0043 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

essa função de membro externa usada ponteiro  
  
 O ponteiro de **this** só pode ser usado para funções de membro não estático.  
  
 Esse erro é idêntico a CAN0043.
