# Erro CXX0031 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

expressão não expansível  
  
 O avaliador de expressão encontrou um erro interno.  
  
 Você pode escrever uma expressão equivalente que possa ser avaliada corretamente.  
  
 Esse erro é idêntico a CAN0031.
