# Erro PRJ0044 (compila&#231;&#227;o de projeto)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

A propriedade “dependências adicionais para a regra personalizada “regra” de compilação alocada ao arquivo “arquivo” não é válido.A propriedade contém a “cadeia de caracteres” que avalia “value”.  
  
 A propriedade de **Dependências Adicionais** avaliada como uma cadeia de caracteres vazia, ou uma cadeia de caracteres que contém caracteres inválidos \(qualquer caractere que não pode estar em um arquivo ou um nome de diretório\).  As regras de compilação personalizados precisam a saída de ação de construção.  
  
 Para obter mais informações, consulte [Especificando ferramentas de compilação personalizadas](../../ide/specifying-custom-build-tools.md).
