# <a name="project-build-error-prj0044"></a>Erro de Build PRJ0044 no Projeto
A propriedade 'Additional Dependencies' para a regra de compilação personalizada 'rule' atribuído ao arquivo 'arquivo' é inválida. A propriedade continha 'string', que é avaliada como 'value'.  
  
 O **dependências adicionais** propriedade avaliada para uma cadeia de caracteres vazia ou uma cadeia de caracteres que contém caracteres inválidos (qualquer caractere que não pode ser um nome de arquivo ou diretório). Compilação personalizada necessidade de regras de saída da ação de compilação.  
  
 Para obter mais informações, consulte [especificando ferramentas de compilação personalizado](../../ide/specifying-custom-build-tools.md).
