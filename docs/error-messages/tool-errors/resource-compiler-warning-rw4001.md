# Aviso RW4001 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Opções de processamento de .EXE \(\/L \/M \/P \/T \/K \/E \/31 ou \/30\)  
  
 Você especificou EXE que processa opções, mas não havia nenhum arquivo executável para processar.  Não use essas opções em um arquivo de .res.
