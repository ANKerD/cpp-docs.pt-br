# <a name="resource-compiler-warning-rw4001"></a>Aviso RW4001 (compilador de recurso)
. Opções de processamento EXE (/L /M/p /T /K /E /31 ou /30)  
  
 Você especificou EXE opções de processamento, mas não havia nenhum arquivo executável para processar. Não use essas opções com um arquivo. res.
