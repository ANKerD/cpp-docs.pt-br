# <a name="expression-evaluator-error-cxx0044"></a>Erro CXX0044 (avaliador de expressão)
o uso do ponteiro _based(void) requer o operador :>  
  
 Um ponteiro com base em `void` não pode ser usado diretamente. Devem formar um ponteiro completo usando o **:>** operador.  
  
 Esse erro é idêntico ao CAN0044.
