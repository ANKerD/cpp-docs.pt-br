# Erro CXX0044 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o uso do ponteiro void \(\) \_based requer:\> operador  
  
 Um ponteiro baseado em `void` não pode ser usado diretamente.  Você deve formar um ponteiro completo que usa o operador de **:\>** .  
  
 Esse erro é idêntico a CAN0044.
