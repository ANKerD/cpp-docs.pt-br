# Erros e avisos de compila&#231;&#227;o do projeto (PRJxxxx)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Esta seção é uma referência a erros gerados pelas ferramentas de compilação.  Para obter ajuda em uma mensagem de erro específico, clique no mouse em um número de erro na janela saída e pressione F1, ou digitar o número de erro na caixa de **Procurar** no índice.
