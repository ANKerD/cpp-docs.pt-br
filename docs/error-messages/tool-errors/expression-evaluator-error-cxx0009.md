# Erro CXX0009 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

falta 'após a constante de caractere  
  
 As aspas simples esperadas no final da constante de caractere estavam faltando.  Digite a expressão, incluindo a constante de caractere entre aspas simples.  
  
 Esse erro é idêntico a CAN0009.
