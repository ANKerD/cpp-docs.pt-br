# <a name="expression-evaluator-error-cxx0009"></a>Erro CXX0009 (avaliador de expressão)
ausente ' após constante de caractere  
  
 A aspa simples esperada no final da constante de caractere estava ausente. Digite novamente a expressão, incluindo a constante de caractere entre aspas simples.  
  
 Esse erro é idêntico ao CAN0009.
