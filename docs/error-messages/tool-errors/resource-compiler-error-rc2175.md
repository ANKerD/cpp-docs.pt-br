# Erro RC2175 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o nome do arquivo de recurso não está no formato 3,00  
  
 O recurso especificado usou um formato anterior à versão 3,00.  O arquivo de recurso deve ser convertido ou recriado usando o formato da versão 3,00 ou posterior.
