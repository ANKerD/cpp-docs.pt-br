# <a name="resource-compiler-fatal-error-rc1018"></a>Erro fatal RC1018 (compilador de recurso)
inesperado '#elif'  
  
 O `#elif` diretiva não apareceu dentro de um `#if`, **#ifdef**, ou **#ifndef** construir.  
  
 Certifique-se de que há um `#if`, **#ifdef**, ou **#ifndef** instrução em vigor antes que essa instrução.
