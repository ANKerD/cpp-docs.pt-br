# <a name="math-error-m6102"></a>Erro M6102 (Math)
desnormalizado  
  
 Uma operação gerou um número muito pequeno de ponto flutuante, que é inválido devido a perda de significância. Desnormalizado exceções de ponto flutuante são mascaradas, fazendo com que eles sejam interceptadas e operada.  
  
 Programa será encerrado com código de saída 130.
