# <a name="nmake-fatal-error-u1036"></a>Erro fatal U1036 (NMAKE)
erro de sintaxe : muitos nomes para esquerda de '='  
  
 É permitida somente uma cadeia de caracteres à esquerda de uma definição de macro.
