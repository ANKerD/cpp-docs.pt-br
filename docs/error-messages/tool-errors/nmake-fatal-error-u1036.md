# <a name="nmake-fatal-error-u1036"></a>Erro fatal U1036 (NMAKE)
Erro de sintaxe: muitos nomes para a esquerda de '='  
  
 É permitida apenas uma cadeia de caracteres à esquerda de uma definição de macro.
