# Erro fatal RC1109 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erro ao criar o nome de recurso  
  
 Não foi possível criar o arquivo especificado de .res.  Verifique se não está sendo criado em uma unidade somente leitura.  Use a opção \/V descobrir se o arquivo está sendo criado.
