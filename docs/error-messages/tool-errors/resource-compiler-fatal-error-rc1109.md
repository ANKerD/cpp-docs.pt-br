# <a name="resource-compiler-fatal-error-rc1109"></a>Erro fatal RC1109 (compilador de recurso)
Erro ao criar o nome do recurso  
  
 Não foi possível criar o arquivo. res especificado. Verifique se que ele não está sendo criado em uma unidade somente leitura. Use a opção /V para saber se o arquivo está sendo criado.
