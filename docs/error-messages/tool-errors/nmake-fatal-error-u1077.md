# <a name="nmake-fatal-error-u1077"></a>Erro fatal U1077 (NMAKE)
'program': 'value' código de retorno  
  
 O comando determinado ou programa chamado pelo NMAKE falhou e retornou o código de saída específico.  
  
 Para suprimir esse erro e continuar a sessão NMAKE, use a opção /I, o **. Ignorar** diretiva ponto ou traço (**-**) modificador de comando. Para continuar a sessão NMAKE relacionados partes da árvore de dependência, use a opção /K.
