# Erro fatal U1077 (NMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“programa”: código de retorno “valor”  
  
 O comando ou o programa determinado chamado por NMAKE falha e retornou o código de saída fornecido.  
  
 Para eliminar esse erro e continuar a sessão de NMAKE, use a opção \/I, a política de ponto de **.IGNORE** , ou o modificador de comando sublinhado \(**\-**\).  Para retomar a sessão de NMAKE para as partes não relacionados de árvore de dependência, use a opção do \/K.
