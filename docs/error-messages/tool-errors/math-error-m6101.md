# <a name="math-error-m6101"></a>Erro M6101 (Math)
inválidos  
  
 Operação inválida.  
  
 Esse erro pode ser causado quando um operando é NaN (não um número) ou infinito.  
  
 Programa será encerrado com código de saída 129.
