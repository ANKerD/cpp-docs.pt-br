# Erro das Ferramentas de Vinculador LNK1248
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o tamanho da imagem \(“"\) tamanho excede o tamanho máximo permitido \(80000000\)  
  
 O vinculador determinou que o tamanho do arquivo de saída excederá o tamanho possível o maior para uma imagem de programa.  Talvez você queira fazer seu programa em vários DLL.  Para obter mais informações sobre o programa executável \(PE\), consulte o artigo “no ponto de PE: Uma descrição do formato de arquivo executável Win32 portátil” em [A Biblioteca MSDN](http://go.microsoft.com/fwlink/?linkid=556).
