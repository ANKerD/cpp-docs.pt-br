# <a name="linker-tools-error-lnk1248"></a>Erro das Ferramentas de Vinculador LNK1248
imagem tamanho ('') excede o tamanho máximo permitido (80000000)  
  
 O vinculador determinou que o tamanho do arquivo de saída excederá o maior tamanho possível para uma imagem do programa. Você talvez queira fazer com que seu programa em várias DLLs. Para obter mais informações sobre o programa executável (PE), consulte o artigo "Emparelhamento dentro a: um Tour do Win32 portátil executável formato do arquivo PE" o [biblioteca MSDN](http://go.microsoft.com/fwlink/?linkid=556).
