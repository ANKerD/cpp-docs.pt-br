# Erro fatal U1000 (NMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erro de sintaxe: "\)” ausente na macro invocação  
  
 Um parêntese esquerdo, **\(**, apareceu sem um parêntese direito compatível, **\)**, em uma invocação macro.  O formato correto está**\)***nome*de **$\(**; `$`*n* é permitido nomes de um caractere.
