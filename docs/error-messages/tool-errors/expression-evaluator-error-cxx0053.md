# <a name="expression-evaluator-error-cxx0053"></a>Erro CXX0053 (avaliador de expressão)
correspodência de símbolo que não é função ao associar pontos de interrupção  
  
 Um símbolo usado como um ponto de interrupção não era uma função. Especificando um membro de dados como um ponto de interrupção pode causar esse erro.  
  
 Esse erro é idêntico ao CAN0053.
