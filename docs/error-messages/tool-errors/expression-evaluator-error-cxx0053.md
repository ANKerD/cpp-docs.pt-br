# Erro CXX0053 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

correspondência de símbolo de nonfunction ao associar pontos de interrupção  
  
 Um símbolo usado como um ponto de interrupção não era uma função.  Especificar um membro de dados como um ponto de interrupção pode causar esse erro.  
  
 Esse erro é idêntico a CAN0053.
