# Erro CXX0066 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

membro estático não atual  
  
 Um membro estático de uma classe não pôde ser localizado ou não foi definido.  Esse erro pode resultar de um membro estático da classe que é declarada, mas não definida ou somente é definido e referenciado nos módulos que não contêm informações de depuração.  
  
 Esse erro é idêntico a CAN0066.
