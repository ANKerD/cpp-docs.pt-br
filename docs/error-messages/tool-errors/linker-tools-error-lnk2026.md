# Erro das Ferramentas de Vinculador LNK2026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

módulo não seguro para a imagem de SAFESEH  
  
 [\/SAFESEH](../Topic/-SAFESEH%20\(Image%20has%20Safe%20Exception%20Handlers\).md) foi especificado, mas um módulo não é compatível com o recurso seguro de manipulação de exceção.  Se desejar usar esse módulo com **\/SAFESEH**, você precisará recompilar o módulo usando o compilador do Visual C\+\+ .NET 2003 \(ou posterior\).
