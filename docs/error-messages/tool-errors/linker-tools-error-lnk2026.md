# <a name="linker-tools-error-lnk2026"></a>Erro das Ferramentas de Vinculador LNK2026
módulo inseguro para imagem /SafeSEH  
  
 [/SafeSEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) foi especificado, mas um módulo não era compatível com a recurso de manipulação de exceção segura. Se você quiser usar este módulo com **/SAFESEH**, será necessário recompilar o módulo usando o compilador do Visual C++ .NET 2003 (ou posterior).
