# Erro RC2111 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tipo inválido de controle  
  
 Cada instrução de CONTROL em uma instrução de **CAIXA DE DIÁLOGO** deve ser um dos seguintes: 3STATE, AUTO3, AUTOCHECK, AUTORRÁDIO, BEDIT, CAIXA FROM SELEÇÃO, CAIXA FROM COMBINAÇÃO, CONTROL, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, ÍCONE, IEDIT, LISTBOX, LTEXT, PUSHBOX, TECLA, RADIOBUTTON, RTEXT, BARRA DE ROLAGEM, USERBUTTON.  
  
 Verifique se essas instruções de permissão CONTROL são estar escrito corretamente.
