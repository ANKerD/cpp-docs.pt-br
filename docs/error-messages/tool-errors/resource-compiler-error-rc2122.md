# Erro RC2122 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

subtipo desconhecido de menu  
  
 O campo *de item definição* da instrução de **MENU** pode conter apenas **MENUITEM** e instruções de **POP\-UP** .
