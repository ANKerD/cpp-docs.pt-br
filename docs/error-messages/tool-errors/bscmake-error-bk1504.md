# Erro BK1504 (BSCMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não pode posicionar no arquivo “nome de arquivo” \[: motivo\]  
  
 BSCMAKE não pode ir para um local no arquivo.  
  
### Para corrigir verificando as seguintes causas possíveis  
  
1.  Um disco cheio.  
  
2.  Um erro de hardware.  
  
3.  O truncamento arquivo devido a espaço insuficiente em disco ou a interrupção do compilador ao criar o arquivo de .sbr.
