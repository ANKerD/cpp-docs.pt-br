# <a name="bscmake-error-bk1504"></a>Erro BK1504 (BSCMAKE)
não é possível posicionar no arquivo 'arquivo' [: motivo]  
  
 BSCMAKE não pode mover para um local no arquivo.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Para corrigir verificando as possíveis causas a seguir  
  
1.  Disco cheio.  
  
2.  Um erro de hardware.  
  
3.  Truncamento de arquivos devido a espaço insuficiente em disco ou interrupção do compilador ao criar o arquivo. SBR.
