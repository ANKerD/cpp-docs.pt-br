# Erro fatal RC1017 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

expressão de inteiro constante inválido  
  
 A expressão em uma política de `#if` não existia ou não avalia como uma constante.
