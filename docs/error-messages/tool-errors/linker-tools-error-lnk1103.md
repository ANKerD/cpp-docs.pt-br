# <a name="linker-tools-error-lnk1103"></a>Erro das Ferramentas de Vinculador LNK1103
depurando informação corrompida; recompile o módulo  
  
 Esse erro pode ser causado porque a compilação foi encerrada antes de um arquivo de objeto válido foi criado.  
  
 Recompile o arquivo de determinado objeto. Se a recompilação não corrigir o problema, observe as circunstâncias do erro, tente isolar o problema e criar um caso de teste pode ser reproduzido. Consulte o arquivo de Ajuda do suporte técnico ou na seção de suporte técnico em um dos seus manuais.
