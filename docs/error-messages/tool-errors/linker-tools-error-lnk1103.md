# Erro das Ferramentas de Vinculador LNK1103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

informações de depuração; corrompido módulo de recompilação  
  
 Esse erro pode ser causado pela compilação foi encerrada antes que um arquivo de objeto válido esteve criado.  
  
 Recompilar o arquivo de objeto determinado.  Se recompilar não corrigir o problema, observe as condições de erro, tente\-as isolar o problema, e criá\-las um caso de teste reprodutível.  Consulte no arquivo de ajuda de suporte técnico, ou a seção de suporte técnico em um dos manuais.
