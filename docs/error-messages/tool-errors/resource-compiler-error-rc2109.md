# <a name="resource-compiler-error-rc2109"></a>Erro RC2109 (compilador de recurso)
constante de diálogo numérico esperado  
  
 A **caixa de diálogo** instrução requer valores inteiros para o *x, y, largura*, e *altura* campos. Verifique se esses valores forem incluídos após o **caixa de diálogo** palavra-chave e que eles não sejam negativos.
