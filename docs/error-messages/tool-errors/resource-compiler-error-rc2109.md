# Erro RC2109 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

constante numérica esperada da caixa de diálogo  
  
 Uma instrução de **CAIXA DE DIÁLOGO** exige valores inteiros para *o x, y o, a largura*, e os campos *de hora* .  Verifique se esses valores são incluídos após a palavra\-chave de **CAIXA DE DIÁLOGO** e que não são negativos.
