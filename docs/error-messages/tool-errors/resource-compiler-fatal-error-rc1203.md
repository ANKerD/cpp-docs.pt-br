# <a name="resource-compiler-fatal-error-rc1203"></a>Erro fatal RC1203 (compilador de recurso)
ID de idioma padrão hexadecimal inválido especificado.  
  
 Especificar idioma padrão (/ l) opção foi seguida por uma ID de idioma hexadecimal inválido. Consulte [idiomas e cadeias de caracteres de país/região](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) no *referência da biblioteca de tempo de execução* para obter uma lista de IDs de idioma válido.
