# Erro fatal CVT1103 (CVTRES)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não é possível ler o nome do arquivo  
  
 Um erro irrecuperável durante a CVTRES tentou ler o arquivo determinado.  
  
 Esse erro pode ser causado por danos no arquivo.
