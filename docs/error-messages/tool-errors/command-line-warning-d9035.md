# Aviso D9035 (linha de comando)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

a opção “default” foi preterida e será removida em uma futura versão  
  
 Você especificou uma opção de compilador que foi removida em uma versão futura do compilador do Visual C\+\+.  Esse aviso é seguido [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) aviso se houver uma substituição sugerido para *option*.  
  
 A opção especificada ainda funciona, mas você deve atualizar sua configuração de criação agora.  No resultado, seu projeto provavelmente continuar ao compilar quando você atualiza o compilador.
