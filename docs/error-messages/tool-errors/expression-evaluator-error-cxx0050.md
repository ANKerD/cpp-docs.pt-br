# Erro CXX0050 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o lado esquerdo do:: devem ser a classe\/struct\/união  
  
 O símbolo no lado esquerdo do operador de escopo\- resolução \(`::`\) não era uma classe, uma estrutura, ou uma união.  
  
 Esse erro é idêntico a CAN0050.
