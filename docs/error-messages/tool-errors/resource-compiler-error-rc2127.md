# Erro RC2127 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

palavras da versão separada por vírgulas esperadas  
  
 Os números de versão em um recurso de versão devem ser do tipo **WORD**, e separados por vírgulas.
