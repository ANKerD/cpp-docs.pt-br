# Erro fatal RC1102 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erro interno: argumentos muitos a RCPP  
  
 Muitos argumentos forem passados para o pré\-processador do compilador do recurso.  Reduza o número de símbolos definidos com os símbolos de definição \(\/d\) padrão configurando os na origem.  Esse erro também podem ser causadas por muitos incluir caminhos de busca de arquivo usando a opção de caminho de pesquisa incluir \(\/i\).
