# Erro das Ferramentas de Vinculador LNK1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erro desconhecido; consulte a documentação de opções de suporte técnico  
  
 Observe as condições de erro, tente\-as isolar o problema e criar um caso de teste, um contato reprodutíveis `Microsoft Product Support Services`.  Para obter informações sobre como examinar e reporta esses erros, consulte [http:\/\/support.microsoft.com\/default.aspx?scid\=kb;en\-us;134650](http://support.microsoft.com/default.aspx?scid=kb;en-us;134650).  
  
 Você pode obter este erro quando você mescla arquivos de cabeçalho padrão \(por exemplo, dos.h\) e seus próprios arquivos.  `#include` os cabeçalhos padrão primeiro, seguido por seus próprios arquivos de cabeçalho.
