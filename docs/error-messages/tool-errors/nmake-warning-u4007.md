# <a name="nmake-warning-u4007"></a>Aviso U4007 (NMAKE)
nome do arquivo 'filename' muito longo. Truncando 8.3  
  
 O nome base do arquivo fornecido tem mais de oito caracteres ou a extensão tem mais de três caracteres. NMAKE truncado no nome para uma base de oito caracteres e uma extensão de três caracteres.  
  
 Se seu sistema de arquivos dá suporte a nomes de arquivo longos, coloque o nome entre aspas duplas (**"**).
