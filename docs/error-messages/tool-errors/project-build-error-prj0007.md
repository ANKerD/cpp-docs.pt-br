# Erro PRJ0007 (compila&#231;&#227;o de projeto)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Não foi possível criar o diretório “” diretório de saída.  
  
 Visual C\+\+ não criou um diretório de saída.  As razões possíveis incluem:  
  
-   O diretório no qual o diretório de saída devia ser criada é somente leitura.  
  
-   O computador tem espaço em disco.  
  
-   Você especificou um caminho de diretório válido.  
  
-   As permissões de usuário não são suficientes para criar o diretório.
