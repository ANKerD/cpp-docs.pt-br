# Erro das Ferramentas de Vinculador LNK1240
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não criam o conteúdo de IDL  
  
 O vinculador desovou MIDL para criar IDL inserido mas houve um problema.  Verifique os erros especificados por MIDL.
