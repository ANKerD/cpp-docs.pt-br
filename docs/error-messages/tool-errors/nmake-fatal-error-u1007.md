# Erro fatal U1007 (NMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

aspas duplas não reservadas no nome  
  
 O nome de destino especificado ou o nome do arquivo continham aspas duplas \(**"**\).  
  
 As aspas duplas podem colocar um nome de arquivo e não podem ser contidas nele.
