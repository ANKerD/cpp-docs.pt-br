# <a name="resource-compiler-error-rc2112"></a>Erro RC2112 (compilador de recurso)
INÍCIO esperado na caixa de diálogo  
  
 O **BEGIN** palavra-chave deve seguir imediatamente o **caixa de diálogo** palavra-chave.
