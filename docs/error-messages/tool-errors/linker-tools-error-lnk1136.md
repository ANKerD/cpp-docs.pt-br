# Erro das Ferramentas de Vinculador LNK1136
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

arquivo inválido ou corrompido  
  
 O arquivo de entrada tem um cabeçalho for danificado ou tamanho pequeno ou de forma anormal zero.  
  
 LNK1136 também pode ser causado se o arquivo foi especificado incorretamente como uma dependência adicional.
