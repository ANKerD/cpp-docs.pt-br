# Erro CXX0072 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erro: ausentes ou desconhecido de informações de tipo  
  
 O arquivo de .pch não obteve vinculado no, ou o código tem uma referência a um tipo que esteja em um módulo não compilado com \/Zi.  
  
 Esse erro é idêntico a CAN0067.
