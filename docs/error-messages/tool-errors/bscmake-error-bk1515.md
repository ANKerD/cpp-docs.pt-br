# Erro BK1515 (BSCMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

bscfile: versão incompatível: não pode atualizar incrementalmente  
  
 O arquivo de .bsc foi criado com outra versão de BSCMAKE.  Recrie o arquivo de .bsc.
