# <a name="bscmake-error-bk1515"></a>Erro BK1515 (BSCMAKE)
bscfile: versão incompatível: não é possível atualizar de forma incremental  
  
 O arquivo. bsc foi criado com outra versão do BSCMAKE. Recrie o arquivo. bsc.
