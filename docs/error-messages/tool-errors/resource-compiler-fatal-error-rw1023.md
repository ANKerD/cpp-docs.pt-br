# <a name="resource-compiler-fatal-error-rw1023"></a>Erro fatal RW1023 (compilador de recurso)
Erro de e/s, gravar o arquivo, unidade completa  
  
 Espaço livre deve ser igual a pelo menos duas vezes o tamanho do arquivo executável que você está criando.
