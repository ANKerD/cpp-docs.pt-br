# <a name="expression-evaluator-error-cxx0004"></a>Erro CXX0004 (avaliador de expressão)
Erro de sintaxe  
  
 A sintaxe da expressão está incorreta.  
  
 Digite novamente a expressão com a sintaxe correta.  
  
 Esse erro é idêntico ao CAN0004.
