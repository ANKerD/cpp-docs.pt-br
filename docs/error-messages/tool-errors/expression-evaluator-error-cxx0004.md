# Erro CXX0004 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erro de sintaxe  
  
 A sintaxe da expressão está incorreta.  
  
 Digite a expressão com a sintaxe correta.  
  
 Esse erro é idêntico a CAN0004.
