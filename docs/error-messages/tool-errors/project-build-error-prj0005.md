# Erro PRJ0005 (compila&#231;&#227;o de projeto)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Não é possível criar um arquivo temporário no diretório “diretório”.  
  
 A chamada para criar um arquivo temporário falha.  As razões da falha incluem:  
  
-   Ficou sem os nomes de arquivo temporário.  
  
-   O diretório temporário é somente leitura.  
  
-   Não há nenhum diretório temporário ou variável de ambiente de TMP.  
  
-   O computador está fora de espaço livre em disco.
