# <a name="nmake-fatal-error-u1097"></a>Erro fatal U1097 (NMAKE)
sintaxe de partes do nome de arquivo exige dependente  
  
 A dependência atual não tem um dependente explícita ou um dependente implícita. Sintaxe de partes de nome de arquivo, que usa a porcentagem (`%`) especificador, representa os componentes do primeiro dependente do destino atual.
