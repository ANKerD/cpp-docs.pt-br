# <a name="math-error-m6107"></a>Erro M6107 (Math)
unemulated  
  
 Foi feita uma tentativa de executar uma instrução de coprocessador que é inválido ou não é suportada pelo emulador.  
  
 Programa é encerrado com código de saída 135.
