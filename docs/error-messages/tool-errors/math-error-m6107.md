# Erro M6107 (Math)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

unemulated  
  
 Foi feita uma tentativa de executar uma instrução de co\-processador que não é válido ou não tem suporte pelo emulador.  
  
 O programa será encerrado com o código de saída 135.
