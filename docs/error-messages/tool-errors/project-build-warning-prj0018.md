# Aviso PRJ0018 (compila&#231;&#227;o de projeto)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

As seguintes variáveis de ambiente não foi localizada:  
  
 Uma variável de ambiente não é definido.  Esse erro listará as variáveis de ambiente que não foram definidos.
