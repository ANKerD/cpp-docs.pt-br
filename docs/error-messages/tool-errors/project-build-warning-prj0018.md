# <a name="project-build-warning-prj0018"></a>Aviso PRJ0018 no Build do Projeto
As variáveis de ambiente a seguir não foram encontradas:  
  
 Uma variável de ambiente não está definida. Esse erro lista as variáveis de ambiente que não foram definidas.  
  
 Consulte [Macros para comandos de compilação personalizada](../../ide/common-macros-for-build-commands-and-properties.md) para obter informações sobre as macros no processo de compilação.
