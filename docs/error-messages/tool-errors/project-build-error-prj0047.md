# <a name="project-build-error-prj0047"></a>Erro de Build PRJ0047 no Projeto
Não foi possível continuar o processo suspenso.  A compilação falhou.  
  
 Ocorreu um erro no ambiente de desenvolvimento.  
  
 Para resolver esse erro, fechar e recarregar o projeto. Se necessário, saia e reinicie o Visual Studio.
