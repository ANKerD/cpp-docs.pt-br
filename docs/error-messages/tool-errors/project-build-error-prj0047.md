# Erro PRJ0047 (compila&#231;&#227;o de projeto)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Não foi possível retomar o processo suspenso.  A compilação falhou.  
  
 Ocorreu um erro no ambiente de desenvolvimento.  
  
 Para resolver este erro, feche e recarregar o projeto.  Se necessário, saída e reinicie o Visual Studio.
