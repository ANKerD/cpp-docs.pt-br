# <a name="linker-tools-warning-lnk4076"></a>Aviso LNK4076 (Ferramentas de Vinculador)
arquivo de status incremental inválido 'filename'; vinculação de maneira não incremental  
  
 LINK não pode gravar o arquivo de status incremental (. ilk). O `filename` está corrompido ou não é um banco de dados de vinculação incremental. Remova o arquivo e vincular novamente.
