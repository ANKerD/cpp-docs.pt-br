# Aviso LNK4076 (Ferramentas de Vinculador)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

arquivo incremental inválido “nome de arquivo” de status; vincular nonincrementally  
  
 O LINK não pode gravar no arquivo incremental de status \(.ilk\).  Ou `filename` estiver corrompido ou não é um base de dados vinculando incremental.  Remova o arquivo e relink.
