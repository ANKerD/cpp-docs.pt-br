# <a name="linker-tools-error-lnk1314"></a>Erro das Ferramentas de Vinculador LNK1314
tabela de símbolos COFF corrompidos ou inválidos (estático indefinido ou rótulo de símbolo)  
  
 O conteúdo gerado pelo compilador para uma seção em um determinado objeto parece estar corrompido.  
  
 Esse erro pode representar uma falha do compilador.
