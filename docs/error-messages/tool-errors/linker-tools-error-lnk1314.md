# Erro das Ferramentas de Vinculador LNK1314
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tabela de símbolo corrompido ou inválido de COFF \(símbolo indefinido estáticos ou do rótulo\)  
  
 O conteúdo completo gerado para uma seção no objeto determinado parece estar corrompido.  
  
 Esse erro pode representar uma falha do compilador.
