# <a name="nmake-warning-u4001"></a>Aviso U4001 (NMAKE)
arquivo de comando somente pode ser invocado pela linha de comando  
  
 Um arquivo de comando, que é invocado com o sinal de arroba (**@**) especificador, não pode conter uma especificação para outro arquivo de comando. Tal aninhamento não é permitido. A especificação foi ignorada.
