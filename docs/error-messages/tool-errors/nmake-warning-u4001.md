# Aviso U4001 (NMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

o arquivo de comando pode ser chamado somente de linha de comando  
  
 Um arquivo de comando, que é invocado pelo especificador de arroba \(**@**\), não pode conter uma especificação para outro arquivo de comando.  Tal aninhamento não é permitido.  A especificação foi ignorada.
