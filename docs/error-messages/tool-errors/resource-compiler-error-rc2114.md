# Erro RC2114 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nome da classe esperado de controle  
  
 O campo de `class` de uma instrução de CONTROL na instrução de **CAIXA DE DIÁLOGO** deve ser um dos seguintes tipos: BOTÃO, CAIXA FROM COMBINAÇÃO, EDIÇÃO, LISTBOX, BARRA DE ROLAGEM, STATIC, ou definida pelo usuário.  Verifique se a classe for escrito corretamente.
