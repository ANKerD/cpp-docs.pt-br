# Erro das Ferramentas de Vinculador LNK1221
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

um subsistema não pode ser inferido e deve ser definido  
  
 O vinculador não tem informações suficientes para inferir que o subsistema que você deseja destino.  
  
 Para corrigir esse erro, use [\/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md).
