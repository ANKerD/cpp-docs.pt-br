# Erro BK1517 (BSCMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

arquivo de origem de sbrfile compilado com \/Yc e \/Yu  
  
 O arquivo de .sbr serefere\-.  Foi recompilado provavelmente com \/Yu depois compile com \/Yc.  Redefinir padrão do compilador para o arquivo de origem a \/Yc, então seleciona **Recompilar** para gerar novos arquivos de .sbr.  Não recompilar o arquivo de origem com a \/Yu.
