# Erro CXX0049 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

OMF obsoleto \- por relink do programa  
  
 O programa usou um OMF antigo \(formato de módulo de objeto\).  
  
 Relink o programa usando a versão atual do vinculador.  
  
 Esse erro é idêntico a CAN0049.
