# <a name="expression-evaluator-error-cxx0010"></a>Erro CXX0010 (avaliador de expressão)
colchete esquerdo ausente  
  
 A expressão contém colchetes desbalanceadas. Digite novamente a expressão entre colchetes equilibradas.  
  
 Esse erro é idêntico ao CAN0010.
