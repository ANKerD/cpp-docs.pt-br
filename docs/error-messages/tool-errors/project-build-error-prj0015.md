# <a name="project-build-error-prj0015"></a>Erro de Build PRJ0015 no Projeto
O dispositivo do nulo está ausente do seu sistema. Estamos não é possível iniciar uma compilação.  
  
 Esse erro pode ser causado por recursos insuficientes do sistema. Feche alguns aplicativos ou reinicialização.  
  
 Ele também pode ser causado se você não tiver privilégios suficientes para o dispositivo do nulo no computador.
