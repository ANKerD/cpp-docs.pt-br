# Erro PRJ0015 (compila&#231;&#227;o de projeto)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Não há dispositivo NULL em seu sistema.Não é possível iniciar uma compilação.  
  
 Esse erro pode ser causado por poucos recursos do sistema.  Fechar alguns aplicativos ou reinicializá\-los.  
  
 Também pode ser causado se você não tiver privilégios suficientes no dispositivo NULL no computador.
