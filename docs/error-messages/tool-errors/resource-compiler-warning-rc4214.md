# Aviso RC4214 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Página de código inválido: ignorado  
  
 O arquivo de .rc continha um argumento de página de código e a página de código especificada não é válida.  Consulte [IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674) para obter mais informações.
