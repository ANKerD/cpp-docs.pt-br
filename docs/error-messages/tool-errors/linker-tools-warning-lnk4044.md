# <a name="linker-tools-warning-lnk4044"></a>Aviso LNK4044 (Ferramentas de Vinculador)
opção não reconhecida 'option'; ignorado  
  
 A opção fornecida não é uma opção válida para esta ferramenta. A ferramenta ignorado a opção.  
  
 Para ver uma lista de opções válidas, execute a ferramenta sem argumentos nem opções.
