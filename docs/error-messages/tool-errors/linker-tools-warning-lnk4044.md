# Aviso LNK4044 (Ferramentas de Vinculador)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opção não reconhecido “default”; ignorado  
  
 A opção especificada não é uma opção válido para essa ferramenta.  A ferramenta ignorado a opção.  
  
 Para ver uma lista de opções válidas, execute a ferramenta sem argumentos as opções ou.
