# Erro fatal RW1016 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

RC que encerra após erros de pré\-processador  
  
 O compilador de recursos paralisado devido a outros erros.  Corrigir os outros erros e recompilar\-los.
