# <a name="profile-guided-optimization-warning-pg1039"></a>Aviso PG1039 (otimização guiada por perfil)
Falha esperando por tempo de espera para varredura.  
  
 O aplicativo não permitiu que [pgosweep](../../build/reference/pgosweep.md) para executar.  Se possível, execute pgosweep quando o aplicativo está em um estado ocioso.
