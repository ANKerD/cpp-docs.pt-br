# Aviso PG1039 (otimiza&#231;&#227;o guiada por perfil)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Hora quietas de espera que falharam de entrada.  
  
 O aplicativo não permite que [pgosweep](../../build/reference/pgosweep.md) ser executado.  Se possível, pgosweep de execução quando o aplicativo está em um estado ocioso.
