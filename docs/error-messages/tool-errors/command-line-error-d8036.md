# Erro D8036 (linha de comando)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“\/option” não é permitida com vários arquivos de origem  
  
 Essas opções do compilador não podem ser usadas com vários arquivos de origem:  
  
-   Listagem do arquivo de assembly de nome \(\/Fa\)  
  
-   Renomear o arquivo de objeto \(\/Fo\)  
  
-   Crie as informações do navegador de origem sem as variáveis locais \(\/Fr\)  
  
-   Crie as informações do navegador de origem que inclui variáveis locais \(\/FR\)
