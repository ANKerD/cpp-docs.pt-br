# <a name="linker-tools-warning-lnk4220"></a>Aviso LNK4220 (Ferramentas de Vinculador)
inválido 'opção de vinculador' valor 'valor'; padrão assumido  
  
 Foi especificado um valor fora do intervalo com o [/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md) opção. O valor padrão para **/TLBID** é 1.
