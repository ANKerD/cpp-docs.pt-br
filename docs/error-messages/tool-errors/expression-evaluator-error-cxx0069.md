# Erro CXX0069 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

a variável precisa do quadro de pilhas  
  
 O avaliador de expressão não pode avaliar a variável porque não ocorre em um quadro de pilhas.  Isso pode ser causado por variáveis declaradas como parte de uma função embutida.
