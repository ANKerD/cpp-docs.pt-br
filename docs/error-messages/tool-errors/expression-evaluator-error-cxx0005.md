# Erro CXX0005 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

operador sem suporte  
  
 Um operador sem suporte do C foi especificado em uma expressão.  Escreva uma expressão equivalente usando os operadores com suporte do C.  
  
 Esse erro é idêntico a CAN0005.
