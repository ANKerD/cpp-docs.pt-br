# Erro CXX0045 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não é uma função  
  
 Uma lista de argumentos foi fornecido para um símbolo no programa que não é o nome de uma função.  
  
## Exemplo  
  
```  
queue( alpha, beta )  
```  
  
 quando `queue` não é uma função.  
  
 Esse erro é idêntico a CAN0045.
