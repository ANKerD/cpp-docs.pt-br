# <a name="expression-evaluator-error-cxx0051"></a>Erro CXX0051 (avaliador de expressão)
mais de um símbolo sobrecarregado especificado em ponto de interrupção  
  
 O avaliador de expressão não pôde determinar que mais de um símbolo sobrecarregado para usar como um ponto de interrupção.  
  
 Esse erro é idêntico ao CAN0051.
