# <a name="resource-compiler-fatal-error-rc1047"></a>Erro fatal RC1047 (compilador de recurso)
"opções de muitos opção, 'string'"  
  
 A opção fornecida foi especificada várias vezes. A cadeia de caracteres fornecida é o argumento para a opção que causou o erro.
