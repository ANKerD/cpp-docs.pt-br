# Erro fatal RC1105 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opção inválido, padrão: muitas alternâncias do \/d  
  
 Muitos símbolos foram definidos usando os símbolos de definição \(\/d\) padrão.  Defina alguns símbolos na origem e recompilar\-los.
