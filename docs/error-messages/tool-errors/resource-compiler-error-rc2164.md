# Erro RC2164 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

valor inesperado em RCDATA  
  
 Os valores *de dados brutos* na instrução de **RCDATA** devem ser inteiros ou cadeias de caracteres, cada um separado por uma vírgula.  Verifique se você não left out uma vírgula nem left out aspas em torno de uma cadeia de caracteres.
