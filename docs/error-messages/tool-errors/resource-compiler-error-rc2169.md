# <a name="resource-compiler-error-rc2169"></a>Erro RC2169 (compilador de recurso)
arquivo de recurso não está no formato 2,03  
  
 O recurso especificado usado em um formato anterior à versão 2.03. O arquivo de recurso deve ser convertido ou recriados usando o formato de versão 3.00 ou posterior.
