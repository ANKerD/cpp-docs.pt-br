# <a name="expression-evaluator-error-cxx0014"></a>Erro CXX0014 (avaliador de expressão)
operando ausente  
  
 Um operador foi especificado sem um operando necessário. Verifique a sintaxe da expressão.  
  
 Esse erro é idêntico ao CAN0014.
