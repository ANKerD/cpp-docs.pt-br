# <a name="bscmake-error-bk1509"></a>Erro BK1509 (BSCMAKE)
sem espaço de heap  
  
 BSCMAKE ficou sem memória, incluindo a memória virtual.  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>Para corrigir usando as seguintes soluções possíveis  
  
1.  Libere espaço em disco.  
  
2.  Aumente o tamanho do arquivo de permuta.  
  
3.  Aumente o tamanho do arquivo de permuta do Windows.  
  
4.  Reduza a memória que BSCMAKE requer usando /Ei ou /Es para eliminar alguns arquivos ou /Em para eliminar os corpos de macro de entrada.
