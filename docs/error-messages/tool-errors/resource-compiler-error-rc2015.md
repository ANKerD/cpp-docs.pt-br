# Erro RC2015 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

muitos caracteres da constante  
  
 Uma constante de caractere contiver mais de dois caracteres.  
  
 As constantes de caractere são limitadas a um caractere \(constantes de caractere padrão\) ou a dois caracteres \(constantes de caracteres longas\).  
  
 Observe que uma sequência de escape \(por exemplo, \\ t para a guia\) é convertida em um caractere único.
