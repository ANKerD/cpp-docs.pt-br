# <a name="resource-compiler-error-rc2163"></a>Erro RC2163 (compilador de recurso)
tipo de acelerador necessários [ASCII ou VIRTKEY]  
  
 O `type` campo o **ACELERADORES** instrução deve conter o valor de ASCII ou VIRTKEY.
