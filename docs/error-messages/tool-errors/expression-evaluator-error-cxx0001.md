# <a name="expression-evaluator-error-cxx0001"></a>Erro CXX0001 (avaliador de expressão)
função de usuário em execução de exceção  
  
 O código que está sendo executado causou uma falha de proteção geral. Esse erro é idêntico ao CAN0001.
