# Erro CXX0001 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

exceção que executa a função do usuário  
  
 O código que está sendo executado causado por uma falha geral de proteção.  Esse erro é idêntico a CAN0001.
