# <a name="linker-tools-error-lnk1332"></a>Erro das Ferramentas de Vinculador LNK1332
detectado\<contagem > tipos de tempo de execução do Windows importado em um módulo e definidos em outro módulo  
  
 Quando ele produziu o destino atual, o vinculador detectado `count`> tipos de tempo de execução do Windows, cada um deles é importada em um módulo e também definida em outro módulo.  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Corrigi cada um dos erros LNK2039 na compilação de acordo com a sugestão na mensagem de erro.
