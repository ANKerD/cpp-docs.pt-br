# Erro CXX0057 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nenhum código no número da linha  
  
 Nenhum código for gerado para o número da linha especificada.  Não pode ser usado como um ponto de interrupção.  
  
 Esse erro é idêntico a CAN0057.
