# Erro fatal RC1101 (compilador de recurso)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nenhum nome de arquivo binário do recurso especificado  
  
 A opção de saída renomear \(\/fo\) não foi seguida por um nome de arquivo.  
  
 Use a seguinte sintaxe para obter a opção \/fo:  
  
 **RC \/fo**   
 ***outfilename infilename.* rc**
