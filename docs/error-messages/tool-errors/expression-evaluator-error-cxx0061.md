# Erro CXX0061 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

profundidade de aninhamento da função excedida  
  
 A expressão contém uma profundidade de aninhamento da função maior que o limite.  Modifique a expressão para reduzir a profundidade de aninhamento.  
  
 Esse erro é idêntico a CAN0061.
