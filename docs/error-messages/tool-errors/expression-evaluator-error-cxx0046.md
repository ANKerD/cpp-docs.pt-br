# <a name="expression-evaluator-error-cxx0046"></a>Erro CXX0046 (avaliador de expressão)
lista de argumentos necessária para função de membro  
  
 Uma expressão de chamada de função de membro, mas não especificou nenhum parâmetro real.  
  
 Esse erro é idêntico ao CAN0046.
