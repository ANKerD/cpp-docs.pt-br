# <a name="resource-compiler-fatal-error-rc1004"></a>Erro fatal RC1004 (compilador de recurso)
fim de arquivo inesperado encontrado  
  
 Esse erro pode ser causado pela ausência de avanço de linha e caracteres de retorno de carro na última linha de um arquivo de texto.
