# <a name="nmake-fatal-error-u1071"></a>Erro fatal U1071 (NMAKE)
Na árvore de dependência de destino 'targetname' do ciclo  
  
 Existe uma dependência circular na árvore de dependência para o destino especificado. O destino específico depende dos dependentes de determinado destino. Dependências circulares são inválidas.
