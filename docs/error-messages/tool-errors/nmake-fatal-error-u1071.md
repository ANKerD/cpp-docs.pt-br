# Erro fatal U1071 (NMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

dê um ciclo na árvore de dependência para o destino “targetname”  
  
 Uma dependência circular existe na árvore de dependência para o destino especificado.  O destino é um determinado dependente de um dos dependentes de destino especificado.  As dependências circulares são inválidos.
