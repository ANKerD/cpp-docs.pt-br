# <a name="nmake-fatal-error-u1076"></a>Erro fatal U1076 (NMAKE)
nome muito extenso  
  
 Uma cadeia de caracteres excedeu um dos seguintes limites:  
  
-   1024 caracteres para um nome de macro.  
  
-   256 caracteres para um nome de caminho de destino.  
  
-   2048 caracteres para um comando.
