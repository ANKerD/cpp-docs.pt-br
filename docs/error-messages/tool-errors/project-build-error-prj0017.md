# Erro PRJ0017 (compila&#231;&#227;o de projeto)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

O diretório de trabalho atual é inválido.  
  
 O caminho para o diretório de trabalho atual for mais longo que o \_MAXPATH.  Para resolver esse erro, não crie seu projeto no nível mais profundo.
