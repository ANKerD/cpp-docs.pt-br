# Erro das Ferramentas de Vinculador LNK1224
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

endereço base da imagem inválido  
  
 Você especificou um endereço base inválido para a imagem.  Os endereços de base devem ser alinhados a \(os quatro os últimos dígitos hexadecimais devem ser zero\) e para imagem a base devem se ajustar dentro de um valor assinado ou não assinado de 32 bits.
