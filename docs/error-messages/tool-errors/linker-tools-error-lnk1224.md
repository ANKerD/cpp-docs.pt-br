# <a name="linker-tools-error-lnk1224"></a>Erro das Ferramentas de Vinculador LNK1224
endereço de base de imagem inválido  
  
 Você especificou um endereço base inválido para a imagem. Endereços de base devem ser alinhado de 64KB (os últimos quatro dígitos hexadecimais devem ser zero) e a imagem base deve se ajustar dentro de um valor de 32 bits assinado ou não assinado.
