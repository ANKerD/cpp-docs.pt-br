# <a name="resource-compiler-error-rc2171"></a>Erro RC2171 (compilador de recurso)

formato desconhecido de cabeçalho DIB

O cabeçalho de bitmap não é um **BITMAPCOREHEADER** ou **BITMAPINFOHEADER** estrutura.
