# Erro CXX0047 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

a lista de argumentos não corresponde a uma função  
  
 Uma expressão chamou uma função com uma lista de parâmetros reais que não corresponde à lista de parâmetro formal de nenhuma função com o mesmo nome definido no programa.  
  
 As funções sobrecarregadas podem ser chamadas somente se houver uma correspondência exata do parâmetro ou uma correspondência que não requer a criação de um objeto.  
  
 Esse erro é idêntico a CAN0047.
