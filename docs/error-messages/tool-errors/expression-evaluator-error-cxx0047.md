# <a name="expression-evaluator-error-cxx0047"></a>Erro CXX0047 (avaliador de expressão)
lista de argumentos não corresponde a uma função  
  
 Uma expressão de chamada de função com uma lista de parâmetros reais que não correspondia a lista de parâmetros formais de qualquer função com o mesmo nome definida no programa.  
  
 Funções sobrecarregadas podem ser chamadas somente se houver uma correspondência exata do parâmetro ou um que não exige a construção de um objeto.  
  
 Esse erro é idêntico ao CAN0047.
