# Aviso D9024 (linha de comando)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tipo de arquivo de origem não reconhecido “nome de arquivo”, arquivo de objeto assumido  
  
 A extensão do arquivo especificado não foi confirmada.  O arquivo foi considerado como um arquivo de objeto e passado ao vinculador.  
  
 As extensões são reconhecidas:  
  
-   .c \(arquivo de origem C\)  
  
-   .cxx arquivo de origem \(C\+\+\)  
  
-   .cpp \(arquivo de origem C\+\+\)  
  
-   .obj de arquivo \(objeto\)  
  
-   arquivo de biblioteca \(.lib\)  
  
-   .def \(arquivo de definição de módulo\)  
  
-   .exp \(a exportações do vinculador arquivo, criado por LINK \/LIB\)
