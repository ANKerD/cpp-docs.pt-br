# Erro das Ferramentas de Vinculador LNK1282
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

não é possível ler o arquivo \/REBASE; foi assinado  
  
 Você tentou alterar o endereço base de um assembly assinado com a opção \/REBASE para [editbin](../Topic/EDITBIN%20Reference.md).  Para fazer isso, primeiro alterar o endereço base e assinar o assembly.
