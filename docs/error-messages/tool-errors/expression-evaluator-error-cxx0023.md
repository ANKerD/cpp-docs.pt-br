# <a name="expression-evaluator-error-cxx0023"></a>Erro CXX0023 (avaliador de expressão)
base inválida  
  
 O avaliador de expressão C não reconhece a base especificada. Somente radixes decimais, hexadecimais e octais são válidos.  
  
 Esse erro é idêntico ao CAN0023.
