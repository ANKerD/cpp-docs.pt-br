# <a name="linker-tools-warning-lnk4002"></a>Aviso LNK4002 (Ferramentas de Vinculador)
símbolo definido no objeto  
  
 O símbolo exibido em sua forma decorada, foi especificado em sua forma não decorada em `object`, mas não foi possível encontrar uma correspondência exclusiva para um símbolo realçada. Esse aviso sempre é precedido por aviso [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) e seguido por um erro fatal [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
