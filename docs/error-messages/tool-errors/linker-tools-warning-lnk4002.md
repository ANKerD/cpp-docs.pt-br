# Aviso LNK4002 (Ferramentas de Vinculador)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

símbolo definido no objeto  
  
 O símbolo, exibido em sua forma decorado, foi especificado em sua forma undecorated em `object`, mas uma correspondência exclusivo para um símbolo decorado não foi localizada.  Esse aviso sempre é precedido [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) seguido pelo aviso e erro fatal [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
