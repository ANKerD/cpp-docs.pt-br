# Erro fatal U1088 (NMAKE)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

separador inválido “::” na regra de inferência  
  
 Uma regra de inferência deve ser seguida por um único dois\-pontos \(**:**\).
