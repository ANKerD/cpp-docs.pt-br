# <a name="project-build-error-prj0028"></a>Erro de Build PRJ0028 no Projeto

Arquivo temporário 'arquivo' contém conteúdo Unicode que não pôde ser convertido na página de código ANSI do usuário.

Um valor foi especificado com o [/MIDL (especificar opções de linha de comando MIDL)](../../build/reference/midl-specify-midl-command-line-options.md) opção de vinculador que não puderam ser resolvida pela página de código do sistema.

A página de código usada quando você especifica o comando MIDL (a página de código de entrada) deve ser o mesmo que a página de código do sistema.
