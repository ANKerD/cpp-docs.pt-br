# Erro PRJ0028 (compila&#231;&#227;o de projeto)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

O arquivo temporário “arquivo” contém o conteúdo Unicode que não puderam ser convertidos para a página de código ANSI do usuário.  
  
 Um valor de foi especificado com a opção do vinculador de [\/MIDL \(Especificar opções de linha de comando de MIDL\)](../../build/reference/midl-specify-midl-command-line-options.md) que não pôde ser resolvida pela página de código do sistema.  
  
 A página de código usada quando você especifica o comando de MIDL \(página de código de entrada\) deve ser igual à página de código do sistema.
