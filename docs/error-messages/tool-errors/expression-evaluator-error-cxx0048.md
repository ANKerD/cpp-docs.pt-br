# <a name="expression-evaluator-error-cxx0048"></a>Erro CXX0048 (avaliador de expressão)
sequência de chamada sem suporte  
  
 Uma função especificada na expressão usa uma sequência de chamada não tem suportada pelo avaliador de expressão C. Você não pode chamar essa função em uma expressão de janela de inspeção.  
  
 Esse erro é idêntico ao CAN0048.
