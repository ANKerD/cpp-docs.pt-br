# Erro CXX0048 (avaliador de express&#227;o)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sequência de chamadas sem suporte  
  
 Uma função especificada na expressão usa uma sequência de chamadas sem suporte do avaliador de expressão C.  Você não pode chamar essa função em uma expressão da janela de inspeção.  
  
 Esse erro é idêntico a CAN0048.
